QRIP/TSR - RESTORE GRAPHICS SCREEN

AX = AC15h
DI = 1092h


Return:
Nothing

Desc: Restores the graphics screen saved by the last call to AX=AC14h

Note: If DI <> 1092h on entry, QRIP chains the call

See Also: AX=AC03h - AX=AC14h

Category: Video - Int 2Fh - Q
