VESA XGA BIOS Extensions - RETURN FEATURE CONNECTOR STATE

AX = 4E06h
DX = XGA handle


Return:
AL = 4Eh if function supported
AH = status
00h  function successful
else error code
BX = Feature Connector State (see #00076)

See Also: AX=4E00h - AX=4E05h
