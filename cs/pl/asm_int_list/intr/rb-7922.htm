SK-UPPS Data Link Interface API - CHANGE DRIVER ADDRESS

AX = 0003h
ES:DI -> new node address (6 bytes)


Return:
AX = completion code (see #03846)

Note: The DLI will refuse to change the address if any protocols (see AX=0007h) are active

See Also: AX=0001h - AX=000Eh
