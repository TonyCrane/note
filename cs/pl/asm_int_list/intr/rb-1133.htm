IBM/Yale EBIOS SERIAL I/O - SET RECEIVE BUFFER

AX = FF02h
CX = length of buffer (0000h to cancel buffer assignment)
DX = port (00h-03h)
ES:BX -> receive buffer


Return:
Nothing
