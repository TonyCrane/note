Int 10/AX=007Fh/BH=06h

Interrups Categories Contents

VIDEO - Paradise VGA, AT&T VDC600 - ENTER MONOCHROME VGA MODE

AX = 007Fh
BH = 06h


Return:
AL = 7Fh if successful (AT&T VDC600)
BH = status (Paradise/Dell)
00h invalid function
7Fh successful

Note: This function also switches to video mode 7

See Also: AX=007Eh - AX=007Fh/BH=03h - AX=007Fh/BH=07h - AH=FFh"Oak"
