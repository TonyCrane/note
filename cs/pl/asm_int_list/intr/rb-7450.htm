MICEMM v4D, RM386 - INSTALLATION CHECK

AX = 5BF0h


Return:
AH = 00h if MICEMM or RM386 present
BX = code segment of driver

Program: MICEMM is the Micronics Expanded Memory Manager; RM386 is the memory manager included in Helix Software's Netroom

See Also: AH=3Fh - AX=5BE0h - AX=5BF1h

Category: Memory Management - Int 67h - M
