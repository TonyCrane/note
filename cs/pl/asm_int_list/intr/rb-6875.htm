Atari Portfolio - ENABLE/DISABLE SYSTEM STATUS LINE

AH = 2Eh
AL = new state
00h disabled
01h enabled
DH,DL = row,column at which to display


Return:
Nothing

See Also: AH=00h"Portfolio" - AH=2Dh"Portfolio" - AH=30h"Portfolio"
