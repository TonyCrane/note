MAGic v1.16+ - GET POSITION OF MAGNIFIED WINDOW

AX = 0005h


Return:
AX = status
0000h successful
BX = vertical position (char row or pixel row)
DX = horizontal position (char column or 8-pixel units)
FFFFh MAGic busy, retry later
BX,DX destroyed
CX destroyed

See Also: AX=0001h - AX=0003h - AX=0004h - AX=0006h -

Category: Access Software (screen readers, etc) - Int 49h - M
