NETWORK REDIRECTOR - REDIRECTED PRINTER MODE

AX = 1125h.

STACK:
WORD subfunction
5D07h get print stream state

Return:
DL = current state
5D08h set print stream state
DL = new state
5D09h finish print job


Return:
CF set on error
AX = error code (see #01680 at INT 21/AH=59h/BX=0000h)
STACK unchanged

Note: This function is called by the DOS 3.1+ kernel

See Also: INT 21/AX=5D07h - INT 21/AX=5D08h - INT 21/AX=5D09h

Category: Network - Int 2Fh - N
