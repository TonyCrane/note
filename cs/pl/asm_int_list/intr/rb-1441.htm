Int 15/AX=53B0h/BH=05h

Interrups Categories Contents

Intel System Management Bus - DISCONNECT

AX = 53B0h
BH = 05h
CX = 6941h ('iA')


Return:
CF clear if successful
AH = 00h (SMBus OK)
CF set if error
AH = error code (03h,04h,05h,0Ah,86h) (see #00484)

Note: This function is supported in connected mode (far CALL entry point) only

See Also: AX=53B0h/BH=01h - AX=53B0h/BH=02h - AX=53B0h/BH=03h - Category: Expansion Bus Bios - Int 15h - I
