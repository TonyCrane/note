VIDEO - Compaq Extensions - CHECK DAC 6-BIT/8-BIT mode

AX = BF0Eh


Return:
CF clear
AL = DAC mode
00h in 6-bit mode
01h in 8-bit mode

Note: Supported by at least the QVision board and the SystemPro XL ROM BIOS

See Also: AX=BF0Ch - AX=BF0Dh

Category: Vendor-specific BIOS Extensions - Int 10h - V
