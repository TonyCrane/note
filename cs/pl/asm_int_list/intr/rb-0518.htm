UNCHAIN - INSTALLATION CHECK

AX = CBFFh


Return:
AX = CCBBh if installed

Program: UNCHAIN is a TSR by Colin Buckley which permits use of Borland development tools with Mode X video in the absence of a second monitor

See Also: AX=CB00h - AX=CB01h - AX=CB06h
