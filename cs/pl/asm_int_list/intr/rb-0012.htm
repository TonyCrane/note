Int 03/AX=0000h

Interrups Categories Contents

Soft-ICE - BACK DOOR COMMANDS - GET Soft-ICE VERSION

AX = 0000h
SI = magic value 4647h ('FG')
DI = magic value 4A4Dh ('JM')


Return:
SI = BCD version (0280h = v2.80)

See Also: INT 03/AX=0910h
