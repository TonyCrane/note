Comwave Microfax Specification - GET STATUS ARRAY

AX = CBDCh
BL = 56h
BH = line number (starts with 1)
CX:DX -> 32-byte status buffer or 80 character string


Return:
AX = number of cards installed, or FFFFh on error

Program: Comwave manufacturers the Microfax line of fax cards which are a high-performance multi-card FAX solution for OEMs.

Note: Defaults to AH=CBh but can be changed

See Also: AX=CBDDh - AX=DA00h

Category: Fax - Int 2Fh - C
