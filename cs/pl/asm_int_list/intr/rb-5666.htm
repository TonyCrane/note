XVIEW - POP UP GIVING A PAGE NUMBER

AX = EE02h
DX = physical page number or anchor page number (see #03112)


Return:
AX = status (see #03111)

Note: Physical page numbers are assigned by the hypertext compiler, and will change if a page is inserted in the middle

See Also: AX=EE00h"XVIEW" - AX=EE01h - AX=EE05h - AX=EE06
