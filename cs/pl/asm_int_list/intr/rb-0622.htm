V10DISK.SYS - SET FORMAT

AH = 08h
AL = number of sectors
CH = cylinder number (bits 8,9 in high bits of CL)
CL = sector number
DH = head
DL = drive


Return:
AH = status code (see #00234)

Program: V10DISK.SYS is a driver for the Flagstaff Engineering 8" floppies
