TI Professional PC - GET EXTRA SYSTEM CONFIGURATION INFORMATION

AH = 0Bh


Return:
AL = drive type byte (see #03206)
BX = extra system configuration word 1 (see #03205)
CX = extra system configuration word 2 (see #03205)
AH destroyed

See Also: AH=00h - AH=01h - AH=02h - AH=03h - Category: Vendor-specific BIOS Extensions - Int 48h - T
