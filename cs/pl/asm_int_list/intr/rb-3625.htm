Novell NetWare - SYNCHRONIZATION SERVICES - CLEAR FILE

AH = EDh
DS:DX -> ASCIZ filename


Return:
AL = status
00h successful
FFh no files found

Desc: Unlock the file and remove it from the log table

Note: This function is supported by NetWare 4.6+, Advanced NetWare 1.0+, and Alloy NTNX

See Also: AH=CBh"NetWare" - AH=CEh - AH=CFh - AH=EBh"NetWare" - AH=ECh"NetWare"
