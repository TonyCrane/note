TopView - LOCATE - FIND WINDOW AT A GIVEN SCREEN LOCATION

AX = 1018h
BH = column
BL = row
ES = segment of object handle for window below which to search
0000h = start search with topmost window


Return:
ES = segment of object handle for window which is visible at the
indicated position, or covered by indicated window
0000h if no window

See Also: AX=1023h - AX=1024h

Category: DESQview/TopView and Quarterdeck Programs - Int 15h - T
