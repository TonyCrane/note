PRINT - END BACKGROUND PRINTING

AX = 8701h

Desc: Used to inform interested programs that PRINT has completed its background processing

Note: Called by PRINT after it has performed some background printing; not called if AX=8700h returned with CF set.

See Also: AX=8700h

Category: Printer Enhancements - Int 2Ah - P
