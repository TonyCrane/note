Int 2C/AX=0007h

Interrups Categories Contents

Cloaking - SET PROTECTED-MODE INTERRUPT VECTOR

AX = 0007h
CL = vector (00h-7Fh)
DX:EBX -> interrupt handler


Return:
CF clear

Notes: This function was first introduced with RM386 (RAM-MAN/386) v6.00. The IDT entry's type remains unchanged

See Also: AX=0006h - INT 31/AX=0205h
