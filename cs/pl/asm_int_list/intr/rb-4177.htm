Cloaking - GET PROTECTED-MODE INTERRUPT VECTOR

AX = 0006h
CL = vector (00h-7Fh)


Return:
CF clear
DX:EBX -> current interrupt handler

Note: This function was first introduced with RM386 (RAM-MAN/386) v6.00

See Also: AX=0007h - INT 31/AX=0204h
