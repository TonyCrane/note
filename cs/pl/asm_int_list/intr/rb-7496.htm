Int 67/AX=DE05h

Interrups Categories Contents

Virtual Control Program Interface - FREE 4K PAGE

AX = DE05h
EDX = physical address of 4K page


Return:
AH = status
00h successful
nonzero failed

Note: Also available in protected mode by calling the protected-mode VCPI entry point (see AX=DE01h,#03664)

See Also: AH=3Fh - AX=DE04h

Category: DOS Extenders - Int 67h - V
