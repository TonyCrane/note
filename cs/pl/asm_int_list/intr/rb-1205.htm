TopView - BEGINC - BEGIN CRITICAL REGION

AX = 101Bh


Return:
Task-switching temporarily disabled

Notes: Will not task-switch until "ENDC" (AX = 101Ch) called unless task voluntarily releases the CPU (upon regaining the CPU, task-switching will again be disabled). Suspends the caller until DOS is free

See Also: AH=0Dh"MultiDOS" - AX=101Ch - AX=DE13h - AX=DE1

See Also: INT 60/DI=0602h
