Realia COBOL - DEBUGGER SUPPORT

Note: Realia COBOL checks for a signature eight bytes prior to the interrupt handler when it starts up

BUG: If the offset of the INT 03 vector is less than 8, the compiler and all executables generated by it will crash the system
