MS Windows - GET DEVICE API ENTRY POINT

AX = 1684h
BX = virtual device (VxD) ID (see #02642)
ES:DI = 0000h:0000h


Return:
ES:DI -> VxD API entry point, or 0:0 if the VxD does not support an API

Note: Some Windows enhanced-mode virtual devices provide services that applications can access. For example, the Virtual Display Device (VDD) provides an API used in turn by WINOLDAP.
