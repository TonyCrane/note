DESQview v2.50+ - SEND MESSAGE 02h WITH ERROR RECOVERY

AH = 12h
BH = 82h

Note: This function is identical to AH=12h/BH=02h, except that DESQview will not pop up a "Programming Error" window, instead returning an error code in AL (see #00455)

See Also: AH=12h/BH=02h
