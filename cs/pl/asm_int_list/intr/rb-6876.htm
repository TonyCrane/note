PC/TCP kernel v2.05+ - icmp_ping - SEND ICMP ECHO REQUEST (PING)

AH = 30h
BX,DX = IP address of host
CX = length of data to send


Return:
CF clear if successful (i.e. reply received)
CF set on error
AX = error code (see #03319 at INT 61"FTP Software")

Note: This function is not supported by Beame&Whiteside's BWPCTCP v3.0a shim

See Also: INT 61"FTP Software" - INT 61/AH=00h"PC/TCP"

Category: Network - Int 61h - P
