Int 7F/AH=B0h

Interrups Categories Contents

Alloy NTNX, MW386 - RELEASE ALL SEMAPHORES FOR USER

AH = B0h
AL = user number
DS = code segment

Note: MW386 ignores AL and DS; it releases all semaphores locked using INT 67 or INT 7F locking functions

See Also: AH=B1h - AH=B2h - AH=B3h - AH=B4h
