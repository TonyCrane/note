HUNTER 16 - ACKNOWLEDGE EVENT

AH = 6Fh
AL = event number (see AH=6Dh)


Return:
AH = 00h if successful

Desc: Acknowledges the event, so the next similar event can be detected

See Also: AH=6Dh"HUNTER" - AH=6Eh

Category: Vendor-specific BIOS Extensions - Int 15h - H
