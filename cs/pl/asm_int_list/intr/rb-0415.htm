SOLLEX SuperVGA - VIDEO SUPPORT CONTROL - INITIALIZE VIDEO SUPPORT

AX = 7F03h
BL = 01h
CX = support request
ES = segment of support code


Return:
AL <> 7Fh if not supported
AL = 7Fh if supported
AH = status
00h successful
01h failed

Note: Initializes the indicated video support by calling ES:0003h; this function may be used to switch the active video support back to ROM after AX=7F03h/BL=02h

See Also: AX=7F03h/BL=00h

Category: Video - Int 10h - S
