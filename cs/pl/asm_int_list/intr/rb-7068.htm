FGDRIVER v4.02 - FG_REVMASK - DISPLAY REVERSED IMAGE (MASKING MAP)

AX = 00A1h
ES:BX -> array containing image stored as a masking map (see #03475)
CX = number of pixel runs in masking map
DX = width of masking map in pixels

Notes: The FGDRIVER functions are rearranged with each major release, but their parameters do not change (see #03474 at AX=0000h). This call is ignored in text modes and in native EGA and VGA graphics modes. The image is drawn with its lower left corner at the current graphics cursor position
