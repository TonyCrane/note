GRTMOUSE v1.00+ - INSTALLATION CHECK

AX = 0100h
CX = 4752h ('GR')
DX = 544Dh ('TM')


Return:
AX = 474Dh ('GM') if installed
CX = version number (CH = major, CL = minor)

Program: GRTMOUSE is a graphical-cursor driver for textmode by Tommer Leyvand

See Also: AX=0101h - AX=0102h - AX=0103h - AX=0104h
