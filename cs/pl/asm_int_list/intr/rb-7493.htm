Virtual Control Program Interface - GET MAX PHYSICAL MEMORY ADDRESS

AX = DE02h


Return:
AH = 00h  successful
EDX = physical address of highest 4K memory page.

AH nonzero:
Failed

See Also: AH=3Fh
