DOS 1+ - SET DISK TRANSFER AREA ADDRESS

AH = 1Ah
DS:DX -> Disk Transfer Area (DTA)

Notes: The DTA is set to PSP:0080h when a program is started. Under the FlashTek X-32 DOS extender, the pointer is in DS:EDX
