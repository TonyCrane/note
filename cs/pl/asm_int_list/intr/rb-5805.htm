DPMI 0.9+ - GET NEXT SELECTOR INCREMENT VALUE

AX = 0003h


Return:
CF clear
AX = value to add to get next sequential selector

Notes: The increment will be a power of two. Not supported by MS Windows 3.0 in Standard mode

See Also: AX=0000h
