DOS Protected Mode Services (DPMS) v1.0 - ENABLE/DISABLE DPMS

AX = 43E2h
BX = new state (0000h disable, 0001h enable)


Return:
AX = 0000h if supported

Note: This function should normally be called only by system software

See Also: AX=43E0h - AX=43E1h - AX=43E3h
