QUICKCACHE II v4.20 - SET BUFFER SIZE

AH = 2Ah
AL = buffer size (1-30)


Return:
AX = status (0000h successful)

Desc: Specify the number of cache sector buffers to dedicate to buffered read and write operations

See Also: AH=2Ch - AH=2Dh - AH=39h - AH=3Ah

Category: Caches/Spoolers - Int 13h - Q
