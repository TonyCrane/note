CauseWay - RelMemLinear32 - RELEASE LINEAR MEMORY BLOCK (32-bit)

AX = FF15h
ESI = linear address of block to be released


Return:
CF clear if successful
CF set on error

See Also: AX=FF0Fh - AX=FF11h - AX=FF15h
