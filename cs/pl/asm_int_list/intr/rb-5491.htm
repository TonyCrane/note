InterWave Game API - CLOSE GAME DEVICE

AX = CD22h
BX = installed program identifier
DX = device handle (from AX=CD21h)


Return:
AL = status
00h successful
FFh inavlid handle or device is not open

Range:
AH=CDh-ECh, selected by scanning for an available multiplex number
