COURIERS.COM - START INPUT

AH = 83h
ES:BX -> circular input buffer
CX = length of buffer
(should be at least 128 bytes if input flow control enabled)


Return:
Nothing

See Also: AH=18h - AH=87h - AH=8Dh - AH=A5h"BAPI"
