HUNTER 16 - GET IDLE TIMEOUT

AX = 6300h


Return:
AH = status
BX = timeout in seconds (0-3600)

Desc: Get the idle timeout value, the interval without keyboard or communications activity before the system shuts down

See Also: AX=6301h

Category: Vendor-specific BIOS Extensions - Int 15h - H
