AX = BF04h
BX = PSP segment of extender


Return:
Nothing???

Notes: If BX is not the PSP segment of the extender, it passes the call down the INT 15 chain; this allows nested instances of the extender. Grabs INT 2Fh and installs handlers for INT 2F/AX=1605h-1607h
