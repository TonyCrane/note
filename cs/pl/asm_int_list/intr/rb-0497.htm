VIDEO - Compaq QVision - GAMMA CORRECTION

AX = BF13h
BL = subfunction
00h load gamma correction table for true-color mode
DS = BIOS data segment
01h set palette RAM bypass
BH = 00h


Return:
Nothing

Note: These functions must be called after a video mode set, and are in effect only for the current video mode

See Also: #00733

Category: Vendor-specific BIOS Extensions - Int 10h - V
