MultiDOS Plus - RELEASE RESOURCE SEMAPHORE BY NAME

AH = 1Ch
DS:SI -> 8-byte name


Return:
AH = status
00h successful
01h not semaphore owner
02h invalid semaphore number
04h out of string space

Notes: (see AH=02h"MultiDOS"). Equivalent to AH=1Ah followed by AH=02h
