FGDRIVER v4.02 - FG_SETVPAGE - SET VISIBLE VIDEO PAGE

AX = 00B6h
BX = new video page (0000h to 003Fh)

Notes: The FGDRIVER functions are rearranged with each major release, but their parameters do not change (see #03474 at AX=0000h). The specified page must be a physical or virtual page
