Interrups Categories Contents

Tandy 1000SL/TL - WRITE TO EEPROM

AX = 7001h
BL = number of word to read (00h-3Fh)
DX = new value for EEPROM word


Return:
CF clear if function supported

Note: The EEPROMs are normally written only by the system setup program; changing the values can badly mess up a Tandy

See Also: AX=7000h

Category: Non-Volatile Config Storage - Int 15h - T
