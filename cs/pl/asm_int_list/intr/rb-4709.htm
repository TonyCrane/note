DR DOS 6.0 TaskMAX - GET SWAP SPACE INFO

AX = 2714h


Return:
CX = total KB of swap space
DX = available KB of swap space

Note: The TaskMAX API is also supported by Novell DOS 7 TASKMGR in both taskswitching and multitasking modes

See Also: AX=2701h

Category: DOS-Based Task Switchers/Multitaskers - Int 2Fh - D
