Novell NetWare - DOS Requester - BROADCAST CALLBACK

AX = 7A21h
CX = server connection (FFFFh if message from Personal NetWare user)
interrupts disabled


Return:
CX = 0000h if broadcast handled
interrupts disabled

Note: This function is only linked into INT 2F when INT 21/AH=DEh/DL=05h has been issued

See Also: AX=7A22h - AX=7A85h - INT 21/AH=DEh

Category: Netware - Int 2Fh - N
