APPC/PC - NETWORK DEVICE CONTROL - DISPLAY

AH = 01h subfn 1B00h
DS:DX -> control block (see #03667)


Return:
Control block updated

See Also: AH=01h/SF=2000h - AH=01h/SF=2B00h - INT 68"Novell"
