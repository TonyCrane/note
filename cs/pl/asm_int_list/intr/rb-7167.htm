Int 62/AH=0Fh

Interrups Categories Contents

Cswitch - LOAD AND RUN PROGRAM FROM DISK

AH = 0Fh
ES:BX -> command line
CX = priority (1-10)
DX = background flag (nonzero allows loading to EMS)


Return:
AX = status
0000h task loader queue is full
0001h  no error

See Also: AH=07h"Cswitch" - AH=10h"Cswitch" - AH=13h"Cswitch"
