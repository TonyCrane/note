FTP Packet Driver - BASIC FUNC - SEND PACKET

AH = 04h
DS:SI -> buffer
CX = length


Return:
CF set on error
DH = error code (see #03305)
CF clear if successful

Note: The buffer may be modified immediately upon return from this call

See Also: AH=0Bh
