TI Professional PC - OPTION ROM DATA AREA POINTER (NOT A VECTOR!)

Desc: The low word of this vector contains the segment of the RAM data area to be used by the expansion ROM at F400h:4000h, and the high word contains the length of the data area; this segment and size are both set to 0000h if no ROM is installed at F400h:4000h
