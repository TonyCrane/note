Int 7B

Interrups Categories Contents

AutoCAD Device Interface - PRINTER/PLOTTER - SEND PLOTTING DATA

AX = data length (0000h-7FFFh)
CX:BX -> data to be plotted
DX = vertical position of line on page


Return:
Nothing

Notes: This interrupt is called by AutoCAD to perform its output. A data length of 0000h indicates that a blank line is to be output

See Also: AX=8001h - AX=8002h - INT 78"AutoCAD" - INT 7A/
