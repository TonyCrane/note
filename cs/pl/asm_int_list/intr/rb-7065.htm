FGDRIVER v4.02 - FG_RESTORE - COPY REGION FROM HIDDEN TO VIS PAGE

AX = 009Eh
BX = left edge column
CX = right edge column
DX = top edge row
SI = bottom edge row

Notes: The FGDRIVER functions are rearranged with each major release, but their parameters do not change (see #03474 at AX=0000h). The left and right edges are adjusted to byte boundaries if necessary

See Also: AX=00A2h - AX=00C6h
