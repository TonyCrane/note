NETWORK REDIRECTOR - FLUSH ALL DISK BUFFERS

AX = 1120h
DS = DOS DS
???


Return:
CF clear (successful)

Notes: This function is called by the DOS 3.1+ kernel. Uses CDS array pointer and LASTDRIVE= entries in DOS list of lists
