SK-UPPS Data Link Interface API - SYNC REQUEST BUFFER

AX = 0012h
DX = protocol ID (see #03849)


Return:
AX = completion code (see #03846).

If AX=0000h:
ES:BX -> Mbuf (see #03847)

Note: If the protocol wants to be informed when an Mbuf becomes available, REQUEST BUFFER (AX=0004h) should be used instead.

See Also: AX=0004h - AX=0005h - AX=0006h - AX=0013h
