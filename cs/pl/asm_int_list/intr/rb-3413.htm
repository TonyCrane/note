Novell NetWare - PRINT SERVICES - START LPT CAPTURE

AH = DFh
DL = 00h


Return:
AL = status
00h successful

Desc: This function redirects the default LPT to a capture file on the file server

Note: This function is supported by NetWare 4.0+, Advanced NetWare 1.0+, and Alloy NTNX; under NTNX, it sends a print break (see INT 17/AH=84h). A print job is queued when the first character of output is captured

See Also: AX=B800h - AX=B804h - AH=DFh/DL=01h - AH=DFh/DL

See Also: AH=DFh/DL=04h - AX=F003h
