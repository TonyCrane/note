AMI BIOS - BIOS-FLASH Interface - VERIFY ALLOCATED MEMORY

AX = E00Ah
BX = number of paragraphs
ES = starting segment of memory


Return:
CF clear if successful
AL = FAh
CF set on error

Desc: Determine whether the specified memory may be used for flash programming

Note: Always returns error if BX is zero on entry

See Also: AX=E000h - AX=E00Bh

Category: Vendor-specific BIOS Extensions - Int 16h - A
