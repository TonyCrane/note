QRIP/TSR - CONVERT PROTECTED AREA TO ENTIRE SCREEN

AX = AC11h
DI = 1092h
BX = mode (0000h = normal protect, 0001h = allow full-screen access)

Note: If DI <> 1092h on entry, QRIP chains the call

See Also: AX=AC0Dh - AX=AC0Eh - AX=AC0Fh - AX=AC10h -
