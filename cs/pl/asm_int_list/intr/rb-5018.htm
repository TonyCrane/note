FaxBIOS interface - COMMAND SUBMISSION

AX = 80FBh
BX = function number (see #02930)
DX:DI -> command buffer
(see #02932,#02935,#02936,#02937,#02938,#02939,#02944,#02950,#02957)


Return:
AL = FFh if submitted OK
CX = result code (see #02931)

Range:
AH=80h to AH=FFh, selected by scanning multiplex numbers for signature

See Also: AX=8000h"FaxBIOS" - AX=CBDDh
