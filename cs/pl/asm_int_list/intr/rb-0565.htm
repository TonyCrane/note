EGA Register Interface Library - INTERROGATE DRIVER

AH = FAh
BX = 0000h


Return:
BX = 0000h if RIL driver not present
ES:BX -> EGA Register Interface version number, if present:
byte 0 = major release number
byte 1 = minor release number

Note: The RIL is provided by EGA.SYS, the Microsoft Mouse driver, the OS/2 compatibility box, and others

See Also: AH=F0h"EGA" - AH=F6h"EGA" - INT 2F/AX=BC00h
