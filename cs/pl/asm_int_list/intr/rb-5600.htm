DESQview v2.26+ External Dev Interface - CREATE TASK

AX = DE06h
BX = mapping context of process containing task
DX = handle of new task

Notes: Driver should pass this call to previous handler after processing it. All DOS, BIOS, and DV API calls are valid in handler
