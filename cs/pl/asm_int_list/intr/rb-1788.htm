HUNTER 16 - CONTROL SCREEN DUMP AREA

AH = 27h
AL = what to dump
00h whole (virtual) window
01h LCD window only


Return:
AL = 00h

Desc: Control whether printscren dumps the whole 80x25 screen or only the part displayed in the LCD window

Note: The Hunter 16 has a 240x64 LCD display which serves as a window into a 640x200 virtual screen

Category: Vendor-specific BIOS Extensions - Int 16h - H
