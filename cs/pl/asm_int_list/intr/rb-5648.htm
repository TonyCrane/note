NDOS - API

AX = E44Dh

Program: NDOS is a version of 4DOS licensed to Symantec for inclusion in the Norton Utilities

Note: As NDOS is a licensed version of 4DOS v3.03, the API is identical to that for 4DOS, except that AH=E4h instead of D4h and the installation check returns AX=44EEh instead of AX=44DDh

See Also: AX=D44Dh - AX=E44Eh
