MS Windows - VADMAD - GET API ENTRY POINT

AX = 1684h
BX = 0444h (virtual device ID for VADMAD device) (see #02642)
ES:DI = 0000h:0000h


Return:
ES:DI -> VxD API entry point (see #02683)
0000h:0000h if the VxD does not support an API



(Table 02683)
Call VADMAD entry point with:
DX = operation
0000h set VADMAD mode
AX = desired mode
0001h set VADMAD channel
AX = desired channel

Note: After setting mode/channel, start the DMA operation with an OUT to I/O port 0Bh (channels 0-3) or D6h (channels 4-7)

See Also: #01268 at INT 20"Windows"

Category: Windows - Int 2Fh - M
