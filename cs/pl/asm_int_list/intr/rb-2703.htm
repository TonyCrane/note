DOS 1+ - GET SYSTEM TIME

AH = 2Ch


Return:
CH = hour
CL = minute
DH = second
DL = 1/100 seconds

Note: On most systems, the resolution of the system clock is about 5/100sec, so returned times generally do not increment by 1. On some systems, DL may always return 00h
