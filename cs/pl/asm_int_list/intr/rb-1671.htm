AMI BIOS - Flash ROM - GET SIZE OF FLASH ROM PROGRAMMING CODE

AX = DB02h


Return:
CF clear
AX = 0000h
BX = size of ROM programming code in bytes

Note: Used by FMUP.EXE, Intel's Flash Memory Update utility

See Also: AX=DB00h - AX=DB03h

Category: Vendor-specific BIOS Extensions - Int 15h - A
