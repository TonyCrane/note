HP Vectra EX-BIOS - F_INS_FREEGETDS - INSTALL VALUES IN FREE VECTOR

AH = 14h
BP = 0012h (driver ID for V_SYSTEM)
ES:DI -> new CS:IP entry point


Return:
AH = status (00h) (see #03744)
BP,DS destroyed

Note: On installation, the given entry point will be invoked with an SF_INIT call (see AX=0200h"SF_INIT"), with the "last used DS" value in BX; the routine should adjust BX and return the new value. If the SF_INIT call returns with error code FEh, the power-on self-test sequence will be called.

See Also: AH=00h"HP Vectra" - AH=04h/BP=0012h - AH=0Eh/BP=0012h - See Also: AH=16h/BP=0012h
