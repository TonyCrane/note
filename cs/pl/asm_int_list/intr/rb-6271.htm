Int 4B/AX=810Bh

Interrups Categories Contents

Virtual DMA Specification - DISABLE DMA TRANSLATION

AX = 810Bh
BX = DMA channel number
DX = 0000h


Return:
CF clear if successful
CF set on error
AL = error code (see #03218)

See Also: AX=810Ch

Category: Disk I/O Enhancements - Int 4Bh - V
