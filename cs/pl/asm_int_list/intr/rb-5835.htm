DPMI 0.9+ - FREE REAL MODE CALLBACK ADDRESS

AX = 0304h
CX:DX = real mode callback address


Return:
CF clear if successful
CF set on error
AX = error code (DPMI 1.0+) (8024h) (see #03143)

Note: The limited DPMI host built into Phar Lap's 286|DOS-Extender v2.5 does not support this function

See Also: AX=0303h
