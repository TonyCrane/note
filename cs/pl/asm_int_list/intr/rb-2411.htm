Intel Plug-and-Play AUTO-CONFIGURATION - STORE ESCD TABLE IN NVRAM

AX = B403h
DS:SI -> configuration table (see #01244)


Return:
CF clear if successful
AX = 0000h
CF set on error
AX = error code (0055h) (see #01243)

Note: Sets the configuration table's checksum field, then copies the table into nonvolatile storage (i.e. FlashROM)

See Also: AX=B400h - AX=B402h

Category: Expansion Bus Bios - Int 1Ah - I
