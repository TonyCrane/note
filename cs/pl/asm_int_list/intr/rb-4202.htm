Int 2C/AX=002Bh/CH=81h

Interrups Categories Contents

Cloaking - PROTECTED-MODE VIRTUAL DMA SERVICES

AX = 002Bh
CH = 81h
CL = subfunction (02h-0Ch)
other registers as appropriate for subfunction


Return:
Varies by function
CF set on error

Notes: These functions are equivalent to the INT 4B/AX=81xxh subfunctions with the same numbers. This function was first introduced with RM386 (RAM-MAN/386) v6.00

See Also: INT 4B/AX=8102h - INT 4B/AX=810Ch
