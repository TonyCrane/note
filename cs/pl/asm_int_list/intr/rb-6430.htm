HP 100LX/200LX GRAPHICS PRIMITIVES - SET CURRENT FONT

AH = 11h
ES:DI -> ptr to font or 0000h:fontID# for built-in font

Note: This function should be called immediately after AH=10h with the pointer supplied by that call

See Also: AH=10h
