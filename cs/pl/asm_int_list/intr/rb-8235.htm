APL*PLUS/PC - REPORT WHETHER NAME IS ELIGIBLE FOR OUTSWAPPING

AL = F8h
BX = STPTR of object


Return:
BX = name's outswapping status
0000h eligible
0001h not eligible

See Also: AL=F6h - AL=F7h - INT C8"APL"
