Novell NetWare - BINDERY SERVICES - ADD BINDERY OBJECT TO SET

AH = E3h subfn 41h
DS:SI -> request buffer (see #01951)
ES:DI -> reply buffer (see #01949)


Return:
AL = status (see #01950)

Desc: Add the specified object to an object's group property

Note: This function is supported by Advanced NetWare 1.0+ and Alloy NTNX

See Also: AH=E3h/SF=40h - AH=E3h/SF=42h - AH=E3h/SF=43h -
