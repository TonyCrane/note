Atari Portfolio - GET PERIPHERAL ID BYTE

AH = 1Ah


Return:
AH = peripheral ID byte
AL = 00h if no peripheral installed

Desc: Get the ID code for the current terminating peripheral

See Also: AH=00h"Portfolio" - AH=18h"Portfolio" - AH=1Bh"Portfolio"
