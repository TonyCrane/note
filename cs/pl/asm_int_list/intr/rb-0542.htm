S3VBE/Core2.0 v3.10+ - DEACTIVATE SPEED-UP

AX = D005h


Return:
AX = FFFFh if supported
BX = status
0000h SpeedUp deactivated
0001h SpeedUp not possible (wrong memory organization for mode)
0002h SpeedUp not possible (linear frame buffer active)

Notes: Called by S3 SpeedUp (see AX=C000h). Functions 06h-FFh (e.g. AX=D006h-D0FFh) are considered reserved by v3.10+ and return AX=5555h
