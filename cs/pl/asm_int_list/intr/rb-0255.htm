VIDEO - Hercules GRAFIX - DISP - SELECT PAGE TO DISPLAY

AH = 45h
AL = page number (0,1)


Return:
Nothing

Desc: Specify which of the two pages of video memory is visible on screen

See Also: AH=05h - AH=42h - AH=43h
