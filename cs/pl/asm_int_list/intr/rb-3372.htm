Novell NetWare, Alloy NTNX - CLEAR FILE (FCB)

AH = CEh
DS:DX -> FCB (see #01345 at AH=0Fh)


Return:
AL = error code

Desc: Unlocks file and removes it from log table, then closes all opened and logged occurrences

Note: This function was added in NetWare 4.0, but was removed some time prior to Advanced NetWare 2.15, and is no longer listed in current Novell documentation

See Also: AH=CAh - AH=CFh - AH=EDh"NetWare"

Category: Netware - Int 21h - N
