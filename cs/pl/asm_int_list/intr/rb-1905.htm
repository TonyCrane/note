AMI BIOS - BIOS-FLASH Interface - FLASH WRITE PROTECT

AX = E006h


Return:
CF clear if successful
AL = FAh
CF set on error

Note: This function performs any delay required to allow the Flash ROM to stabilize in the write-protected state

See Also: AX=E000h - AX=E004h - AX=E007h

Category: Vendor-specific BIOS Extensions - Int 16h - A
