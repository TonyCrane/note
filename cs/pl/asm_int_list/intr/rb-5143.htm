QRIP/TSR - GET FONT DIRECTORY

AX = AC07h
DI = 1092h


Return:
DX:BX -> 80-byte internal buffer containing ASCIZ font directory name

Notes: If DI <> 1092h on entry, QRIP chains the call. The application may change the font directory by overwriting the returned buffer, ensuring that it does not exceed 80 bytes incl. NUL. If changed, the new path will not become effective until the graphics screen is reinitialized

See Also: AX=AC08h - AX=ACF0h

Category: Video - Int 2Fh - Q
