Novell NetWare, Alloy NTNX - SET FCB RE-OPEN MODE

AH = E8h
DL = mode
00h no automatic re-open
01h auto re-open


Return:
AL = error code

Desc: Provided backward compatibility with a bug in CP/M and early DOS vers

Note: This function was added in NetWare 4.6, but was removed some time prior to Advanced NetWare 2.15, and is no longer listed in current Novell documentation

Category: Netware - Int 21h - N
