Novell NetWare - SNMP.EXE - INSTALLATION CHECK

AX = 7A4Fh
BX = 0001h


Return:
AX = 7AFFh if installed

Note: This call conflicts with RARPD.EXE (both SNMP and RARPD are supplied with LAN Workplace for DOS!), such that running SNMP will uninstall RARPD as SNMP checks whether it is already installed!
