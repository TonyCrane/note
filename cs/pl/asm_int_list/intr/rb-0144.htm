VIDEO - Realtek RTVGA - SET USER GRAPHICS CHARACTERS

AX = 1110h
ES:BP -> user table
CX = bytes per character
BL = row specifier
00h user set
DL = number of rows
01h 14 rows
02h 25 rows
03h 43 rows


Return:
Nothing

Note: This function is meant to be called immediately after a mode set; results are unpredictable at other times

See Also: AX=110Fh"Realtek" - AX=1112h"Realtek" - AX=1121h
