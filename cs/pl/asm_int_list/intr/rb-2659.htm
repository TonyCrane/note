Phar Lap 386/DOS-Ext VMM v2.3+ - SET SIZE OF BUFFER FOR DOS CALLS

AX = 2530h
ECX = size of data buffer in bytes (1024 to 65536)


Return:
CF clear if successful
CF set on error
EAX = error code
08h insufficient low memory
81h invalid size

See Also: AX=2517h
