NETWORK - PC LAN Program, Microsoft Networking - GET VERSION

AX = B809h


Return:
AH = minor version (decimal)
AL = major version

Notes: This function is supported in this form by PC LAN Program, LAN Manager, the DOS LAN Requester, 10NET v5.0, and Microsoft Networking under WfWg 3.11. 10NET returns version 1.10 (AX=0A01h) for compatibility

See Also: AX=4E53h - AX=B800h - AX=B809h"LANtastic"
