KEYBOARD - AX PC - READ SHIFT KEY STATUS

AH = 51h


Return:
AL = standard shift key states (see #00582,#00587)
AH = Kana lock (00h off, 01h on)

Note: Also supported by K3PLUS v6.00+, but K3PLUS always returns AH=00h

See Also: AH=02h - AH=12h - AH=22h

Category: Japanese - Int 16h - K
