DOS 2+ - GET INTERRUPT VECTOR

AH = 35h
AL = interrupt number


Return:
ES:BX -> current interrupt handler

Note: Under DR DOS 5.0+, this function does not use any of the DOS-internal stacks and may thus be called at any time

See Also: AH=25h - AX=2503h
