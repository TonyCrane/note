DESQview v2.50+ - DISPATCHINTAFTERDOS - INTERRUPT ANOTHER TASK

AX = DE2Ah
BX = segment of handle for task to interrupt or 0000h for caller
DX:CX -> interrupt routine
BP,SI,DI,DS,ES as required by interrupt routine


Return:
Nothing

Notes: DESQview 2.50-2.52 are distributed as part of DESQview/X v1.0x.. This call is the same as AX=DE20h except that it will delay interrupting the specified task until after it has exited DOS

See Also: AX=1021h - AX=DE20h
