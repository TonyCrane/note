Alloy NTNX, MW386 - GET CHANNEL STATUS

AH = 15h
CL = 00h
AL = channel number
DX:DI -> status structure (see #03958)


Return:
AL = status (00h-03h) (see #03954)

See Also: AH=15h/CL=01h 


Format of Alloy channel status structure:

Offset  Size    Description     (Table 03958)
00h    BYTE    channel status

bit 0:
Channel open

bit 1:
Channel buffer contains received data

bit 7:
Channel locked
01h    BYTE    sender ID

Category: Network - Int 7Fh - A
