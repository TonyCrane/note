Alloy MW386 - GET DEVICE NAME FROM PHYSICAL DEVICE NUMBER

AH = 8Ch
DX = physical device number
ES:DI -> 17-byte buffer for ASCIZ device name


Return:
AL = status (see also AH=81h)
00h successful
ES:DI buffer filled

See Also: AH=88h - AH=8Bh

Category: Network - Int 17h - A
