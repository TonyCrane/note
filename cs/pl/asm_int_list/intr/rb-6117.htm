FLOATING POINT EMULATION - OPCODE D8h

Desc: This interrupt is used to emulate floating-point instructions with an opcode of D8h

Note: The floating-point emulators in Borland and Microsoft languages and Lahey FORTRAN use this interrupt

See Also: INT 35 - INT 3E

Category: Runtime Support - Int 34h - F
