HUNTER 16 - CONTROL BATTERY CHANGE FACILITIES (APM)

AH = 7Eh
AL = 00h do not prompt
else prompt

Desc: Select whether the user is prompted for the battery state if the battery cap has been off when the system is turned on

See Also: AH=7Fh

Category: Vendor-specific BIOS Extensions - Int 15h - H
