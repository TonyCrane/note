Int 62/AH=0Ch

Interrups Categories Contents

Cswitch - READ MESSAGE

AH = 0Ch
CX = number of bytes to read
DS:SI -> buffer
DX = queue number (0-63)


Return:
AX = status
FFFFh bad queue number
else  number of bytes transferred

See Also: AH=0Ah - AH=0Bh
