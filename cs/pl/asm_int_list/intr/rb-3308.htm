Attachmate Extra! - DISABLE HOST BUFFER UPDATES

AH = B8h
AL = window number (1-4=host B-E)
DL = 01h

Notes: Only valid in CUT mode. Next AID keystroke (eg Enter) enables host buffer updates

See Also: AH=A4h
