PCMCIA Socket Services v1.00 - RESUME ERROR DETECTION CODE

AH = 9Ah
AL = adapter number
BH = EDC generator


Return:
CF clear if successful
AH destroyed
CF set on error
AH = error code (01h,04h) (see #00656)

See Also: AH=99h - AH=98h

Category: Expansion Bus Bios - Int 1Ah - P
