STARLITE architecture - BIND TO NETWORK DEVICE

AX = 5F09h
DS:DX -> ASCIZ name of the device driver to attach to


Return:
CF set on error
AX = error code (see #01680 at AH=59h/BX=0000h)
CF clear if successful

Note: The STARLITE distributed file system can attach to multiple networks simultaneously

See Also: AX=5F0Ah

Category: Other Operating Systems - Int 21h - S
