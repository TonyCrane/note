SNAP.EXE 3.2+ - SNAP_CHKINSTALL - INSTALLATION CHECK

AX = 6F01h.

STACK:
WORD 0000h


Return:
AX = status
0000h SNAP is resident
other SNAP not present
STACK unchanged

Program: SNAP.EXE is a TSR written by IBM and Carnegie Mellon University which implements the Simple Network Application Protocol, and is required by PCVENUS (a network shell). The combination of SNAP and PCVENUS allows the use of the Andrew File System as one or more networked drives.

See Also: AX=6900h - AX=7400h

Category: Network - Int 1Ah - S
