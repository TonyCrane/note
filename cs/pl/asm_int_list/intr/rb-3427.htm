Software Carousel - CHANGE PARTITION NAME

AX = DF06h
BL = task number
CX = length of new name (00h to use default, max 18h)
DS:SI -> new name


Return:
AL = status
00h Carousel not running
01h successful
FFh unsucessful
AH = error code (01h) (see #01813)

See Also: AX=DF04h

Category: DOS-Based Task Switchers/Multitaskers - Int 21h - S
