DBLSPACE.BIN v6.2 - SET AUTOMOUNT DRIVES

AX = 4A11h
BX = 000Ah
CX:DX = bitmask of drives on which to enable AutoMount
(DX bit 0 = A:, CX bit 0 = P:, etc.)


Return:
AX = 0000h if supported
CX:DX = old mask or 0000h:0000h on error
BX destroyed

See Also: AX=4A11h/BX=000Bh

Category: File/Disk Compression - Int 2Fh - D
