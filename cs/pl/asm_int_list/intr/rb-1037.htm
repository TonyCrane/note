COMM-DRV v14.0 - FLUSH COMMUNICATION BUFFERS

AH = 83h
DX = port number
AL = subfunction
00h flush input buffer
01h flush output buffer
02h flush both buffers


Return:
AH bit 7 set on error
AH bit 7 clear if successful

See Also: AH=81h"COMM-DRV" - AH=84h"COMM-DRV"
