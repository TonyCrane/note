PCMCIA Socket Services v1.00 - SET ADAPTER

AH = 86h
AL = adapter number
DH = new adapter attributes (see #00672)


Return:
CF clear if successful
AH destroyed
CF set on error
AH = error code (01h) (see #00656)

See Also: AH=84h"PCMCIA" - AH=85h"PCMCIA"
