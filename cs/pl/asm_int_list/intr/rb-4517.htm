MS Windows - WINDOWS ENHANCED MODE INIT COMPLETE BROADCAST

AX = 1608h

Notes: Called after all installable devices have been initialized. Real-mode software may be called between the Windows enhanced-mode init call (AX=1605h) and this call; the software must detect this situation

See Also: AX=1605h - AX=1609h
