Int 7F/AH=14h/CL=02h

Interrups Categories Contents

Alloy NTNX, MW386 - SET CHANNEL BUFFER POINTER

AH = 14h
CL = 02h
AL = channel number
DX:DI -> receive buffer


Return:
AL = status (00h-03h) (see #03954)

Note: May be called from within a receive ISR or when a datagram is pending

See Also: AH=10h/CL=00h - AH=14h/CL=00h
