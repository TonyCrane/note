SYSTEM - REQUEST POWER OFF (CONVERTIBLE,HP 95LX)

AH = 42h
AL = suspend type
00h to use system profile
01h to force suspend regardless of system profile


Return:
Nothing

Note: The HP 95LX apparently suspends regardless of the value in AL; on power-up, execution will resume following the instruction calling this function

See Also: AH=44h

Category: Bios - Int 15h - S
