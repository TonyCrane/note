Int E0/CL=8Eh

Interrups Categories Contents

ConcCP/M,DR Multiuser DOS - P_DISPATCH - CALL DISPATCHER

CL = 8Eh
DX = FFFFh (optional) to force dispatch


Return:
Nothing

Desc: Allow other processes of the same or higher priority to run if they are ready

Notes: If DX=FFFFh, a dispatch is forced even if no other process is ready. Also supported by REAL/32

See Also: INT E0/CL=91h - INT 15/AX=1000h - INT 2F/AX=1680h

Category: Other Operating Systems - Int E0h - C
