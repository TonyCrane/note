Int 15/AH=0Fh

Interrups Categories Contents

SYSTEM - FORMAT UNIT PERIODIC INTERRUPT (PS ESDI drives only)

AH = 0Fh
AL = phase code
00h reserved
01h surface analysis
02h formatting


Return:
CF clear if formatting should continue
CF set if formatting should terminate

Note: Called during ESDI drive formatting after each cylinder is completed

See Also: INT 13/AH=1Ah

Category: Disk I/O Enhancements - Int 15h - S
