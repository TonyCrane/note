10NET v3.3+ - SET SHARED DEVICE ENTRY

AX = 1502h
DS:SI -> node ID, 12 bytes blank-padded
ES:DI -> valid shared device table entry


Return:
CF clear if successful
CF set on error
AX = error code (see #03740)

See Also: AX=1501h - AX=1503h - AX=9502h
