SWBIOS - INSTALLATION CHECK

AH = F9h
DL = drive number (80h,81h)


Return:
CF clear
DX = configuration word
bit 15 set if other SWBIOS extensions available
CF set on error

Program: SWBIOS is a TSR by Ontrack Computer Systems. XBIOS is a driver in some versions of Disk Manager that is loaded from the disk MBR, replacing the ROM BIOS disk support eg adding LBA mode support, and read/write multiple.

Note: Disk Manager also supports these calls

See Also: AH=EEh - AX=E000h"XBIOS" - AH=FFh"EZ-Drive"
