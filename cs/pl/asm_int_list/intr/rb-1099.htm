3com BAPI SERIAL I/O - GET ECM WATCH STATE

AH = B2h


Return:
AH = return code (00h,07h,0Ah) (see #00394)
AL = watch flag (00h disabled, 01h enabled)

Desc: Determine whether the ECM character is enabled

See Also: AH=B0h - AH=B1h
