Int E0/CL=91h

Interrups Categories Contents

ConcCP/M,DR Multiuser DOS - P_PRIORITY - SET PROCESS PRIORITY

CL = 91h
DL = new priority (00h highest to FFh lowest)


Return:
Nothing

Notes: Sets priority of calling process; transient processes are initialized to priority C8h. Also supported by REAL/32

See Also: INT E0/CL=8Eh

Category: Other Operating Systems - Int E0h - C
