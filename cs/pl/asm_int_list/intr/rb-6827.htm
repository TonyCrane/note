Atari Portfolio - GET/SET CURSOR MODE

AH = 0Fh
AL = function
00h get mode
01h set mode
BL = new cursor mode
00h cursor off
01h underline
02h block
02h force mode (set size to reflect NumLock state)


Return:
BL = current/old cursor mode
