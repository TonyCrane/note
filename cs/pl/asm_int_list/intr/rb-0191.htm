Cirrus Logic BIOS v3.02 - LCD panel - SET AUTOMAP/EMULATE ATTRIBUTES

AH = 12h
BL = 8Ch
AL = new state
00h enable automap
01h disable automap and emulate attributes


Return:
Nothing

See Also: AH=12h/BL=84h - AH=12h/BL=8Bh
