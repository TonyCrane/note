Int 15/AH=12h/BH=0Bh

Interrups Categories Contents

TopView - SEND MESSAGE - SUBFROM - RESET OBJECT BITS

AH = 12h
BH = 0Bh
BL = object
00h handle is DWORD on top of stack

pointer:
Reset control flags

keyboard:
Reset control flags
04h clear control flags on KEYBOARD object (handle on top of stack)
05h clear control flags on task's default KEYBOARD object.

STACK:
DWORD   which bits to clear (see #00450,#00451)


Return:
STACK popped

See Also: AH=12h/BH=0Ah"OBJECT"
