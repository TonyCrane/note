Blank-It Screen Blanker - GET BLANKING TIMEOUT

AX = 0225h
BX = 6904h


Return:
CF clear if successful
BX = timeout (see AX=0225h/BX=6901h)
CF set on error
AL = error code

See Also: AX=0225h/BX=6900h - AX=0225h/BX=6901h
