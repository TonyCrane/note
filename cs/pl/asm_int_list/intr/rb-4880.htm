JAM.SYS v1.10+ - Detach - UNMOUNT COMPRESSED DRIVE

AX = 5203h
DL = drive number (0-default, 1-A:, etc.) to detach


Return:
AH = status (00h,01h,04h,05h,08h,09h,0Bh,0Dh) (see #02835)
AL = host drive error code (see #02598 at INT 2F/AX=0802h)

See Also: AX=5202h

Category: File/Disk Compression - Int 2Fh - J
