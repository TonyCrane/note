Windows95 - CLOSE-AWARENESS - CANCEL CLOSE

AX = 168Fh
DH = 03h
DL = 00h (reserved)


Return:
AX = status
0000h successful
else failed

Desc: Cancels a close request which has already been acknowledged if the application determines that it will not exit at this time

See Also: AX=168Fh/DH=00h - AX=168Fh/DH=03h

Category: Windows - Int 2Fh - W
