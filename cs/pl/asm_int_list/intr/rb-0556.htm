Interrups Categories Contents

EGA Register Interface Library - WRITE REGISTER RANGE

AH = F3h
CH = starting register
CL = number of registers (>1)
DX = group index (00h,08h,10h,18h) (see #00223)
ES:BX -> buffer, CL bytes


Return:
Nothing

Note: The RIL is provided by EGA.SYS, the Microsoft Mouse driver, the OS/2 compatibility box, and others

See Also: AX=7F05h - AH=F1h"EGA" - AH=F2h"EGA" - AH=F4h"E
