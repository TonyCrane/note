Alloy NTNX, MW386 - CHANNEL CONTROL - RELEASE BUFFER

AH = 10h
CL = 04h
AL = channel number


Return:
AL = status (00h-03h) (see #03951)

Notes: Unlocks buffer after received datagram has been processed. May not be invoked from within a hardware interrupt handler

See Also: AH=10h/CL=00h
