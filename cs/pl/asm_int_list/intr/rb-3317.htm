Novell NetWare - PRINT SERVICES - GET BANNER USER NAME

AX = B808h
ES:BX -> 12-byte buffer for user name


Return:
AL = status
00h successful

Desc: Get the user name which is printed on the banner page

Notes: This function is supported by Advanced NetWare 2.1+. The default name is the login name of the user

See Also: AX=B809h
