Novell NetWare - SYNCHRONIZATION SERVICES - RELEASE PHYS RECORD SET

AH = C3h

Desc: Unlock all currently-locked physical records in the log table, but do not remove them from the table

Note: This function is supported by NetWare 4.6+, Advanced NetWare 1.0+, and Alloy NTNX
