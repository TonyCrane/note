Novell NetWare - PRINT SERVICES - FLUSH LPT CAPTURE

AH = DFh
DL = 03h


Return:
AL = status
00h successful

Desc: This function closes the current capture file for the default LPT and starts printing it if it is not a permanent capture file

Notes: This function is supported by NetWare 4.0+, Advanced NetWare 1.0+, and Alloy NTNX; under NTNX, it sends a print break (see INT 17/AH=84h). If more data is sent to the LPT port after this call, a new capture file will be opeend
