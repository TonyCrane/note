Compaq Systempro - MULTIPROCESSOR END-OF-DISPATCH

AX = E10Eh
ES:BX -> start of 2nd processor's execution


Return:
AL = status
0Fh successful (halted)
00h failure (not halted)

See Also: AX=E00Fh - AX=E200h

Category: Vendor-specific BIOS Extensions - Int 15h - C
