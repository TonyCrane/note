CtrlAlt Associates VGAPAL.COM v1.00 - INSTALLATION CHECK

AX = CA00h
BX = 7670h ("vp")

Return:

Program: VGAPAL is a TSR supplied with STACKEY which makes VGA palette settings permanent across mode switches
