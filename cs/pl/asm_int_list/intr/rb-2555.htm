DOS 1+ - READ CHARACTER FROM STDAUX

AH = 03h


Return:
AL = character read

Notes: Keyboard checked for ^C/^Break, and INT 23 executed if detected. STDAUX is usually the first serial port

See Also: AH=04h - INT 14/AH=02h - INT E0/CL=03h
