PC/TCP kernel v2.02+ - net_alarm - SET TIMED ASYNCHRONOUS EVENT

AH = 2Bh
BX = network descriptor
CX,DX = time before alarm in milliseconds
DS:SI -> handler which will receive call (see #03467 at INT 61/AH=1Fh)
ES:DI = 32-bit cookie passed to handler


Return:
CF clear if successful
CF set on error
AX = error code (see #03319 at INT 61"FTP Software")

Notes: This function will cause a NET_AS_ALARM to be generated; it is intended for TSRs, etc. to regain control periodically. This function is not supported by Beame&Whiteside's BWPCTCP v3.0a shim

See Also: INT 61"FTP Software" - INT 61/AH=00h"PC/TCP"
