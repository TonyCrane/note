HUNTER 16 - SET BACKLIGHT POWER UP STATE

AH = 74h
AL = new level
01h off
02h unchanged
03h change level
BL = desired level (00h-7Fh)


Return:
AH = status

Desc: Select the backlight level to use when the Hunter is next turned on

See Also: AH=64h - AH=75h - INT 10/AH=64h - INT 10/AH=78h

Category: Vendor-specific BIOS Extensions - Int 15h - H
