ConcCP/M,DR Multiuser DOS - C_ATTACH - ATTACH TO DEFAULT CONSOLE

CL = 92h


Return:
AX = status

Notes: Also supported by REAL/32. If the console is currently owned by another process, this function waits until the console is available
