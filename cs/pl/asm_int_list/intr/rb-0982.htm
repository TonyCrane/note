I1541 - RESET ALL DEVICES

AX = 4003h


Return:
Nothing

Desc: Send a reset pulse of 100ms to all CBM devices

Note: It is necessary to wait about 2 seconds after reset before executing other instructions

See Also: AX=4000h - AX=4004h

Category: TSR Libraries - Int 14h - I
