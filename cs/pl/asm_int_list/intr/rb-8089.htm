Alloy MW386 - UNLOCK SEMAPHORE FOR USER

AH = 42h
AL = user ID
DS:DX -> ASCIZ semaphore name


Return:
AL = status
00h successful
01h invalid function
03h unable to unlock semaphore

See Also: AH=02h - AH=41h"MW386" - AH=42h"NTNX"
