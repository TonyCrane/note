TEXTCAP 2.0 - UNINSTALL

AX = 4253h


Return:
AX = segment of resident code

Notes: The uninstall code does not check whether interrupt vectors have been chained by other programs. The caller must free the main memory block (using the returned segment)

See Also: AX=4252h - AX=4254h
