Atari Portfolio - GET/SET CLOCK TICK SPEED

AH = 1Eh
AL = function
00h get speed
01h set speed
BX = clock tick speed
0000h tick every 128 seconds
0001h tick every second


Return:
If function 00h
BX = clock tick speed

Note: The faster tick speed also uses more power

See Also: AH=00h"Portfolio" - AH=1Ch"Portfolio" - AH=1Fh"Portfolio"
