Interrups Categories Contents

CauseWay - SetDOSTrans - SET DOS TRANSFER BUFFER

AX = FF26h
BX = real-mode segment of new transfer buffer
ECX = new transfer buffer size
DX = protected-mode selector for new transfer buffer


Return:
Nothing

Note: The specific buffer must be located in conventional memory; only the first 64K will be used

See Also: AX=FF25h
