G8BPQ v4.00+ - HOST MODE - GET BUFFER COUNTS FOR STREAM

AH = 07h
AL = stream number (01h-40h)


Return:
BX = number of pending receive frames
CX = number of unacknowledged sent frames
DX = number of buffers available

See Also: AH=02h"G8BPQ" - AH=03h"G8BPQ"
