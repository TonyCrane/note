SpaceManager - INSTALLATION CHECK

AX = C300h
DX = 0000h


Return:
AL = FFh if any SpaceManager programs installed
BX = 6F73h
CX = 6F68h

Program: SpaceManager is an enhancement for MS-DOS DoubleSpace by Vertisoft Systems, Inc. 

Range:
AH=C3h to AH=FFh, selected by scanning for first free multiplex number

See Also: AX=C300h/BX=4F53h

Category: File/Disk Compression - Int 2Fh - S
