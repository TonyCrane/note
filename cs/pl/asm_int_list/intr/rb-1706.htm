DESQview v2.23+ - DOSUSER - GET HANDLE OF TASK CURRENTLY USING DOS

AX = DE1Fh


Return:
BX = segment of task handle or 0000h if no tasks are using DOS

Note: This API call may be made from within a hardware interrupt handler

See Also: AX=DE13h - INT 21/AH=34h
