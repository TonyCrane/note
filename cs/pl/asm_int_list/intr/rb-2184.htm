Alloy NTNX, MW386 - ADD PRINTER TO SPOOLER

AH = 89h
AL = mode
00h NTNX compatibility
DX = NTNX printer number (see #00643)
01h MW386
DX = MW386 printer number


Return:
AL = status (see #00642)

Note: The specified printer is added to the spooler's list of available printers

See Also: AH=88h - AH=8Bh

Category: Network - Int 17h - A
