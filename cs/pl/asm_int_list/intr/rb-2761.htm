DIET v1.43e - TSR INSTALLATION CHECK

AX = 37D0h
BX = 899Dh ('DI' + 'ET')


Return:
AL = FFh if not present as TSR (default return value from DOS)
AX = 0000h if installed as a TSR
CX = 899Dh
DX = version ID

Program: DIET is a transparent file compressor/decompressor by Teddy Matsumoto
