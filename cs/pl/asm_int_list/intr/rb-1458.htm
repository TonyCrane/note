Chips & Technologies '65530' BIOS - POST INITIALIZATION NOTIFICATION

AX = 5F31h


Return:
Nothing

Desc: This function is called after the video BIOS completes power-up initialization and just prior to displaying the sign-on message

See Also: AX=5F33h - AX=5F35h - INT 10/AX=5F50h
