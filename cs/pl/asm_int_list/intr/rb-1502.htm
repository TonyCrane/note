HUNTER 16 - CONTROL HOT KEY POWER OPTION

AH = 7Ch
AL = 00h prevent HOT key power option
else allow HOT key power option

Desc: Allow or prevent changing the power options

See Also: AH=7Bh - AH=7Dh

Category: Vendor-specific BIOS Extensions - Int 15h - H
