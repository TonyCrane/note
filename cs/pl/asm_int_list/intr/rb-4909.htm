IPLAY v1.00b - INSTALLATION CHECK

AX = 60FFh
DL = 00h (function number)
BX = 5344h ('SD')
CX = 4D50h ('MP')


Return:
AX = 4F4Bh ('OK') if installed

Program: IPLAY is the Inertia Player by Prime and Excalibur for .MODules (digitized music files)

Note: In version 1.00b, any value for DL except 01h invokes this function

See Also: AX=60FFh/DL=01h

Category: Sound/Speech - Int 2Fh - I
