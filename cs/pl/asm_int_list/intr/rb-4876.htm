ODIHLP.EXE - INSTALLATION CHECK

AX = 5100h


Return:
AL = FFh if installed
BX = 0000h
DX:SI -> signature string "ODI$HLP$"

Program: ODIHLP is a real-mode helper allowing the Windows for Workgroups 3.11 protected-mode NDIS3 protocol to work with real-mode ODI drivers and LSL.COM

Note: The returned signature string might be the first field of a structure

See Also: AX=C000h"LSL.COM"
