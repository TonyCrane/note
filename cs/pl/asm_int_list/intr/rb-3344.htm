Novell NetWare - SYNCHRONIZATION SERVICES - EXAMINE SEMAPHORE

AX = C501h
CX:DX = semaphore handle


Return:
AL = status
00h successful
CX = semaphore value (-127 to 127)
DL = count of processes which have the semaphore open
FFh invalid handle

Note: This function is supported by NetWare 4.6+, Advanced NetWare 1.0+, and Alloy NTNX

See Also: AX=C500h"NetWare" - AX=C502h - AX=C504h - AX=F220h/SF=01h
