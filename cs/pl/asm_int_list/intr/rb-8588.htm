Int E0/CL=BAh

Interrups Categories Contents

REAL/32 - A_TIMEOUT - SET AUX DEVICE TIMEOUT

CL = BAh
DX = timeout value in system ticks (see #04048)
0000h-FFEFh = new number of system ticks
FFFDh = get current value without changing
FFFEh = start timeout count
FFFFh = never timeout


Return:
AX = status, or current timeout value if DX=FFFDh on entry
may destroy SI,DI

See Also: INT E0/CL=B9h

Category: Other Operating Systems - Int E0h - R
