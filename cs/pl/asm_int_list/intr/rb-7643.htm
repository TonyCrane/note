HP Vectra EX-BIOS - F_SYSTEM SF_INIT - START INITIALIZING DRIVER

AX = 0200h
BP = driver ID (see #03741)
BX = "last-used DS" value from HP global data area (see #03748)


Return:
AH = status (see #03744)
BX = new "last-used DS" value
BP,DS destroyed

See Also: AH=00h"HP Vectra" - AX=0202h - AH=0Ch"F_INS_FIXOWNDS"
