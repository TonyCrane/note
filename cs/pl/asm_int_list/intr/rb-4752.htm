Windows 3.x - SWITCHING DOS TO FOREGROUND

AX = 4006h

Notes: Called by Windows when the DOS box is about to be placed in the foreground and the video driver should restore any necessary state information (this may be called only in Standard mode). This callout is not made unless the application has first called AX=4000h

See Also: AX=4002h - AX=4005h
