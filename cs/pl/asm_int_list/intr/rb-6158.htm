VIDEO - RELOCATED DEFAULT INT 10 VIDEO SERVICES (EGA,VGA)

Desc: Contains the address of the original INT 10 handler which an EGA+ video adapter replaces with its own on-board BIOS code

See Also: INT 10/AH=00h - INT 10/AH=0Eh - INT 6D"VGA"

Note: Not used by PS/2 built-in VGA or XGA

Category: Video - Int 42h - V
