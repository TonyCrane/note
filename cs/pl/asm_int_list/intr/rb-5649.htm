NDOS - AWAITING USER INPUT

AX = E44Eh
BX = condition
0000h NDOS is ready to display prompt
0001h NDOS has displayed the prompt, about to accept user input


Return:
Handler must preserve SI, DI, BP, SP, DS, ES, and SS

See Also: AX=E44Dh
