Super PC-Kwik v3.20+ - GET RESIDENT CODE SEGMENT

AH = A0h
SI = 4358h


Return:
AX = segment of resident code

Note: PC Tools PC-Cache 5.x and Qualitas Qcache 4.00 are OEM versions of Super PC-Kwik, and thus support this call (note that PC-Cache v5.5 corresponds to PC-Kwik v3.27)
