Advanced Power Management v1.2 - TURN OFF SYSTEM

AX = 5307h
CX = 0003h
BX = 0001h (device ID for all power-managed devices)


Return:
After system is resumed
CF clear

Notes: If supported by the system's power supply, turns off the system power
