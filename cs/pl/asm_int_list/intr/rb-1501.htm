HUNTER 16 - CONTROL HOT KEY

AH = 7Bh
AL = 00h prevent HOT key
else allow HOT key

Desc: Allow or prevent the HOT key function which is used to examine and change the Hunter setup

See Also: AH=71h - AH=7Ah - AH=7Ch

Category: Vendor-specific BIOS Extensions - Int 15h - H
