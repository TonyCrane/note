HP Vectra EX-BIOS - F_STR_GET_INDEX - GET INDEX FOR STRING

AH = 48h
BP = 0012h (driver ID of V_SYSTEM)
ES:DI -> ASCIZ string


Return:
AH = status (00h,other) (see #03744 at AH=00h"HP Vectra")
BP,DS destroyed
---if successful---
BX = index for the given string
