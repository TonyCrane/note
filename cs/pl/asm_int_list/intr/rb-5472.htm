InterWave Game API - SUSPEND PROGRAM

AX = CD03h
BX = installed program identifier
CX = InterWave section to be released
0000h all
0001h synthesizer
0002h Codec


Return:
AL = status
00h successful
BX = base port
CL = IRQ
CH = DMA channel (Codec only)
else failed
