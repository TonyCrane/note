Vanderaart TEXT WINDOWS - PUT CHARACTER IN WINDOW

AX = 0002h
BL = character
BH = attribute
DL = column
DH = row
DI = window handle


Return:
AX = status
0000h if successful
FFFFh if outside window

See Also: AX=0000h
