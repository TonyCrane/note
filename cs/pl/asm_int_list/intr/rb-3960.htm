Novell NetWare v4+ - NDS GET SERVER NAME AND ADDRESS

AX = F268h subfn 35h
CX = length of request buffer in bytes
DX = length of reply buffer in bytes
DS:SI -> request buffer (see #02471)
ES:DI -> reply buffer (see #02472)


Return:
AL = status (see #02860 at INT 2F/AX=7A20h/BX=0000h)
reply buffer filled

Note: The length specified in CX must be exactly 13 bytes more than the length field at offset 09h in the request buffer for this function to be successful

See Also: AX=F268h/SF=16h 


Format of NetWare "NDS Get Server Name and Address" request buffer:

Offset  Size    Description     (Table 02471)
00h    BYTE    02h
01h    DWORD   ??? (FFFFFFFFh)
05h    DWORD   ??? (00000202h)
09h    DWORD   length (0000000Ch)
0Dh    DWORD   ??? (00000000h)

15h    DWORD   ??? (00000400h)
