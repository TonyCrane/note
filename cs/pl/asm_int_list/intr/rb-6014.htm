Mouse Systems MOUSE DRIVER v7.01+ - TOGGLE IGNORE ACCELERATION CMDS

AX = 0044h
CX = CDEFh


Return:
AX = new state of "Ignore Application Acceleration Commands" flag

Note: This function is also supported by the Genius Mouse 9.06 driver

See Also: AX=0045h
