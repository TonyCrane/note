CauseWay v1.3 - GetSels - ALLOCATE MULTIPLE SELECTORS

AX = FF29h
CX = number of selectors to allocate


Return:
BX = base selector

Desc: Allocate multiple contiguous selectors, initializing each to have a base address and limit of zero

See Also: AX=FF00h - AX=FF04h
