VIDEO - Paradise VGA, AT&T VDC600 - SET NON-VGA OPERATION

AX = 007Fh
BH = 01h


Return:
AL = 7Fh if successful (AT&T VDC600)
BH = status (Paradise/Dell)
00h invalid function
7Fh successful

Note: Color modes (0,1,2,3,4,5,6) will set non-VGA CGA operation, monochrome mode 7 will set non-VGA MDA/Hercules operation

See Also: AX=007Fh/BH=00h - AX=007Fh/BH=02h
