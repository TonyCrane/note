INT16.COM - INSTALLATION CHECK

AX = E337h


Return:
AX = 0013h if installed

Program: INT16.COM is an INT16 keyboard BIOS replacement for INT 16/AH=00h-02h,10h-12h (based on K3) written by Martin Gerdes and published in c't 05/1990.

See Also: AX=ED58h
