FGDRIVER v4.02 - FG_TEXTC - DISPLAY STRING OF CHARACTERS (CLIPPED)

AX = 00C9h
CX = length of string
ES:BX -> string

Notes: The FGDRIVER functions are rearranged with each major release, but their parameters do not change (see #03474). The string is displayed starting at the text cursor position using the current text attribute (text modes) or color index (graphics modes), showing only the portion within the current clipping window. The text cursor position is updated after this call

See Also: AX=00C8h
