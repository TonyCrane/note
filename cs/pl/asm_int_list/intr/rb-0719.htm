Int 13/AX=4B01h

Interrups Categories Contents

Bootable CD-ROM - GET STATUS

AX = 4B01h
DL = drive number
DS:SI -> empty specification packet (see #00281)


Return:
CF clear if successful
CF set on error
AX = return codes
DS:SI specification packet filled

Note: Same as AX=4B00h, but does not terminate emulation

See Also: AH=48h - AH=4Ah - AX=4B00h - AH=4Ch - AH=4D

Category: Disk I/O Enhancements - Int 13h - B
