Int 1F/AH=FCh/BL=02h

Interrups Categories Contents

C&T SuperState BIOS - GET ALARM TIME

AH = FCh
BL = 02h


Return:
CH = hour by BCD
CL = min by BCD
DH = sec by BCD

See Also: AH=FCh/BL=00h - AH=FCh/BL=03h - AH=FCh/BL=04h

Category: Vendor-specific BIOS Extensions - Int 1Fh - C
