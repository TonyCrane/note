BW-TCP - HW DRIVER (ETHDEV.SYS) - ALLOCATE AND MAP EMS FOR DRIVER

AH = 18h


Return:
CF clear if successful
CF set on error
AL = error code

Range:
INT 4C to INT FB, selected by configuration

Notes: Calls function 17h after EMS allocated and mapped. The Beame&Whiteside TCP/IP protocol stack uses two consecutive interrupts (62h and 63h by default); the BW-NFS client uses a third consecutive interrupt (64h by default) if it is loaded
