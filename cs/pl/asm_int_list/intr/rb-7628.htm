VIDEO - many VGA - VIDEO BIOS ENTRY POINT

Desc: Points at the original INT 10 entry point set up by the VGA BIOS

Notes: Used by IBM, ATI VGA Wonder, Paradise, Video7, and NCR, and many others. The Diamond Stealth64 BIOS uses this vector to store the INT 42 vector of the inactive video (see INT 10/AH=12h/BL=35h)

See Also: INT 10/AH=00h - INT 10/AH=0Eh - INT 10/AH=12h/BL=10h

Category: Video - Int 6Dh - V
