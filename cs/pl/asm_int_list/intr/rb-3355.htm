Novell NetWare - TRANSACTION TRACKING SYSTEM - ABORT TRANSACTION

AX = C703h


Return:
CF clear if successful
AL = 00h
CF set on error
AL = error code (FDh-FFh) (see #01809)

Note: This function is supported by NetWare 4.0+ and Advanced NetWare 1.0+

See Also: AX=C700h"NetWare" - AX=C701h - AX=C704h - AX=F222h/SF=03h
