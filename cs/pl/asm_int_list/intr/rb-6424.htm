HP 95LX/100LX/200LX GRAPHICS PRIMITIVES - SET LINE TYPE

AH = 0Bh
CX = new line type

Note: The line type specifies 16 bits which are repeated over and over while drawing the pixels of a line

See Also: AH=09h - AH=0Ah
