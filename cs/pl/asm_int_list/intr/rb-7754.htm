Int 6F/AH=17h

Interrups Categories Contents

10NET v3.3+ - MOUNT

AH = 17h
AL = local drive number (0=A:)
BL = remote drive letter or '1'..'3' for LPTn or '4' or '5' for COMx
DS:DX -> node ID, 12 bytes blank-padded


Return:
CF clear if successful
CF set on error
AX = error code (see #03740)

See Also: AH=18h
