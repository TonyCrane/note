COLAP - INSTALLATION CHECK

AX = C700h
CX = 434Ch  "CL"


Return:
AL = FFh if installed

Program: Colap.com is a freeware TSR by Eric Meyer which controls contrast and brightness of color laptops by changing the VGA palette colors; for Toshiba laptops there are more hotkeys for shutting off display and harddisk.
