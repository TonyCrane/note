LANtastic v4.1+ - SET EXTENDED QUEUE ENTRY

AX = 5FA9h
BX = handle of opened queue entry
DS:SI -> queue entry (see #01732)


Return:
CF clear if successful
CF set on error
AX = error code

Note: Functions exactly the same as AX=5FA1h except the spooled filename is also set. This call supports direct despooling.

See Also: AX=5FA1h
