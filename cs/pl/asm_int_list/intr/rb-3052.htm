STARLITE architecture - REMOVE GLOBAL NETWORK NAME OF OBJECT

AX = 5F08h
DS:SI -> ASCIZ network name (not local name) of object to unshare


Return:
CF set on error
AX = error code (see #01680 at AH=59h/BX=0000h)
CF clear if successful

See Also: AX=5F07h"STARLITE"

Category: Other Operating Systems - Int 21h - S
