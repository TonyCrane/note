MultiDOS Plus IODRV - INITIALIZE PORT

AH = 04h


Return:
Port initialized; if Hayes-compatible modem, a connection has been
established

Note: The port number is stored at offset BEh in the Task Control Block (see #00456 at INT 15/AH=13h"MultiDOS")

See Also: AH=00h - AH=05h"MultiDOS" - AH=20h"MultiDOS" -
