MultiDOS Plus - INITIALIZE PORT

AH = 20h
AL = port parameters (see #00300 at AH=00h"SERIAL")
DX = port number (0-3)


Return:
AH = status
00h successful
41h no such port
64h monitor mode already active

See Also: AH=00h"SERIAL" - AH=04h"MultiDOS" - AH=21h"MultiDOS" - Category: Serial I/O - Int 14h - M
