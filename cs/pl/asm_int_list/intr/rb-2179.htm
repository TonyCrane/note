Alloy NTNX, MW386 - GENERATE PRINT BREAK

AH = 84h
AL = mode
00h NTNX compatibility
02h MW386 v2+
BX = logical device number
00h-03h = LPT1-LPT4
04h-07h = COM1-COM4

Note: Closes spool file and tells spooler to queue the print job (LPT1 only under MW386 in NTNX compatibility mode)

Category: Network - Int 17h - A
