VIDEO - AX PC - GET/SET SCROLL MODE

AX = 8200h
BL = new scroll mode or FFh to get current mode
00h dynamic, 01h software


Return:
AL = scroll mode (current mode if BL=FFh, previous mode otherwise)

See Also: AH=06h - AH=07h

Category: Japanese - Int 10h - V
