DOS 3.3+ DISPLAY.SYS internal - SET ACTIVE CODE PAGE

AX = AD01h
BX = new code page (see #01757 at INT 21/AX=6602h)


Return:
CF clear if successful
AX = 0001h
CF set on error (unsupported code page)
AX = 0000h

See Also: AX=AD02h"DISPLAY"
