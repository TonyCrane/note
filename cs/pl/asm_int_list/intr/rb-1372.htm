HP 100LX/200LX - SET DISPLAY CONTROL STATUS

AH = 45h
AL = new display control status

bit 0:
DISPCTL flag +C instead of -C

bit 1:
DISPCTL flag +K instead of -K

See Also: AH=46h"HP" - AH=47h"HP"
