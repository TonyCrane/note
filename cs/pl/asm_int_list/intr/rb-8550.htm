Int E0/CL=8Ch

Interrups Categories Contents

ConcCP/M,DR Multiuser DOS - Q_CWRITE - CONDITIONALLY WRITE MSG QUEUE

CL = 8Ch
DS:DX -> queue parameter block (QPB) (see #04044,#04045)


Return:
AX = status (0000h success, FFFFh failure)
CX = error code (see #04021)

Note: Also supported by REAL/32

See Also: INT E0/CL=8Ah - INT E0/CL=8Bh

Category: Other Operating Systems - Int E0h - C
