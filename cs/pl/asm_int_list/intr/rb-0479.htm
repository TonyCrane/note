VIDEO - Compaq Extensions - SELECT INTERNAL MONITOR

AX = BF01h


Return:
Nothing

Desc: Specify that the internal monitor become the active monitor

Note: All registers preserved and the external monitor is blanked

See Also: AX=BF00h - AX=BF12h - AH=12h/BL=35h

Category: Vendor-specific BIOS Extensions - Int 10h - V
