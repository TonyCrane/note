Novell NetWare, Alloy NTNX - RELEASE RECORD (FCB)

AH = C0h
DS:DX -> non-extended FCB (see #01345 at AH=0Fh)
BX:CX = offset


Return:
AL = error code (see #01807)

Notes: Unlocks record but does not remove it from log table. This function was added in NetWare 4.6, but was removed some time prior to Advanced NetWare 2.15, and is no longer listed in current Novell documentation

See Also: AH=BDh - AH=BFh - AH=C1h"NetWare" - AH=C3h
