HUNTER 16 - GET PROTOCOL NAME

AH = 28h
AL = protocol handle
DS:BX -> 8 character buffer for protocol name


Return:
AH = status
00h successful
DS:BX buffer filled with the protocol name
FFh failed

See Also: AH=27h"HUNTER" - AH=29h"HUNTER"
