IBM/Yale EBIOS SERIAL I/O - READ CHARACTER, NO WAIT

AH = FCh
DX = port (00h-03h)


Return:
AH = RS232 status bits (see #00304 at AH=03h)
AL = character

See Also: AH=02h - AH=0Ch - AX=FF02h
