AMI BIOS - BIOS-FLASH Interface - RESTORE INTERNAL CACHE STATUS

AX = E00Ch
ES:DI -> buffer containing internal cache status (minimum 4Kbytes)


Return:
CF clear if successful
AL = FAh
CF set on error

Note: Always returns error if the hardware does not contain internal cache or this call is made in protected mode

See Also: AX=E000h - AX=E00Bh

Category: Vendor-specific BIOS Extensions - Int 16h - A
