HP OmniShare - Pen Driver - SET THE COMMUNICATION LED STATE

AX = BA11h
BL = new LED state (00h steady, 01h flashing)


Return:
CF clear if successful
CF set on error

Note: This function is for use by communication software to give visual feedback of active communications even if the display is in standby mode.

See Also: AX=BA10h - AX=BA24h

Category: Non-Traditional Input Devices - Int 15h - H
