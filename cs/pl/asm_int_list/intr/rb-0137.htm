VIDEO - TEXT-MODE CHARGEN - LOAD ROM MONOCHROME PATTERNS (PS,EGA,VGA)

AX = 1101h
BL = block to load


Return:
Nothing

Notes: (see also AX=1100h). This function will cause a mode set, completely resetting the video environment, but without clearing the video buffer. The "monochrome" patters are 8x14 pixels in size

See Also: AX=1100h - AX=1102h - AX=1103h - AX=1104h -

See Also: MEM 0040h:0084h 

Index:
Text mode;font|text mode;screen rows
