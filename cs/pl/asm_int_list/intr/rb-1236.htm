TopView - SEND MESSAGE - READ - WAIT FOR TIMER TO EXPIRE

AH = 12h
BX = 0400h.

STACK:
DWORD timer's handle


Return:
After timer expires.

STACK:
DWORD time in 1/100 sec after midnight when timer expired

See Also: AH=12h/BH=0Ah - AH=12h/BH=84h
