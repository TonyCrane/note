Alloy NTNX, MW386 - GET MAXIMUM PACKET SIZE

AH = 15h
CL = 03h
DX:DI -> WORD for return value


Return:
Buffer WORD filled with maximum packet size (4096 for MW386)

See Also: AH=15h/CL=02h
