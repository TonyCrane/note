DOS 3.0+ ASSIGN - GET DRIVE ASSIGNMENT TABLE

AX = 0601h


Return:
ES = segment of ASSIGN work area and assignment table

Note: The 26 bytes starting at ES:0103h specify which drive each of A: to Z: is mapped to. Initially set to 01h 02h 03h....

See Also: AX=0600h - AX=AF14h"WinDOS"
