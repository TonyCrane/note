KAOSHIDE - INSTALLATION CHECK

AX = E000h


Return:
AL = FFh if installed

Program: KAOSHIDE ('Hidden Kaos') is a PD joke TSR which randomly capitalizes alphabetic keys, written by Philip Maland. Although not a virus, v2.0 uses some viral-like techniques to hide itself from memory-reports by reducing DOS memory size.

Category: Joke Programs - Int 2Fh - K
