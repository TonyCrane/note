PCED/ATTRIB - INSTALLATION CHECK

AX = 48C2h
BL = 00h


Return:
AX = 0000h if installed
BX = ATTRIB resident segment

Program: ATTRIB is a resident file attribute changer included as part of the PCED package by Cove Software

Note: Chains if BL <> 00h on entry

See Also: AX=48C0h - AX=48C1h - AX=48C3h

Category: Resident Utilities - Int 2Fh - P
