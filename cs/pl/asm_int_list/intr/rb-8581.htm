Int E0/CL=B3h

Interrups Categories Contents

REAL/32 - MP_ALLOC - ALLOCATE PHYSICAL MEMORY

CL = B3h
DX = number of 4K pages to allocate


Return:
AX = number of first 4K page allocated, or FFFFh on error

Note: Memory allocated with this function is not automatically freed when the process terminates

See Also: INT E0/CL=B4h - INT E0/CL=B5h

Category: Other Operating Systems - Int E0h - R
