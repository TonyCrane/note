DESQview v2.20+ - GET ERROR HANDLING

AX = DE16h


Return:
BL = current mode
00h always post system error
01h return carry flag set on failed mailbox writes
02h return CF set on failed mailbox writes and NULL on failed
GETMEM calls

See Also: AX=DE15h
