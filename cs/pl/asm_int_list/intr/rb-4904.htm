IOMEGA DRIVERS - LOCK MEDIA IN DRIVE

AX = 5711h
BX = program ID??? (0201h used by GUEST.EXE)
DX = 496Fh ('Io')


Return:
CF clear if successful (storage medium in drive)
AX = new lock count
CF set on error (drive empty)

See Also: AX=5700h - AX=5701h - AX=5710h - AX=5712h

Category: Disk I/O Enhancements - Int 2Fh - I
