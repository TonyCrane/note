VMiX v2+ - sys_bitblt - PUT GRAPHICAL OBJECT AT CURSOR POSITION

AH = 1Ch.

STACK:
WORD    AND/OR pixel with background (00h = OR, 01h = AND)
DWORD   pointer to object bitmap
WORD    object width in pixels
WORD    object height in pixels


Return:
Nothing

See Also: AH=1Dh"VMiX"
