PCMCIA Socket Services v1.00 - READ ERROR DETECTION CODE

AH = 9Ch
AL = adapter number
BH = EDC generator


Return:
CF clear if successful
AH destroyed
DL/DX = computed checksum or CRC
CF set on error
AH = error code (01h,04h) (see #00656)

See Also: AH=95h - AH=96h - AH=98h - AH=99h - Category: Expansion Bus Bios - Int 1Ah - P
