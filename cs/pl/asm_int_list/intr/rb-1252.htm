Int 15/AH=12h/BH=0Dh

Interrups Categories Contents

TopView - SEND MESSAGE - CLOSE - CLOSE OBJECT

AH = 12h
BH = 0Dh
BL = object
00h handle is DWORD on top of stack

timer:
Close

keyboard:
Detach from window and discard queued input

pointer:
Stop taking input

panel:
Close

mailbox:
Close, unlock, and discard any pending messages
02h close given task's mailbox (task's handle on top of stack)
03h close task's default mailbox
04h close KEYBOARD object (handle on top of stack)
05h close task's default KEYBOARD
06h close given task's OBJECTQ (task's handle on top of stack)
07h close current task's OBJECTQ


Return:
STACK popped if handle passed on stack

Notes: When an OBJECTQ is closed, each object in the OBJECTQ is sent an ERASE message (AH=12h/BH=0Eh). When a panel object is closed, the panel file and any panels currently in use are freed; window and keyboard objects created by APPLY are not affected, but field mode input ceases. Open but idle timer objects consume a small amount of CPU time

See Also: AH=12h/BH=0Ch - AH=12h/BH=0Eh - AH=12h/BH=14h"LOCK"
