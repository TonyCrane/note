DR DOS 5.0-6.0 - DETERMINE DOS TYPE

AX = 4412h
CF set


Return:
CF set if not DR DOS
AX = error code (see #01680 at AH=59h/BX=0000h)
CF clear if DR DOS
DX = AX = version code (see #01580)

Note: This obsolete call, which is no longer supported in Novell DOS 7, is identical to AX=4452h

See Also: AX=4452h

Category: Other Operating Systems - Int 21h - D
