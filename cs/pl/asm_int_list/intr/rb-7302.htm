TI Professional PC - SYSTEM INFORMATION (NOT A VECTOR!)

Desc: The low word of this vector contains the system memory size in paragraphs; the third byte contains the number of outstanding interrupt requests, and the fourth byte contains a description of the installed drive types (see #03517)

See Also: INT 60"TI Professional PC" - INT 67"TI Professional" 


Bitfields for TI Professional drive type information:
