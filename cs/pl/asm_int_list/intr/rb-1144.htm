MultiDOS Plus - GIVE UP TIME SLICE

AH = 00h


Return:
Nothing

Note: If issued by the highest-priority task while MultiDOS is using priority-based rather than round-robin scheduling, control will be returned to the caller immediately

See Also: AH=03h"MultiDOS" - AX=1000h
