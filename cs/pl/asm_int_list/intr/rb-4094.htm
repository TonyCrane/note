TopWare Network OS v5.10+ - DELETE FILE FROM SPOOLING QUEUE

AX = FFCBh
CH = printer number (00h LPT1, 01h LPT2, 02h LPT3)
CL = printer server station address
DS:DX -> filename (12 bytes)


Return:
AL = status (00h successful, else failed)

See Also: AX=FFCAh
