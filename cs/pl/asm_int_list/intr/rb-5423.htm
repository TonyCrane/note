Communicating Applications Specification - ABORT CURRENT EVENT

AX = CB02h


Return:
AX >= 0: event handle of aborted event.

AX < 0:
Error code (see #03048)

Note: Termination could take up to 30 seconds

See Also: AX=CB08h - AX=CB10h
