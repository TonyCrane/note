DR Multiuser DOS - A_CATTACH - CONDITIONALLY ATTACH TO AUX DEVICE

CL = A7h


Return:
AX = status
0000h attached
FFFFh unable to attach

Desc: Attaches the default auxiliary device to the calling process if it is available

Notes: Does not block if the device is already in use. Also supported by REAL/32

See Also: INT E0/CL=A5h - INT E0/CL=A6h - INT E0/CL=A8h - Category: Other Operating Systems - Int E0h - D
