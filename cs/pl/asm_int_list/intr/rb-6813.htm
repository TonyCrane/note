PC/TCP kernel v2.05+ - net_releaseall - CLOSE ALL NON-GLOBAL DESCRS

AH = 09h


Return:
CF clear

Range:
INT 20 to INT E0, selected by configuration

Notes: This call performs function 08h on every non-global network descriptor. Global descriptors must be released individually.
