HP Vectra EX-BIOS - F_INQUIRE_FIRST - GET PHYSICAL DRIVER VECTOR

AH = 0Ah
BP = 002Ah (driver ID for V_SINPUT)


Return:
AH = status (see #03744)
BX = vector address for first HP-HIL device
BP,DS destroyed

Note: All seven HP-HIL devices use consecutive vectors in the HP_VECTOR_TABLE
