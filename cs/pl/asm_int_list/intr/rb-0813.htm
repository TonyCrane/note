Tandy 2000 - SERIAL - RESET COMM PORT

AH = 00h
AL = RS-232C parameters (see #00302)
DL = port number
DH = protocol

bit 0:
Use XON/XOFF on received data

bit 1:
Use XON/XOFF when transmitting


Return:
AH = line status (see #00304)
AL = modem status (see #00305)

Note: This interrupt is identical to INT 53 on the Tandy 2000
