NETWORK REDIRECTOR - WRITE TO REMOTE FILE

AX = 1109h
ES:DI -> SFT
SFT DPB field -> DPB of drive containing file
CX = number of bytes
SS = DOS DS
SDA DTA field -> user buffer


Return:
CF set on error
AX = DOS error code (see #01680 at INT 21/AH=59h/BX=0000h)
CF clear if successful
CX = number of bytes written
SFT updated

Notes: This function is called by the DOS 3.1+ kernel. PrintCache v3.1 PCACHE.EXE intercepts this function for SFTs where the Device Driver Header field points at PCACHE, but does not intercept any other network redirector functions

See Also: AX=1107h - AX=1108h - INT 21/AH=40h - INT 21/AX
