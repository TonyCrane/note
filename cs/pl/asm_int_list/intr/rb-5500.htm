MDEBUG display driver - RESTORE GRAPHIC DATA

AX = D003h


Return:
CF set on error
CF clear if successful
display restored to the mode it was in before calling AX=D002h and
the display memory is restored

Note: MDEBUG calls this function only once every popup session just before it exits to normal DOS.

See Also: AX=D000h - AX=D002h
