Common ISDN API (CAPI) v2.0 - CAPI_GET_MESSAGE

AL = 04h
AH = CAPI version number * 10 (14h for v2.0)
DX = application ID (see AH=01h)


Return:
AX = status (0000h,11xxh) (see #04070)
ES:BX -> message if successful (see #04071)

Range:
INT 00 to INT FF, selectable by program parameter

Notes: The caller is required to provide at least 512 bytes of stack space. The returned message may be overwritten by the next call to this function

See Also: AH=04h - INT F1/AL=03h

Category: Uncatagorized - Int F1h - C
