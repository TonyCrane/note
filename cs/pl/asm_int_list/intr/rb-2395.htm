PCI BIOS v2.0c+ - FIND PCI DEVICE (32-bit)

AX = B182h
CX = device ID (see #00735,#00742,#00743,#00873,#00875)
DX = vendor ID (see #00732 at AX=B102h)
SI = device index (0-n)

As for AX=B102h
