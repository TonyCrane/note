Novell NetWare - TRANSACTION TRACKING SYSTEM - INSTALLATION CHECK

AX = C702h


Return:
AL = status
00h not available
01h available
FDh available but disabled

Desc: Determine whether the default file server supports TTS

Note: This function is supported by NetWare 4.0+ and Advanced NetWare 1.0+

See Also: AX=C700h - AX=C701h - AX=C703h - AX=F222h/SF=00
