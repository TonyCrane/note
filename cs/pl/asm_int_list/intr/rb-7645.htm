MS Windows 3.0 - DOS APPLICATION SWITCH HOOK

AX = 0204h


Return:
AX = status
0000h switch is allowed
other switch not allowed

Note: Intercepting this call will allow a DOS application to ensure that Windows will not switch away from it.
