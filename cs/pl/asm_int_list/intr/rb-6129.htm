Int 3F

Interrups Categories Contents

Overlay manager interrupt (Microsoft LINK.EXE, Borland TLINK VROOMM)

Notes: INT 3F is the default, and may be overridden while linking. This vector is modified but not restored by Direct Access v4.0, and may be left dangling by other programs written with the same version of compiled BASIC

See Also: INT FE"OVERLAY"
