Int 14/AH=1Dh

Interrups Categories Contents

X00 FOSSIL - DEACTIVATE PORT

AH = 1Dh
DX = port number


Return:
None

Notes: This is a duplicate of AH=05h, so that AH=05h may be made compatible with the PS/2 BIOS in a future release. Ignored if the port was never activated with AH=04h or AH=1Ch

See Also: AH=05h"FOSSIL" - AH=1Ch
