DPMI 0.9+ - GET PAGE SIZE

AX = 0604h


Return:
CF clear if successful
BX:CX = page size in bytes
CF set on error
AX = error code (DPMI 1.0+) (see also #03143)
8001h unsupported, 16-bit host

BUG: The Borland C++ 3.1 DPMILOAD returns with CF clear but BX and CX unchanged
