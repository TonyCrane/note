Nanosoft MD386 - GET INTERNAL HANDLE TABLE

AX = 5D05h
BX = handle number
ES:DI -> 1024-byte buffer for handle table


Return:
AH = status (see #03648)
buffer filled if AH=00h

Note: Used for debugging purposes

See Also: AX=5D04h
