TelAPI - CHECK FOR BREAK SIGNAL

AH = 08h
AL = circuit number


Return:
CF clear if no breaks
AL = 00h
CX = 0000h
CF set if one or more breaks received

Note: This function also clears ??? flag

See Also: INT 14/AX=FF00h
