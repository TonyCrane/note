ARTICOM - GET INTERNAL SEND/RECEIVE VECTOR

AX = 8035h


Return:
ES:BX -> address of current send/receive routine

Note: This function returns the address of the routine which is called inside A-REDIR.EXE each time a character is received or sent on the active COM port.

See Also: AX=8000h - AX=8025h - INT 21/AH=35h

Category: Serial I/O - Int 14h - A
