DESQview v2.50+ - SEND MESSAGE 00h WITH ERROR RECOVERY

AH = 12h
BH = 80h

Note: This function is identical to AH=12h/BH=00h, except that DESQview will not pop up a "Programming Error" window, instead returning an error code in AL (see #00455)

See Also: AH=12h/BH=00h 



(Table 00455)
Values for DESQview error code:
00h    no error
01h    invalid values
02h    alias invalid
03h    handle valid but wrong type
04h    invalid handle

Category: DESQview/TopView and Quarterdeck Programs - Int 15h - D
