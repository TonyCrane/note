DOS 3.0+ PRINT - RESTART PRINT QUEUE AFTER STATUS READ

AX = 0105h


Return:
CF clear if successful
CF set on error
AX = error code (see #02594)

Desc: Restart PRINT's activities once an application finishes examining the print queue

See Also: AX=0104h

Category: Caches/Spoolers - Int 2Fh - D
