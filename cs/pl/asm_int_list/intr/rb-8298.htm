Media Vision PCM.COM - GET INTERNAL DMA BUFFER ADDRESS

SI = 8000h


Return:
DX:AX -> DMA buffer

Program: PCM.COM is a superset of the standard PCM driver which provides additional functions for fine control of the driver 

InstallCheck:
For the Media Vision PCM.COM "shark" functions, test for the
signature "PCM-SHARK" at offset 107h in the INT 94 handler's segment

See Also: SI=8001h - SI=8004h
