COURIERS.COM - CONFIGURE PORT

AH = 82h
AL = port number (1-4)
BX = speed (bps)
CX = bit flags

bit 0:
Enable input flow control

bit 1:
Enable output flow control

bit 2:
Use X.PC protocol (not yet implemented)


Return:
Nothing

See Also: AH=00h - AH=8Ch - INT 7A"X.PC"
