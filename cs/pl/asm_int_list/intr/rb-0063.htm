CPU-generated (80386+ native mode) - PAGE FAULT

Desc: This interrupt is generated in protected and virtual-86 modes on attempting to access a 4K memory page whose page table entry has the "present" bit cleared

Notes: Used to implement virtual memory--when the page fault occurs, the operating system can load the appropriate page from disk. A protected-mode program written in the Flat memory model will often generate this interrupt (typically reported as an Exception #14 or Exception 0E with a register dump) when dereferencing an uninitialized or corrupted pointer

See Also: INT 0B"CPU"

Category: CPU-Generated - Int 0Eh - C
