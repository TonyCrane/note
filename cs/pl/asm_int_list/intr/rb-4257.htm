DOS 3.1+ PRINT - GIVE PRINT A TIME SLICE

AX = 0080h


Return:
After PRINT executes

Notes: PRINT returns AL=01h if AH=00h but AL is not 80h on entry. This function is not supported by the Novell DOS 7 PRINT.COM
