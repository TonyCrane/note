FGDRIVER v4.02 - FG_MAPRGB - MAP COLOR COMPONENTS INTO PALETTE VAL

AX = 006Fh
BX = red component
CX = green component
DX = blue component


Return:
AX = mode-specific palette value corresponding to specified components

Notes: The FGDRIVER functions are rearranged with each major release, but their parameters do not change (see #03474 at AX=0000h). This function is only meaningful in 16-color graphics modes

See Also: AX=004Eh - AX=00B3h
