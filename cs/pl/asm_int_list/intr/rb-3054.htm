STARLITE architecture - DETACH FROM NETWORK DEVICE

AX = 5F0Ah
DS:DX -> ASCIZ name of device driver to detach from


Return:
CF set on error
AX = error code (see #01680 at AH=59h/BX=0000h)
CF clear if successful

See Also: AX=5F09h

Category: Other Operating Systems - Int 21h - S
