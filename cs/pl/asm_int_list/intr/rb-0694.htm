QUICKCACHE II v4.20 - SET LOCK POOL SIZE

AH = 35h
BX = number of sectors in lock pool


Return:
AX = status (0000h successful)

Desc: Specify the number of cache sector buffers which may be dedicated to data locked into the cache

See Also: AH=34h

Category: Caches/Spoolers - Int 13h - Q
