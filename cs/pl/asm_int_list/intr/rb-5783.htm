RTS Control TSR - INSTALLATION CHECK

AX = FEEFh


Return:
AX = EFFEh if installed
BX = port address

Program: RTS Control TSR is a utility by Michal Szokolo to lower the RTS signal on a COM port during disk accesses to avoid losing incoming data
