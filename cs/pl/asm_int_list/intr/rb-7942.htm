SK-UPPS Data Link Interface API - DLI 2.8+ - INTERRUPT REQUEST

AX = 0017h


Return:
AX = completion code (see #03846)

Notes: Not supported by all DLIs (will return AX=FFFFh, if not supported).. Must be called with interrupts disabled; when the client re-enables interrupts, an interrupt will be generated and reported to the client through an Event Upcall (see #03853 [EV_ISR_START/EV_ISR_END])

See Also: AX=0007h
