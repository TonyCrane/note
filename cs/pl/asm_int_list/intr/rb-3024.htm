DOS 3.1+ network - GET REDIRECTED PRINTER MODE

AX = 5D07h


Return:
DL = mode
00h redirected output is combined
01h redirected output in separate print jobs

Note: MS-DOS and DR DOS 3.41/5.0 simply push AX on the stack and call INT 2F/AX=1125h

See Also: AX=5D08h - AX=5D09h - INT 2F/AX=1125h
