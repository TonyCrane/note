BitFOSS - INSTALLATION CHECK

AH = 2Bh
CX = 6269h ('bi')
DX = 7446h ('tF')


Return:
AL = FFh if not installed
AL = 4Fh ('O') if bitFOSS is installed
CX = 5353h ('SS')
DX = 494Ch ('IL')
ES = resident code segment
ES:DI -> identification data

Program: BitFOSS is a revision 5 FOSSIL driver

See Also: AH=2Bh/CX=6269h"bitFOSSI" - INT 11/AH=BCh
