ARTICOM - DETACH ASYNCHRONOUS RESOURCE

AX = 8005h
DX = port to detach (COM1=0, COM2=1, ...)


Return:
AX = error code, if error (see #00360)

Note: Only a previously attached resource can be detached

See Also: AX=8000h - AX=8003h - AX=8004h
