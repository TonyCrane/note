BW-TCP - TCPIP.SYS - SET INTERNET ADDRESS???

AH = 26h
CL:CH:DL:DH = Internet address

Nothing

Range:
INT 4D to INT FC, selected by configuration

Note: This function sets a different variable than AH=25h returns
