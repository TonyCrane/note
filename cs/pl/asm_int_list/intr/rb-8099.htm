Interrups Categories Contents

Alloy MW386 - GET USER NAME

AH = A0h
DL = user number (FFh = caller)
DH = machine number for user
ES:DI -> 17-byte buffer for ASCIZ user name


Return:
CF set if invalid user number

See Also: AH=03h - AH=A1h
