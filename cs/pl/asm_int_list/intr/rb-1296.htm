MultiDOS Plus - CHECK IF MultiDOS FOREGROUND OR BACKGROUND

AH = 14h


Return:
AX = current state
0000h MultiDOS Plus command prompt is background task
0001h command prompt is foreground task

See Also: AH=0Bh"MultiDOS"
