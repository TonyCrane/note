Int 31/AX=FF23h

Interrups Categories Contents

CauseWay - RelMemDOS - RELEASE CONVENTIONAL MEMORY BLOCK

AX = FF23h
DX = initial selector for block


Return:
CF clear if successful
CF set on error
AX = DOS error code (see #01680 at INT 21/AH=59h/BX=0000h)

Note: All descriptors allocated for the block are freed

See Also: AX=FF21h - AX=FF22h - INT 21/AH=4Ah

Category: DOS Extenders - Int 31h - C
