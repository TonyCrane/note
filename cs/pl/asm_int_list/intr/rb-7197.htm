DESQview/X - SOCKET API

InstallCheck:
Test for the string "dvxunix" (yes, lowercase) at offset 9
from the interrupt handler start (see #03515)

Notes: Parameters are passed by patching (!) data field immediately following the entry point, as detailed below (see #03515); the preferred method for calling the socket API is via INT 15/AX=DE2Eh

See Also: INT 15/AX=DE2Eh - INT BE"DESQview" 

Index:
Installation check;DESQview/X socket interface


Format of DESQview/X socket interrupt handler entry:

Offset  Size    Description     (Table 03515)
00h  3 BYTEs   near jump or short jump + NOP to actual interrupt handler
03h    WORD    offset from following pointer for initial top of local stack
05h    DWORD   pointer to argument/stack block (see INT 15/AX=DE2Eh)
09h  7 BYTEs   signature "dvxunix"
