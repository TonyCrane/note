HARD DISK - CHECK IF DRIVE READY

AH = 10h
DL = drive number (80h = first, 81h = second hard disk)


Return:
CF set on error
CF clear if successful
AH = status (see #00234 at AH=01h)

See Also: AH=11h

Category: Bios - Int 13h - H
