VIDEO - Hercules GRAFIX - LEVEL - SELECT DRAWING FUNCTION

AH = 44h
AL = drawing function
00h clear pixels
01h set pixels
02h invert pixels


Return:
Nothing

Desc: Specify how graphics output will change the display
