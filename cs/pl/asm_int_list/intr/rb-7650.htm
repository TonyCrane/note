HP Vectra EX-BIOS - F_SYSTEM SF_SET_ATTR - SET DEVICE ATTRIBUTES

AX = 020Ch
BP = driver ID (see #03741)
BX,CX = new attributes
---for V_LTOUCH, V_LPOINTER---
BX = new logical width
CX = new logical height


Return:
AH = status (see #03744)
BP,DS destroyed
