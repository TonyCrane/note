Int 15/AX=D600h/BL=01h

Interrups Categories Contents

Later PS/2s - WRITE BOOT DEVICE ID

AX = D600h
BL = 01h
DX = device ID


Return:
CF clear on success
AH = 00h
CF set on error
AH = status (86h for function not supported)

See Also: AX=D600h/BL=00h - AX=D601h/BL=01h
