DJGPP GO32.EXE DOS EXTENDER - RELOCATED IRQ3

Notes: This vector is overwritten when GO32 starts but is not restored by early versions of the extender. The newest versions of GO32 dynamically allocate the vectors used for the relocated IRQs, much as DESQview does (see INT 50"DESQview")
