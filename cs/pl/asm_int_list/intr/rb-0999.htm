HP Vectra EX-BIOS - F14_INQUIRE - INSTALLATION CHECK

AX = 6F00h
BX <> 4850h (usually set to 0000h for simplicity)


Return:
BX = 4850h ("HP") if HP Extended BIOS serial port extensions available
AX destroyed

Note: Supported by original HP Vectra AT and by ES/QS/RS series Vectras

See Also: AX=6F01h - AX=6F02h - AX=6F03h - AX=6F04h -

See Also: INT 17/AX=6F00h - INT 33/AX=6F00h
