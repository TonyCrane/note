FOSSIL - REMOVE AN EXTERNAL APPLICATION FUNCTION

AH = 7Fh
AL = code assigned to external application
ES:DX -> entry point


Return:
AX = 1954h
BL = code assigned to application (same as input AL)
DH = status
00h failed
01h successful

See Also: AH=7Eh

Category: Serial I/O - Int 14h - F
