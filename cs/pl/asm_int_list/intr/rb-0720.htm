Bootable CD-ROM - INITIATE DISK EMULATION AND BOOT

AH = 4Ch
AL = 00h
DS:SI -> specification packet (see #00281)


Return:
Never, if successful
CF set (error while attempting to boot)
AX = error codes

See Also: AH=48h - AH=4Ah - AX=4B00h - AH=4Dh
