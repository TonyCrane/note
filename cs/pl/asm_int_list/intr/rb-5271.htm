CATC USB4DOS Device Manager - HUB DEVICE ATTACHED

AX = BC00h
BL = 03h
BH = address and device type

bits 7-1:
USB address of hub

bit 0:
Device speed
=0 low-speed device
=1 full-speed device.
CX:DX -> callback function


Return:
AX = USB address of attached device, or 0000h on failure

See Also: AX=BC00h/BL=04h - AX=BC00h/BL=00h
