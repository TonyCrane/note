Atari Portfolio - SET PERIPHERAL ID BYTE

AH = 1Bh
AL = which peripheral to set
00h serial
01h parallel
DL = peripheral ID


Return:
Nothing

Desc: Specify that a peripheral is compatible with the BIOS serial or parallel services

See Also: AH=1Ah"Portfolio" - AH=1Ch"Portfolio"

Category: Vendor-specific BIOS Extensions - Int 61h - A
