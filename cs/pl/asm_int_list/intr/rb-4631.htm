Windows95 - CLOSE-AWARENESS - ENABLE/DISABLE CLOSE COMMAND

AX = 168Fh
DH = 00h
DL = new state
00h disabled
01h enabled


Return:
AX = status
0000h successful
else failed

Desc: Enable or disable the system menu Close command for an application

See Also: AX=168Fh/DH=01h - AX=168Fh/DH=02h

Category: Windows - Int 2Fh - W
