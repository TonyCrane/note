Novell NetWare - CONNECTION SERVICES - CHANGE USER PASSWORD (OLD)

AH = E3h subfn 01h
DS:SI -> request buffer (see #01891)
ES:DI -> reply buffer (see #01895)


Return:
AL = status
00h successful
else error code

Note: Supported by NetWare 4.0+, Advanced NetWare 1.0+, and Alloy NTNX

See Also: AH=E3h/SF=0Ah - AH=E3h/SF=32h - AH=E3h/SF=64h -

Format of NetWare "Change User Password (old)" request packet: Offset Size Description (Table 01891) 00h WORD length of following data 02h BYTE 01h (subfunction "Change User Password (old)") 03h BYTE length of user name 04h N BYTEs user name. BYTE length of old password N BYTEs old password. BYTE length of new password N BYTEs new password

See Also: #01895
