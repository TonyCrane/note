FGDRIVER v4.02 - FG_FLOOD - FLOOD FILL ARBITRARY CLOSED REGION

AX = 0035h
BX = starting column
CX = starting row

Desc: Fill the bounded region around the specified point (respecting clipping region) with the currrent color

Notes: The FGDRIVER functions are rearranged with each major release, but their parameters do not change (see #03474 at AX=0000h). This function is ignored in text modes

See Also: AX=0028h - AX=0085h

Category: Video - Int 62h - F
