G8BPQ v4.00+ - HOST MODE - GET STREAM STATUS

AH = 04h
AL = stream number (01h-40h)


Return:
CX = state (0000h disconnected, 0001h connected)
DX = delta state (0000h no change, 0001h changed since last check)

See Also: AH=00h"G8BPQ" - AH=02h"G8BPQ" - AH=05h"G8BPQ"
