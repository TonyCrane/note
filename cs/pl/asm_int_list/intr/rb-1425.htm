APM SL Enhanced v1.0 - GET CALENDAR EVENT TIME

AX = 5380h
BH = 08h


Return:
CF clear if successful
CH = hours
CL = minutes
SI = seconds
CF set on error
AH = error code (see #00483)

Desc: Gets calendar event time from CMOS ram

See Also: AX=5380h/BH=06h - AX=5380h/BH=09h - AX=5380h/BH=0Ah -

(Table 00483) Values for APM SL error code: 02h no alarm set 03h no battery

Category: Power Management - Int 15h - A
