Alloy MW386 - WRITE BYTE TO CONSOLE PORT

AH = C6h
AL = byte to write


Return:
CF clear if successful
CF set on error (caller is not remote user)

Note: Any terminal driver data translation will be bypassed

See Also: AH=C3h - AH=C7h
