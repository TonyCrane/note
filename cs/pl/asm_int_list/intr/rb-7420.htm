LIM EMS - MAP MEMORY

AH = 44h
AL = physical page number (0-3)
BX = logical page number
or FFFFh to unmap (QEMM)
DX = handle


Return:
AH = status (00h,80h,81h,83h,84h,8Ah,8Bh) (see #03648)

See Also: AH=69h
