VIDEO - Hercules GRAFIX - GETPT - FIND PIXEL VALUE

AH = 47h
DI = x (0-719)
BP = y (0-347)


Return:
AL = 00h pixel clear
AL = 01h pixel set

Note: Function 43h specifies which page is used

See Also: AH=0Dh - AH=46h
