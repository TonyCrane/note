SK-UPPS Data Link Interface API - DISABLE PROTOCOL

AX = 0008h
BX = protocol ID (see #03849)


Return:
AX = completion code (see #03846)

Note: The DLI will handle any Mbufs given to TRANSMIT FRAME (AX=0006h) as if they had been given to TRANSMIT FRAME RELEASE (AX=0013h).. All multicast addresses registered for this protocol will be disabled.. After this function returns, the protocol's Upcall routines will no longer be called.

See Also: AX=0007h - AX=000Bh
