Alloy MW386 - GET PORT INFORMATION

AH = 30h
CX = MW386 port number


Return:
AL = result
FFh if port not found
else driver unit number
BL = port mode
BH = port type
02h remote
DH = owner's machine ID
DL = owner's user ID

See Also: INT 17/AH=8Bh

Category: Network - Int 7Fh - A
