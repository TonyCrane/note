Atari Portfolio - FORMAT CREDIT CARD MEMORY

AH = 07h
AL = drive number (00h = A:,01h = B:)


Return:
CF clear if successful
CF set on error
AH = error code (see #00234 at INT 13/AH=01h)

Note: Do not use this function to format the internal disk (drive number 02h)

See Also: AH=00h"Portfolio" - AH=08h"Portfolio" - AH=09h"Portfolio"

Category: Vendor-specific BIOS Extensions - Int 61h - A
