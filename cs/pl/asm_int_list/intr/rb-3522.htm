Novell NetWare - QUEUE SERVICES - DESTROY QUEUE

AH = E3h subfn 65h
DS:SI -> request buffer (see #01964)
ES:DI -> reply buffer (see #01969)


Return:
AL = status (00h,96h,9Ch,D0h,D1h,FFh) (see also AH=E3h/SF=64h)
FFh hardware failure

Desc: Abort all active jobs, detach all job servers, remove all job entries, delete all job files, remove the queue object and its properties from the bindery, and delete the queue's subdirectory

Notes: This function is supported by Advanced NetWare 2.1+. Caller must have SUPERVISOR privileges

See Also: AH=E3h/SF=64h - AH=E3h/SF=66h - AH=E3h/SF=68h - See Also: AX=F217h/SF=65h 


Format of NetWare "Destroy Queue" request buffer:

Offset  Size    Description     (Table 01964)
00h    WORD    0005h (length of following data)
02h    BYTE    65h (subfunction "Destroy Queue")
03h    DWORD   (big-endian) object ID of queue

See Also: #01969

Category: Netware - Int 21h - N
