Novell NetWare - WORKSTATION - GET DRIVE CONNECTION ID TABLE

AX = EF02h


Return:
ES:SI -> network shell's 32-byte drive conection ID table
AX = 0000h

Notes: This function is supported by Advanced NetWare 1.0+. Each byte in the connection ID table corresponds to a drive and contains either the connection ID (1-8) of the server for that drive or 00h if the drive is not mapped to a file server

See Also: AX=EF01h - AX=EF03h - AX=F002h
