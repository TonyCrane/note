NETWORK REDIRECTOR - PROCESS TERMINATION HOOK

AX = 1122h
SS = DOS DS
DS = PSP of process about to terminate

???

Notes: This function is called by the DOS 3.1+ kernel. After calling this function, the kernel calls INT 2F/AX=111Dh
