Novell DOS 7+ - MEMORYMAX CREATE ACCESS KEY

AX = 12FFh
BX = 0406h


Return:
CF clear if successful
AX = 0000h (successful)
CX = 0000h
SI:DI = access key
CF set on error
AX = function status
0000h function supported
CX error code
80h if access key already exists
0001h (AX > 0) function not supported
CX undefined

Note: This function is supported by Novell DOS 7+ EMM386.EXE 3.0+ and possibly by HIMEM.SYS 2.3+.

See Also: AX=12FFh/BX=0506h
