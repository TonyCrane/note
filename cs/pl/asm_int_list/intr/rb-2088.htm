PC Tools v5.1-8.0 DESKTOP - DEFINE STANDARD PULLDOWN MENUS

AX = FFFAh

Notes: Available only when popped up. Adds the "Window" option to the "Desktop" option which is the only one available when no accessories are active. Unlike AX=FFEEh, no additional menu items are added between "Desktop" and "Window"

See Also: AX=FFEEh - AX=FFFBh

Category: Resident Utilities - Int 16h - P
