DECnet DOS LOCAL AREA TRANSPORT - LAT SERVICE TABLE RESET

AX = D600h
DH = FFh


Return:
AX = number of service table entries
BX = status
0000h service table has not overflowed
FFFFh service table has overflowed

See Also: AX=D500h
