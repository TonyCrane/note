Novell NetWare - WORKSTATION - GET DRIVE HANDLE TABLE

AX = EF00h


Return:
ES:SI -> network shell's 32-byte drive handle table
AX = 0000h

Notes: This function is supported by Advanced NetWare 1.0+. Each byte in the drive handle table contains the directory handle for the corresponding drive, or 00h if not mapped to a directory

See Also: AX=EF01h - AX=EF02h - AX=EF03h - AX=EF04h
