Tandy 2000 - SERIAL - FLUSH COMM BUFFER

AH = 04h
DL = port number
DH = protocol

bit 0:
Use XON/XOFF on received data

bit 1:
Use XON/XOFF when transmitting


Return:
Nothing

Desc: Clears the serial interface buffer

Note: This interrupt is identical to INT 53 on the Tandy 2000

See Also: AH=00h"Tandy 2000" - INT 53"Tandy 2000"
