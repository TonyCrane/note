AMI BIOS - BIOS-FLASH Interface - FLASH WRITE ENABLE

AX = E007h


Return:
CF clear if successful
AL = FAh
CF set on error

Note: This function performs any delay required to allow the Flash ROM to stabilize in the write-enabled state

See Also: AX=E000h - AX=E005h - AX=E006h - AX=E008h
