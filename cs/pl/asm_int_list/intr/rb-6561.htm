HP 95LX System Manager - SET DATE PARSING RULE

DI = 0B12h.

STACK:
2 WORDs unused dummies (for calls from high level languages)
WORD    new parsing rule (see #03299)


Return:
???
STACK unchanged

See Also: INT 60/DI=0B13h 



(Table 03299)
Values for HP 95LX date parsing rule:
01h    day-month-year
02h    month-day-year
03h    year-month-day
04h    "DMYO"
05h    "MDYO"
OR with 08h to get any year
