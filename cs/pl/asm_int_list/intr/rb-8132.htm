Alloy MW386 - ASSIGN DIRECTORY TO GROUP

AH = F1h
AL = group number
DS:DX -> ASCIZ directory name

Notes: Performs permanent assignment to a group; no immediate action is taken unless the directory has been restricted with AH=F0h. May be used to restrict a nonexistent directory

See Also: AH=F0h
