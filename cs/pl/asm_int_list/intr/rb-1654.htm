AMI PCI BIOS v1.00.05.AX1 - ???

AH = DAh
AL = function (00h-08h,12h,14h,15h,19h,88h-8Eh,92h,99h)
other registers vary by function


Return:
CF clear if successful
varies by function
CF set on error
AH = error code (86h unsupported [sub]function)

Note: Functions not listed above always return CF set and AH=86h; in the examined BIOS, functions 02h-04h,06h-07h,89h-8Bh, and 8Dh also always return CF set and AH=86h
