FGDRIVER v4.02 - FG_SETHPAGE - SET HIDDEN VIDEO PAGE

AX = 00AEh
BX = new hidden page (0000h to 003Fh)

Notes: The FGDRIVER functions are rearranged with each major release, but their parameters do not change (see #03474 at AX=0000h). The specified page must be a physical page or a virtual page

See Also: AX=0043h - AX=00B2h - AX=00B6h
