SBSIM - GETADDRESS - GET SELECTED DRIVER'S ENTRY POINT

BX = 0002h
AX = driver (00h = FM,01h = DDBV,02h = memvoice,03h = mixer,04h = MIDI)


Return:
CF clear if successful
DX:AX -> entry point
CF set on error
