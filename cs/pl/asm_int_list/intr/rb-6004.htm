MS MOUSE v8.10+ - SET ACCELERATION PROFILE NAMES

AX = 002Eh
BL = flag (if nonzero, fill ES:SI buffer with default names on return)
ES:SI -> 64-byte buffer containing profile names (16 bytes per name)


Return:
AX = status (0000h success)
FFFEh error for ATI Mouse driver
ES:SI buffer filled with default names if BL nonzero on entry

Notes: Not supported by Logitech driver v6.10. Supported by ATI Mouse driver v7.04

See Also: AX=002Ch - AX=002Dh - AX=012Eh - AX=022Eh

Category: Mouse/Pointing Device - Int 33h - M
