CauseWay - SetSelDet - SET SELECTOR BASE AND LIMIT

AX = FF09h
BX = selector
CX:DX = new base address
SI:DI = new byte-granular limit


Return:
CF clear if successful
CF set on error

See Also: AX=FF07h - AX=FF0Ah
