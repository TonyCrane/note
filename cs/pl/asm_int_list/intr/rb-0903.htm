Digiboard DigiCHANNEL PC/X* - CLEAR TRANSMIT BUFFER

AH = 11h
DX = port number (00h-03h) (04h-43h for XAPCM232.SYS)


Return:
AH = status
00h successful
FFh error

See Also: AH=09h"Digiboard" - AH=10h"Digiboard"
