TIME - RESET RTC DATE/TIME ACTIVATED POWER-ON MODE (IBM)

AH = 0Dh


Return:
CF clear if successful
CF set on error

Desc: Cancel a previously-set power-on alarm

Note: IBM classifies this function as optional

See Also: AH=07h - AH=0Ch - AH=0Eh

Category: Bios - Int 1Ah - T
