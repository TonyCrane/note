TelAPI - telclose - TERMINATE TELNET CONNECTION

AH = E1h
BX = connection ID


Return:
AX = status (0000h,FFF7h,maybe others) (see #00397)

Note: Flushes and releases all buffers and data space used by the connection

See Also: AH=E0h - AH=E6h - AX=FF00h
