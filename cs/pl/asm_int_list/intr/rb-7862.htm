Novell NetWare - SPX Driver - ABORT SPX CONNECTION

BX = 0014h
DX = connection ID to terminate


Return:
Nothing

Notes: This function is supported by Advanced NetWare 2.1+. This function does not tell the other side that the connection has been terminated. Also aborts any outstanding Establish Connection, Terminate Connection, and Send Sequenced Packet commands

See Also: BX=0011h - BX=0013h
