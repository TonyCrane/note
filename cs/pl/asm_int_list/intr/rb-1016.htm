COMM-DRV v14.0 - READ PORT METRICS - GET I/O BUFFER SIZES

AX = 8003h
DX = port number


Return:
AX = number of characters in input buffer
BX = input buffer size
CX = number of characters in output buffer
DX = output buffer size
