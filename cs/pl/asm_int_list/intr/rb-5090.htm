STRETCH - INSTALLATION CHECK

AX = 96C7h


Return:
AX = AAAAh if installed

Program: Stretch.exe is a TSR for Toshiba Laptops with WD90C24 video chip to avoid blank screen areas in VGA text modes by increasing inter-line spacing
