HP Vectra EX-BIOS - F_IO_CTRL SF_CRV_SELF_TEST - PERFORM SELFTEST

AX = 0410h
BP = 0114h (driver ID for V_HPHIL)
BX = device address indicator (see #03760)
DH = HP HIL device major address
ES:SI -> buffer for result


Return:
AH = status (see #03744)
CX = number of bytes in buffer
BP,DS destroyed

Note: This function is only available if the driver is in the open state (see AX=020Eh"SF_OPEN")

See Also: AH=00h"HP Vectra" - AX=040Ch/BP=0114h - AX=0412h/BP=0114h
