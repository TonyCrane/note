IBM Digitized Sound Package MIDI driver - GET DIGITIZED SOUND CAPABIL

AX = 0701h


Return:
AX = digitized sound capabilities
0000h if digitized sound driver (functions 06xxh) not available

InstallCheck:
Test for the signature "MIDI" six bytes before the interrupt
handler

Note: Also supported by MIDPAK, the successor to the Digitized Sound Package's MIDI driver

See Also: AX=0688h

Category: Sound/Speech - Int 66h - I
