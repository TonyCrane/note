AVATAR.SYS - QUERY DRIVER STATE

AX = 1A3Fh (AL='?')
ES:DI -> buffer
CX = length of buffer in bytes


Return:
CF clear
CX = actual size of returned info

Note: The returned information consists of multiple letters whose meanings are described under AX=1A21h

See Also: AX=1A00h/BX=4156h - AX=1A21h - AX=1A44h
