Novell NetWare - WORKSTATION - SET END OF JOB STATUS

AH = BBh
AL = new EOJ flag
00h disable EOJs
01h enable EOJs


Return:
AL = old EOJ flag

Desc: Specify whether the network shell should automatically generate an End of Job call when the root command processor regains control

Note: This function is supported by NetWare 4.0+, Advanced NetWare 1.0+, and Alloy NTNX

See Also: AH=19h - AH=D6h

Category: Netware - Int 21h - N
