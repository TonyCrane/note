3com BAPI SERIAL I/O - SEND SHORT BREAK

AH = A6h
DH = session ID (00h for external session managment)


Return:
AH = return code (00h,03h,07h,0Ah,0Bh) (see #00394)

Desc: Generate a short break signal; if data delivery was turned off by the break, wait for the host to turn it on again

See Also: AH=1Ah - AH=8Ah - AH=FAh - AH=A0h"BAPI"

Category: Serial I/O - Int 14h - @
