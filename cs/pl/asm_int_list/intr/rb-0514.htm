UNCHAIN - UPDATE PAGE 3 OFFSET

AX = CB03h
BX = offset of page 3


Return:
Nothing

Desc: Inform UNCHAIN of the location of video page 3 in video memory; the default is 7D00h (32000, for 320x200)

See Also: AX=CB01h - AX=CB02h - AX=CB04h

Category: Video - Int 10h - U
