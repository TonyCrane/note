TIME - SET SYSTEM TIME

AH = 01h
CX:DX = number of clock ticks since midnight


Return:
Nothing

Notes: There are approximately 18.2 clock ticks per second, 1800B0h per 24 hrs (except on Tandy 2000, where the clock runs at 20 ticks per second). This call resets the midnight-passed flag

See Also: AH=00h - AH=03h - INT 21/AH=2Dh
