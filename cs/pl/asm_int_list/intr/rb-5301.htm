DIMWIT - INSTALLATION CHECK

AX = C000h


Return:
AL = FFh if installed
ES:DI -> signature string "DIMDOS" or "DIMWIN"

Program: DIMWIT is a freeware Windows-aware screen blanker by Larry Board 

Range:
AH=C0h to AH=FFh, selected by scanning for first free multiplex number

See Also: AX=C000h"AD-DOS" - AX=C001h"DIMWIT"
