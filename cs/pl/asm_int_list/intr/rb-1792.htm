HUNTER 16 - REDEFINE RAW KEY CODES

AH = 2Ch
AL = Matrix code of key (see AH=29h)
BL = new key code


Return:
AL = status (00h success, nonzero failed)

Desc: Redefine the key code generated by holding the PAW key down and pressing the key in AL

See Also: AH=24h - AH=2Bh

Category: Vendor-specific BIOS Extensions - Int 16h - H
