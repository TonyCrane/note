VIDEO - ALTERNATE FUNCTION SELECT (PS,EGA,VGA,MCGA) - ALTERNATE PRTSC

AH = 12h
BL = 20h  select alternate print screen routine


Return:
Nothing

Desc: Installs a PrtSc routine from the video card's BIOS to replace the default PrtSc handler from the ROM BIOS, which usually does not understand screen heights other than 25 lines

Note: Some adapters disable print-screen instead of enhancing it

See Also: INT 05"PRINT SCREEN"

Category: Video - Int 10h - V
