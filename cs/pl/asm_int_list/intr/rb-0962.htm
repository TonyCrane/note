HUNTER 16 - CONTROL HANDSHAKE LINES

AH = 23h
BH = handshake line to set (00h RTS, 01h DTR)
BL = new level (00h low, 01h high)


Return:
Nothing

Desc: Sets the handshake lines of COM1 to the desired level

See Also: AH=21h"HUNTER" - AH=24h"HUNTER"
