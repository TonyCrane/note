ASAP v1.0 - INSTALLATION CHECK

AX = F0F1h
DX = 0000h


Return:
DX = segment of resident code
0000h if not installed

Program: ASAP (Automatic Screen Access Program) is a shareware screen reader by MicroTalk

See Also: AX=F0F0h - INT 10/AX=3800h

Category: Access Software (screen readers, etc) - Int 14h - A
