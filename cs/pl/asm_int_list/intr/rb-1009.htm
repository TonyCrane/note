ARTICOM - INSTALLATION CHECK

AX = 8000h


Return:
AL = FFh if installed
BH = major version
BL = minor version

Program: ArtiCom is an asynchronous communications driver by Artisoft which works on top of NetBIOS and allows modem/serial-port sharing by programs using INT 14 for serial I/O.

Note: ArtiCom supports 32 simultaneous COM ports using multiport cards and drivers
