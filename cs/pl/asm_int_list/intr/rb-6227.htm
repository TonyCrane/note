Int 49/AH=02h

Interrups Categories Contents

TI Professional PC - CRT - SET CURSOR POSITION

AH = 02h
DH = column
DL = row


Return:
DX destroyed

Notes: AH=00h,04h,05h,0Bh,0Ch,0Dh,0Fh are documented as NOPs. The TI swaps the row and column compared to the equivalent IBM call

See Also: AH=01h - AH=03h
