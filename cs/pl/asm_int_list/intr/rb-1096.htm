3com BAPI SERIAL I/O - INSTALLATION CHECK

AX = AF00h
BX = AAAAh


Return:
AX = AF01h if installed
BH = protocol type (if BX=AAAAh on entry)
01h NetManage TCP/IP
BL = version for protocol type (if BX=AAAAh on entry)

Note: Early versions of the BAPI and the ROM BIOS simply destroy AX; this behavior is used to determine whether the newer functions (AH=B0h, AH=B1h,etc) are available

See Also: AH=A0h"BAPI"
