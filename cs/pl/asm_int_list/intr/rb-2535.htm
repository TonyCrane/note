Return:
CF clear

01h    ???
02h    ???
03h    hook/unhook VMM "hook device service" service.
EAX = request (0 = unhook, nonzero = hook).

Return:
If EAX nonzero on entry, Splitter's service 04h replaces VMM
service 0090h; otherwise, default handler is restored
