K3 v1.5x, K3PLUS v5.xx - COPY INTO EXTENDED BUFFER

AH = 25h
CX = number of keystrokes to copy
ES:SI -> buffer containing keystrokes


Return:
CF clear if successful
CF set on error (i.e. buffer full)
CX = number of keystrokes NOT transferred
ES:SI -> first keystroke not transferred

Note: This function was normally unsupported under K3PLUS v6.00-v6.22 and is no longer supported by v6.30+; use AX=AF25h instead

See Also: AH=05h - AH=20h"K3" - AX=AF25h - INT 2F/AX=D44F
