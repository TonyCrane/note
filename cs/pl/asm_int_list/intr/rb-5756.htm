Borland C++ 3.0 DPMILOAD.EXE - SET PROTECTED-MODE INTERRUPT VECTOR

AX = FB42h
BX = 0025h
CL = interrupt number
SI:DX = selector:offset of new handler


Return:
DX = status (0000h,0004h,FFF2h) (see #03129)
