Interrups Categories Contents

Compaq 386 and newer - READ CURRENT CPU SPEED

AH = F1h


Return:
AL = speed code (see #00614)
if AL = 09h, CX = speed code

Note: Also supported by some versions of AMI BIOS dated June 1992 or later

See Also: AH=F0h - AH=F3h

Category: Vendor-specific BIOS Extensions - Int 16h - C
