HP 95LX System Manager - SET CURSOR POSITION

DI = 030Bh.

STACK:
2 WORDs unused dummies (for calls from high level languages)
WORD    row (-3 is topmost, 0 is first non-reserved line)
WORD    column


Return:
???
STACK unchanged

Note: Cursor is hidden if the specified position is not on the physical display
