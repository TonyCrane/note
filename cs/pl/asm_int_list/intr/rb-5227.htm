DOS 3.3+ GRAFTABL.COM - INSTALLATION CHECK

AX = B000h


Return:
AL = status
00h not installed, OK to install
01h not installed, not OK to install
FFh installed

Notes: Called by DISPLAY.SYS. Documented for DOS 5.0, but undocumented in prior versions

See Also: AX=2300h - AX=2E00h - AX=B001h
