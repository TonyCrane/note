VGARAM v1.00 - INSTALLATION CHECK

AX = B0B1h
ES:DI -> 6 byte signature "VGARAM"


Return:
AX = B1B0h if installed,.

DS:BX -> VGARAM Status byte:
0 = OFF, 1 = ON

Program: VGARAM is a utility by Brett Warthen which makes VGA memory which is not used in text modes available for DOS
