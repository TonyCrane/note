SK-UPPS Data Link Interface API - GET DRIVER VERSION

AX = 0000h


Return:
AH = major DLI version
AL = minor DLI version
DX = OEM signature (see #03845).

CL = DLI 2.8+ topology:
1=Ethernet, 2=FDDI, 3=Token-Ring

See Also: AX=0002h - AX=000Bh - AX=000Dh - AX=000Eh
