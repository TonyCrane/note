MultiLink Advanced v1.0+ - DEQUEUE SYSTEM RESOURCE

AH = 01h
BX = resource identifier


Return:
AL = status
00h successful
02h user error

Note: The indicated resource may be dequeued even if it was enqueued by another task

See Also: AH=00h"MultiLink" - AH=02h"MultiLink"

Category: DOS-Based Task Switchers/Multitaskers - Int 7Fh - M
