Connection Manager - TERMINATE CONNECTION CLIENT ACTIVITY

AH = 0Dh
DX = FFFFh


Return:
AH = return code
00h successful
nonzero operation not terminated

Program: Connection Manager by Softwarehouse Corp. permits the sharing of serial ports over an IPX or NetBIOS-based network

Desc: End all Connection Client TSR activity to allow it to be removed from memory

See Also: AH=05h/DX=FFFFh - AH=6Fh/BX=FFFFh
