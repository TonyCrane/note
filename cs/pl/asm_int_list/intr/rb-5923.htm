Int 31/AX=FF20h

Interrups Categories Contents

CauseWay - UnLockMemNear - UNLOCK APP-RELATIVE MEMORY REGION

AX = FF20h
ESI = starting application-relative linear address
ECX = size of region in bytes


Return:
CF clear if successful
CF set on error

Desc: Allow memory to be swapped out by the virtual memory manager

Note: Locks are applied on 4K page boundaries, so memory above and below the specified region may be unlocked as well

See Also: AX=FF1Eh - AX=FF1Fh
