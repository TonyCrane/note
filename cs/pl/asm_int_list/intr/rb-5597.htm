DESQview v2.26+ External Dev Interface - DV TERMINATION

AX = DE03h
BX = mapping context of DESQview
DX = handle of DESQview system task

Notes: Driver should pass this call to previous handler before doing its work. DESQview makes this call when it is exiting, but before unhooking any interrupt vectors

See Also: AX=DE02h - AX=DE0Fh - INT 15/AX=5407h
