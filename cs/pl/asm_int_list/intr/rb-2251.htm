Gravis Ultra Sound YEA_GUS.EXE - GET STATUS

AX = 6900h


Return:
AX = amount of DRAM on card or 0000h if GUS not available

Program: YEA_GUS is a driver for the Graphics Ultra Sound which hooks INT 18h and then shells out the the program requiring its services

See Also: AX=6901h - AX=690Ah - AX=690Bh

Category: Sound/Speech - Int 18h - G
