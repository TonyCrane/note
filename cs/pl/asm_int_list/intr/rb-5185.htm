DOS 5+ KEYB.COM - GET KEYBOARD MAPPING

AX = AD83h


Return:
BL = current state
00h US keyboard
FFh foreign keyboard

Note: The states 00h and FFh correspond with the byte stored at offset +02h in KEYBs INT 09h handler

See Also: AX=AD82h - AX=AD85h - INT 09
