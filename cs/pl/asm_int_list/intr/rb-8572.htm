Int E0/CL=A6h

Interrups Categories Contents

DR Multiuser DOS - A_DETACH - DETACH FROM AUX DEVICE

CL = A6h


Return:
AX = status
0000h successfully detached
FFFFh detach failed
CX = error code

Note: Also supported by REAL/32

See Also: INT E0/CL=93h - INT E0/CL=A5h - INT E0/CL=A7h

Category: Other Operating Systems - Int E0h - D
