Int 15/AH=22h

Interrups Categories Contents

SYSTEM - later PS/2s - LOCATE ROM BASIC

AH = 22h


Return:
CF set on error
AH = status (86h if function not supported)
CF clear on success
AH = 00h
ES:BX -> ROM BASIC

Notes: If this function is not supported, ROM BASIC is at F600h:0000h. IBM classifies this function as optional

See Also: INT 86"BASIC" - INT F0"BASIC"

Category: Bios - Int 15h - S
