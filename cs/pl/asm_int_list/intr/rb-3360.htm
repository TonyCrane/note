Novell NetWare - TRANSACTION TRACKING SYSTEM - SET WORKSTN THRESHOLDS

AX = C708h
CL = maximum logical record locks (default 0)
CH = maximum physical record locks (default 0)


Return:
AL = status
00h successful

Desc: Specify the per-workstation limits on record locks allowed before an implicit transaction is begun

Notes: This function is supported by NetWare 4.0+ and Advanced NetWare 1.0+. If either limit is set to FFh, implicit transactions are disabled for the corresponding lock type

See Also: AX=C706h - AX=C707h - AX=F222h/SF=08h
