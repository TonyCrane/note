QRIP/TSR - EXECUTE RIP END SCENE COMMAND

AX = AC0Bh
DI = 1092h


Return:
Nothing

Notes: If DI <> 1092h on entry, QRIP chains the call. Enables all mouse buttons and cancels the RIP timeout

See Also: AX=AC01h - AX=AC04h - AX=ACF0h
