HUNTER 16 - SET CONTRAST POWER UP STATE

AH = 75h
AL = new level
01h off
02h unchanged
03h change level
BL = desired level (00h-7Fh)


Return:
AH = status

Desc: Select the LCD contrast level to use when the Hunter is next turned on

See Also: AH=63h - AH=74h

Category: Vendor-specific BIOS Extensions - Int 15h - H
