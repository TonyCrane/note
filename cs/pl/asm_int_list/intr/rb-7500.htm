Virtual Control Program Interface - SET DEBUG REGISTERS

AX = DE09h
ES:DI -> array of 8 DWORDs holding new values of debug registers


Return:
AH = 00h

Note: Values for DR4 and DR5 ignored

See Also: AH=3Fh - AX=DE08h
