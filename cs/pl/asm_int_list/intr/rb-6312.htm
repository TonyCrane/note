PC Thuis Organizer Shell - WRITE FILE

AX = 0003h
ES:BX -> data to be written
CX = number of bytes to write
DS:SI -> filename


Return:
AX = status
0000h successful
FFFFh error

See Also: AX=0004h"Shell"
