Voyetra - AAPISG - API

AX = 4150h ('AP')
BH = C1h
BL = function (00h-13h)
00h initialize (fails except first time called)
???


Return:
AX = status???
0000h successful
0001h failed

Program: AAPISG is a driver by Voyetra for the Aztech Sound Galaxy sound board

BUG: The function range check uses JL instead of JB, so it will cause a crash if BL >= 80h on entry

See Also: AX=4331h - AX=564Dh - AX=5658h
