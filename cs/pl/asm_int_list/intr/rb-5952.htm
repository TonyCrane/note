NOISE.SYS v0.53+ - READ URANDOM BYTES

AX = 6E16h
CX = number of bytes
ES:DI -> buffer


Return:
CF set on error
AL = error code (FDh,FEh) (see #03166)
CF clear if successful
CX = number of random bytes read

See Also: AX=6E00h - AX=6E12h - AX=6E17h

Category: Security - Int 32h - N
