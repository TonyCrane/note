Novell NetWare - CONNECTION SERVICES - MAP OBJECT TO NUMBER (OLD)

AH = E3h subfn 03h
DS:SI -> request buffer
ES:DI -> reply buffer


Return:
AL = status
00h successful
else error code

Note: Supported by NetWare 4.0+, Advanced NetWare 1.0+, and Alloy NTNX

See Also: AH=E3h/SF=01h - AH=E3h/SF=02h - AH=E3h/SF=0Ah
