DR Multiuser DOS - A_STATOUT - GET OUTPUT STATUS OF AUX DEVICE

CL = 08h


Return:
AL = status
00h not ready
FFh ready for output

Desc: Determine whether the current AUX device is able to accept more output

See Also: INT E0/CL=04h - INT E0/CL=07h

Category: Other Operating Systems - Int E0h - D
