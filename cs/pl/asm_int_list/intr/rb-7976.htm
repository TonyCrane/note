XLOAD - GET XLOAD MEMORY SIZE

AX = 5857h
BL = 5Bh


Return:
AX = 0000h
CX = segment of XLOAD TSR
DX = size of memory block in which XLOAD TSR is located

See Also: AX=5857h/BL=57h - AX=5857h/BL=5Ah
