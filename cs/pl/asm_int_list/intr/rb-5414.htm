TBSCANX - INSTALLATION CHECK

AX = CA00h
BX = 5442h ('TB')


Return:
AL = state
00h not installed
FFh installed
BX = 7462h ('tb') if BX was 5442h on entry

Program: TBSCANX is a resident virus scanning module by Frans Veldman.

Notes: Programs may perform virus checks on themselves, other program files, or their data files by invoking the TBSCANX API.. Current versions of TBScanX don't actually check the value of BX

See Also: AX=4653h/CX=0002h - AX=4653h/CX=0007h - AX=C900h

Category: Virus/Antivirus - Int 2Fh - T
