DOS 3.0+ internal - SET YEAR/LENGTH OF FEBRUARY

AX = 121Bh
CL = year - 1980


Return:
AL = number of days in February

Note: Requires DS to be set to the DOS data segment

See Also: INT 21/AH=2Bh"DATE"
