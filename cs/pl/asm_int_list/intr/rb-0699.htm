QUICKCACHE II v4.20 - SET WRITE BUFFER SIZE FOR SPECIFIC DRIVE

AH = 3Ah
AL = new size of write buffer
DL = drive number (00h = A:)


Return:
AX = status (0000h successful)

See Also: AH=2Ah - AH=39h
