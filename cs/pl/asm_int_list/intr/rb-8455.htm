PC-MOS/386 v5.01 - CHANGE TERMINAL DRIVER

AH = 23h
BX = task ID (FFFFh for current task)
DS:SI -> entry point of the new Device Driver Terminal


Return:
CF clear if successful
CF set on error
AX = error code (see #01680 at INT 21/AH=59h/BX=0000h)
