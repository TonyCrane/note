HP 95LX System Manager - SEND DATA OVER COMM CHANNEL

DI = 0E08h.

STACK:
2 WORDs unused dummies (for calls from high level languages)
WORD    comm channel handle
DWORD   pointer to data to be sent
WORD    option flags

bit 0:
Send partial buffer

bit 1:
Turn on receiver after sending
DWORD   pointer to WORD containing length of data to be sent


Return:
Length WORD updated to contain number of bytes actually sent???
STACK unchanged

See Also: INT 60/DI=0E09h - INT 60/DI=0E0Bh

Category: Vendor-specific BIOS Extensions - Int 60h - H
