MS Windows WINOLDAP - GET CLIPBOARD DATA

AX = 1705h
DX = clipboard format supported by WinOldAp (see #02723)
ES:BX -> buffer


Return:
AX = status
nonzero success
0000h   error, or no data in this format in Clipboard

See Also: AX=1700h - AX=1704h - INT 16/AX=CB03h
