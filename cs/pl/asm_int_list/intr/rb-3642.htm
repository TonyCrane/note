Novell NetWare - PRINT SERVICES - GET LPT CAPTURE STATUS

AX = F003h


Return:
AH = status
00h not active
FFh active
AL = connection ID (01h-08h)

Note: This function is supported by Advanced NetWare 1.0+

See Also: AX=B800h - AX=B804h - AH=DFh/DL=00h - AH=DFh/DL
