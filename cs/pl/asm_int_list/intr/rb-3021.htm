DOS 3.1+ internal - SHARE.EXE - CLOSE ALL FILES FOR GIVEN PROCESS

AX = 5D04h
DS:DX -> DOS parameter list (see #01686), only computer ID and
process ID fields used


Return:
CF set on error
AX = error code (see #01680)
CF clear if successful

Notes: Error unless SHARE is loaded (calls [SysFileTable-2Ch]) (see #01636 at AH=52h). Not supported by DR DOS 3.41 and 5.0; returns error code 01h. Not supported by Novell DOS 7

See Also: AX=5D02h - AX=5D03h - INT 2F/AX=111Dh

Category: DOS Kernel - Int 21h - D
