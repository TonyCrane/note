HP Vectra EX-BIOS - F_INQUIRE - GET DRIVER PHYSICAL DESCRIBE RECORD

AH = 06h
BP = 002Ah (driver ID for V_SINPUT)
AL = HP-HIL physical device number (01h-07h)


Return:
AH = status (see #03744)
ES:SI -> physical describe record for driver
BP,DS destroyed

Note: The returned physical describe record must not be modified

See Also: AH=00h"HP Vectra" - AX=0400h/BP=002Ah - AH=08h/BP=002Ah - See Also: AH=0Ch/BP=002Ah
