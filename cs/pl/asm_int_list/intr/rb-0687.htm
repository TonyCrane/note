QUICKCACHE II v4.20 - SET FLUSH COUNT

AH = 2Eh
BX = flush count


Return:
AX = status (0000h successful)

Desc: Specify how many dirty sectors the cache should write after each flush interval (see AH=25h"QUICKCACHE") when buffered writes are enabled

See Also: AH=21h"QUICKCACHE" - AH=25h"QUICKCACHE" - AH=2Ch

Category: Caches/Spoolers - Int 13h - Q
