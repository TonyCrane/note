NETWORK - END DOS CRITICAL SECTIONS 0 THROUGH 7

AH = 82h

Notes: Called by the INT 21h function dispatcher for function 0 and functions greater than 0Ch except 59h, and on process termination. The handler should reawaken any tasks which were suspended due to an attempt to enter one of the critical sections 0 through 7

See Also: AH=81h
