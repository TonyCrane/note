Alloy NTNX,MW386 - RESTART PRINTER

AH = 98h
AL = 00h
DL = printer number (FFh=current)


Return:
AL = status
00h successful
01h incorrect printer
02h task not found

Note: MW386 supports this function for compatibility only; it is a NOP

Category: Network - Int 17h - A
