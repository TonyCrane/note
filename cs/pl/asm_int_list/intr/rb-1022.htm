ARTICOM - SET SEND AND RECEIVE TIMEOUTS

AX = 8009h
BX = send timeout in ticks
CX = receive timeout in ticks
DX = port index (COM1=0, COM2=1, ...)


Return:
Nothing

See Also: AX=8000h - AX=800Ah

Category: Serial I/O - Int 14h - A
