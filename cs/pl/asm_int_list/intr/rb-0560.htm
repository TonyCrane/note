EGA Register Interface Library - WRITE REGISTER SET

AH = F5h
CX = number of registers to write (>1)
ES:BX -> table of records (see #00224)


Return:
Nothing

Note: The RIL is provided by EGA.SYS, the Microsoft Mouse driver, the OS/2 compatibility box, and others

See Also: AX=7F05h - AH=F1h"EGA" - AH=F3h"EGA" - AH=F4h"E
