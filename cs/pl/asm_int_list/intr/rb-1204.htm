TopView - OSTACK - SWITCH TO TASK'S INTERNAL STACK

AX = 101Ah


Return:
Stack switched

Notes: This call may not be nested; a second call must be preceded by a call to "USTACK" (AX=1025h). While TopView requires many API calls to be executed while on the task's internal stack, DESQview allows those calls to be executed regardless of the current stack

See Also: AX=1025h
