DESQview v2.20+ - SEND MESSAGE - GETFLAGS - GET OBJECT FLAGS

AH = 12h
BH = 16h
BL = object
00h DWORD on top of stack
mailbox, keyboard, or pointer only
02h mailbox for task whose handle is on top of stack
03h mailbox for current task
04h keyboard for task whose handle is on top of stack
05h keyboard for current task


Return:
STACK: DWORD current control flags (see #00453,#00454)

Notes: Only available if the API level has been set to at least 2.20. If a mailbox has bits 4 or 5 set, you must use "READINTO" rather than "READ" (see AH=12h/BH=13h"READINTO") to retrieve messages
