RIFS - CLIENT - REMAP DRIVE

AX = 0002h
BH = local drive number
BL = remote drive number


Return:
CF clear if successful
CF set on error

See Also: AX=0000h - AX=0003h - AX=0004h - INT 21/AX=5F03
