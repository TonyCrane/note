AX (Japanese AT) PRINTER - Shift-JIS to JIS CONVERSION

AH = 52h
DX = 2-byte shift-JIS code


Return:
DX = JIS code or 0000h on error

Note: One of AH=51h and AH=52h converts from JIS (Japanese Industry Standard) characters to Shift-JIS characters, and the other performs the opposite conversion

See Also: AH=51h
