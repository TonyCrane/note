CauseWay - SetSelDet32 - SET SELECTOR BASE AND LIMIT (32-bit)

AX = FF0Ah
BX = selector
EDX = new base address
ECX = new byte-granular limit


Return:
CF clear if successful
CF set on error

See Also: AX=FF08h - AX=FF09h
