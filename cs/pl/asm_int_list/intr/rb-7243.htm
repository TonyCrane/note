Adaptec controllers - DRIVE 1 DATA

Desc: This vector stores the first four bytes of the parameter table for hard disk 1

Notes: These vectors are used by the following Adaptec controllers: ACB 2370 A/B/C, ACB 2372 A/B/C, ACB 2333 A/B, 2322B-8, 2322B-16. These vectors are NOT used by the following Adaptec controllers: ACB 2310, ACB 2312, ACB 2320D, ACB 2322D
