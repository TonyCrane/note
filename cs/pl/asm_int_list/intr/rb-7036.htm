FGDRIVER v4.02 - FG_MUSICB - PLAY SERIES OF NOTES IN BACKGROUND

AX = 0081h
CX = number of repetitions (negative = continuous play)
ES:BX -> '$'-terminated music string in BASIC PLAY format

Notes: The FGDRIVER functions are rearranged with each major release, but their parameters do not change (see #03474 at AX=0000h). This call is ignored if asynchronous sound (AX=00A5h,AX=00A8h, AX=00AAh) is already in progress

See Also: AX=0059h - AX=0081h - AX=00BDh - AX=00DCh
