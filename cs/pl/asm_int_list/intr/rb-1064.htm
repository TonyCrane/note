3com BAPI SERIAL I/O - READ CHARACTER

AH = A3h
DH = session ID (00h for external session managment)


Return:
AH = return code (00h,02h,03h,07h,0Ah,0Bh) (see #00394)
AL = character read or 00h if none available

See Also: AH=A0h"BAPI" - AH=A2h"BAPI" - AH=A5h"BAPI" - AH=A7h"BAPI"
