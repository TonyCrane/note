HUNTER 16 - GET/SET BOOST CHARGE (SPM)

AH = 8Dh
AL = function
00h get

Return:
AL = Fast Charge state (00h prevented, else allowed)
nonzero set
BH = Fast Charge state (00h prevent, nonzero allow)


Return:
AH = status
00h success
FFh failure

Desc: Control whether Fast Charging (200ma rather than 70ma) is allowed

See Also: AH=8Ch

Category: Vendor-specific BIOS Extensions - Int 15h - H
