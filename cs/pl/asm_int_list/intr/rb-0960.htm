MultiDOS Plus - GET PORT STATUS

AH = 23h
DX = port number


Return:
AH = line status (see #00304 at AH=03h)
AL = modem status (see #00305 at AH=03h)

See Also: AH=03h - AH=07h"MultiDOS" - AH=20h"MultiDOS"
