MTRTSR - INSTALLATION CHECK

AX = 6D74h ('mt')


Return:
AX = 4D54h ('MT') if installed

Program: MTRTSR is a part of the shareware MTR package by Mechon Mamre which permits searching the large corpus of classical Hebrew texts in the package
