Banyan VINES - GET PCCONFIG BASE ADDRESS

AH = 06h
DS:DX -> 4-byte buffer for far pointer to PCCONFIG area


Return:
AX = 0000h successful

Note: The PCCONFIG area is located in the code segment of the BANV handler

See Also: AH=05h"VINES" - AH=0Bh"VINES"
