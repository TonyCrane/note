Novell NetWare - FILE SERVICES - PURGE ERASED FILES (OLD)

AH = E2h subfn 10h
DS:SI -> request buffer (see #01865)
ES:DI -> reply buffer (see #01863)


Return:
AL = status (00h,C6h) (see #01866)

Desc: Purges files marked for deletion on the file server by the calling workstation

Note: This function is supported by NetWare 4.0+, Advanced NetWare 1.0+, and Alloy NTNX

See Also: AH=13h - AH=41h - AH=E2h/SF=11h - AH=E3h/SF=CEh 


Format of NetWare "Purge Erased Files" request buffer:

Offset  Size    Description     (Table 01865)
00h    WORD    0001h (length of following data)
02h    BYTE    10h (subfunction "Purge Erased Files")

See Also: #01863

Category: Netware - Int 21h - N
