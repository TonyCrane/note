Phar Lap 386/DOS-Extender - CONVERT LINEAR TO PHYSICAL ADDRESS

AX = 2509h
EBX = linear address to convert


Return:
CF clear if successful
ECX = physical address (carry flag clear)
CF set if linear address not mapped in page tables

See Also: AX=2508h
