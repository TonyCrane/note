DESQview v2.23+ - ENTERC - INCREMENT CRITICAL NESTING COUNT

AX = DE1Ch


Return:
Nothing

Notes: Similar to AX=101Bh, but begins the critical region without ensuring that DOS is free. The official documentation states that this call should be paired with "ENDC" (AX=101Ch); no mention is made of AX=DE1Bh. This API call may be made from within a hardware interrupt handler

See Also: AX=101Bh - AX=101Ch - AX=DE13h - AX=DE1Bh
