Int 4E/AH=01h

Interrups Categories Contents

TI Professional PC - TIME-OF-DAY CLOCK - SET BIOS TIME

AH = 01h
CH = hours
CL = minutes
DH = seconds
DL = hundredths


Return:
Nothing

Note: The BIOS does not validate the data passed to this function

See Also: AH=00h - AH=02h

Category: Vendor-specific BIOS Extensions - Int 4Eh - T
