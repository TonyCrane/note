HP Vectra EX-BIOS - F16_INQUIRE - Extended BIOS INSTALLATION CHECK

AX = 6F00h
BX <> 4850h (usually set to 0000h for simplicity)


Return:
BX = 4850h ("HP") if present

Notes: Called by recent MS Mouse drivers looking for an HP-HIL mouse. Supported by the original HP Vectra AT and ES/QS/RS series HP Vectras
