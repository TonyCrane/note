OS/2 v2.1+ Virtual DOS Machine - OS/2 API DosForceDelete

AH = 64h
BX = 00CBh (API ordinal)
CX = 636Ch (magic value "cl")
DS:DX -> ASCIZ filename


Return:
CF clear if successful
AX destroyed
CF set on error
AX = error code (02h,03h,05h) (see #01680 at AH=59h/BX=0000h)

Desc: Delete a file without saving it to the undelete directory

See Also: AH=41h - AH=64h/CX=636Ch

Category: Other Operating Systems - Int 21h - O
