SCSILink - RESET SCSI BUS

AH = 00h

Program: SCSILink is a TSR by Cross Products Ltd which allows its PC-hosted cross assemblers and similar products to communicate with Cross Products hardware debuggers

Desc: Reset all devices on the SCSI bus 

InstallCheck:
Test for the signature "SCSILINK" immediately prior to the
interrupt handler

Index:
Installation check;SCSILink

Category: Debuggers/Debugging Tools - Int 7Dh - S
