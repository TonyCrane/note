AD-DOS - GET AND RESET VxD API STATUS

AX = C020h


Return:
AL = 00h if successful
BX = VxD API Status
0000h no error
0001h error
0100h neither Windows 3.X enhanced mode nor Windows/386 2.x
is running
0200h VM API entry point not found (VxD not installed)

Range:
AH=C0h to AH=FFh, selected by scanning for first free multiplex number

Note: This call resets the VxD API Status to zero

See Also: AX=1602h - AX=1607h"CALL OUT API" 

Index:
Screen saver;AD-DOS

Category: Video - Int 2Fh - A
