HUNTER 16 - GET/SET BATTERY CAPACITY (APM)

AH = 80h
AL = function
00h get

Return:
AL = capacity (500mA + AL * 50mA, max 40 = 2500mA)
else set
BL = battery capacity (same as AL above)

See Also: AH=61h"HUNTER" - AH=81h"HUNTER"

Category: Vendor-specific BIOS Extensions - Int 15h - H
