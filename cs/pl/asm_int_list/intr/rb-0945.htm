X00 FOSSIL - DESTRUCTIVE READ WITH NO WAIT

AH = 20h
DX = port number


Return:
AH = 00h if character was available
AL = next character (removed from receive buffer)
AX = FFFFh if no character available

See Also: AH=0Ch - AH=21h"X00"

Category: Serial I/O - Int 14h - X
