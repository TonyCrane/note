Frank Kintrup TSR Utilities - INSTALLATION CHECK

AX = C000h


Return:
AL = FFh if installed
BX = program ID (see #03031)
CX = 464Bh (signature "FK")
DX = revision number (DH = major, DL = minor)
ES = resident segment of TSR

Range:
AH=C0h to AH=FFh, selected by searching for a free multiplex number

See Also: AX=C001h"Kintrup" - AX=C002h"Kintrup"
