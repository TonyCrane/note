DimVGA v2.0+ - INSTALLATION CHECK

AX = AA01h


Return:
AX = FFFFh if installed, unchanged
BX = version (v1.5+ only), BH = major, BL = minor (v1.5 = 0105h)
CX = resident segment (v3.1+)

Program: DimVGA is a public domain screen saver by Menno Pieters
