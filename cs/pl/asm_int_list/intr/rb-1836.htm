HP Vectra EX-BIOS - F16_KBD_RESET - RESET KEYBOARD TO DEFAULTS

AX = 6F09h


Return:
AH = 00h (successful)

Desc: Reset all keyboard mappings to their default translators, and reset typematic values to their defaults

Note: Supported by the original HP Vectra AT and ES/QS/RS series HP Vectras

See Also: AX=6F04h - AX=6F07h - AX=6F08h

Category: Vendor-specific BIOS Extensions - Int 16h - H
