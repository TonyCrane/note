WinDOS v2.11 - GET MAIN SHELL PSP

AX = AF0Ah


Return:
AX = shell's PSP (same as DS on last call to AX=AF06h)

Desc: Get the PSP address of the last program to call AX=AF06h (normally the main command interpreter)

See Also: AX=AF00h"WinDOS" - AX=AF06h"WinDOS" - AX=AF09h
