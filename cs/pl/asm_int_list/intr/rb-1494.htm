HUNTER 16 - REDIRECT SERIAL NUMBER

AH = 72h
AL nonzero
DS:BX -> new serial number (6 ASCII digits)


Return:
AH = 00h if successful

Desc: Install a new serial number

See Also: AH=6Ch - AX=7200h

Category: Vendor-specific BIOS Extensions - Int 15h - H
