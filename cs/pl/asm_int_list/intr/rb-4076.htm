TopWare Network OS v5.10+ - SET SPOOLER PRINTING PRIORITY

AX = FFB1h
CH = printer number (00h LPT1, 01h LPT2, 02h LPT3)
CH = new priority (00h normal, 01h high)


Return:
Nothing
