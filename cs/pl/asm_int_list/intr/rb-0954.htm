HUNTER 16 - EXTENDED CONTROL

AH = 21h
AL = command
01h force transmission of buffer
02h clear transmit buffer
03h clear receive buffer
DX = port (00h COM1, 01h COM2)


Return:
AH = extended status

Desc: Executes the command on the selected port
