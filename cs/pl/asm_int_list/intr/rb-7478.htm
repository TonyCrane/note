Nanosoft MD386 - SET DEBUG EXCEPTION HANDLER

AX = 5E02h
CX:BX -> exception handler

Note: The specified exception handler is called with a simulated interrupt whenever a debug exception occurs which was caused by a hardware breakpoint set with the debug registers

See Also: AX=5E04h - AX=5E05h
