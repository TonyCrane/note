FGDRIVER v4.02 - FG_ERASE - CLEAR THE ACTIVE VIDEO PAGE

AX = 0028h

Notes: The FGDRIVER functions are rearranged with each major release, but their parameters do not change (see #03474 at AX=0000h). This function sets each pixel to 0 in graphics modes, each character cell to a blank with a gray foreground attribute in text modes

See Also: AX=0029h - AX=002Bh - AX=0035h
