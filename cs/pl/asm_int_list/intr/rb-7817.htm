TARGA.DEV - SET DMA CHANNEL

AH = 02h
AL = interface board DMA channel


Return:
CF set on error
AL = error code (see #03786)

Note: If this routine is not called, the DMA channel is the driver's default (usually 3)

See Also: AH=00h - AH=03h"TARGA"
