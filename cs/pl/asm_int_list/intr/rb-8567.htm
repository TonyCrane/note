Int E0/CL=A1h

Interrups Categories Contents

REAL/32 - L_CATTACH - CONDITIONALLY ATTACH TO DEFAULT LIST DEVICE

CL = A1h


Return:
AX = status (00h = successful, FFh = unable to attach)
may destroy SI,DI

Desc: Attach the default list device to the calling process only if it is currently available

See Also: INT E0/CL=92h - INT E0/CL=9Eh - INT E0/CL=A0h

Category: Other Operating Systems - Int E0h - R
