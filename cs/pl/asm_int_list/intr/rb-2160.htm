Brother P-Touch - INSTALLATION CHECK

AX = 4B00h


Return:
AX = 4249h ('BI' = Brother International) if installed
BX = program ID
4550h ('EP') for "Epson FX-80 Emulator"
5047h ('PG') for "Printer to Disk Utility"
CX = numbero of parallel port being emulated or intercepted

See Also: AX=4B01h

Category: Power Management - Int 17h - B
