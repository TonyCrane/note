NOISE.SYS v0.53+ - GET ENTROPY ESTIMATE

AX = 6E11h


Return:
CF set on error
AL = error code (00h,FDh,FEh) (see #03166)
CF clear if successful
EBX = estimated bit count (refer to note below)
CL = FRACBITS (number of fractional bits)
EDX = low 32-bits of total number of samples added

Note: The estimated number of fresh random bits is equal to (EAX >> FRACBITS) + ((EAX & ((1 << FRACBITS)-1) / (1 << FRACBITS))

See Also: AH=6Eh - AX=6E00h

Category: Security - Int 32h - N
