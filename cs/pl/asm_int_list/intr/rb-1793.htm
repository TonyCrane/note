HUNTER 16 - CONTROL BREAK KEYS

AH = 2Dh
AL = enabled break keys
bit 0 Ctrl-C
bit 1 Ctrl-Break
BX = 0708h
CX = 0910h
DX = 1112h


Return:
AL = status (00h success, FFh failed)

See Also: AH=21h"HUNTER"
