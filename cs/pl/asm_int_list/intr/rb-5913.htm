CauseWay - GetMemNear - ALLOCATE APPLICATION-RELATIVE MEMORY

AX = FF16h
EBX = size in bytes of block to allocate


Return:
CF clear if successful
ESI = application-relative linear address of allocated block
CF set on error

See Also: AX=FF17h - AX=FF18h - AX=FF19h
