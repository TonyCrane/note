APM SL Enhanced v1.0 - GET CALENDAR EVENT DATE

AX = 5380h
BH = 0Ah


Return:
CF clear if successful
SI = century
DI = year
CH = month
CL = day
CF set on error
AH = error code (see #00483)

Desc: Reads calendar event date from Extended CMOS ram

See Also: AX=5380h/BH=08h - AX=5380h/BH=0Bh - AX=5380h/BH=0Ch - Category: Power Management - Int 15h - A
