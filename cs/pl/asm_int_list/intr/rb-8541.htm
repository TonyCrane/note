Int E0/CL=81h

Interrups Categories Contents

REAL/32 - M_FREE - DEALLOCATE MEMORY

CL = 81h
DS:DX -> Memory Free Parameter Block (MFPB) (see #04041)


Return:
AX = status (0000h success, else error code)

See Also: INT E0/CL=81h 


Format of REAL/32 Memory Free Parameter Block (MFPB) :

Offset  Size    Description     (Table 04041)
00h    WORD    starting segment of block to be freed
02h    WORD    reserved (0)

See Also: #04040

Category: Other Operating Systems - Int E0h - R
