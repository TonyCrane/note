DOS 3.3+ KEYB.COM - SET KEYBOARD CODE PAGE

AX = AD81h
BX = code page (see #01757 at INT 21/AX=6601h)


Return:
CF set on error
AX = 0001h (code page not available)
CF clear if successful

Notes: Called by DISPLAY.SYS. This function was undocumented prior to the release of DOS 5.0

See Also: AX=AD80h - AX=AD82h

Category: Keyboard Enhancers - Int 2Fh - D
