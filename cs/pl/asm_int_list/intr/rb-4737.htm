Kingswood TSR Windows - CLOSE WINDOW

AX = 3901h
BL = 05h


Return:
AX = error code (0000h if successful)
SI,DI,DS,ES preserved

Notes: Close the last opened TSR window.

See Also: AH=39h/BL=00h - AX=3901h/BL=02h
