COMM-DRV v14.0 - PORT CLEANUP

AH = 82h
DX = port number


Return:
AH bit 7 set on error
AH bit 7 clear if successful

Desc: Reset the port to its state before the AH=81h initialization and unhook any interrupts used by the port

See Also: AH=81h"COMM-DRV" - AH=83h"COMM-DRV"
