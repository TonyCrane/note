Everex Extended Video BIOS - GET PAGING FUNCTION POINTER FOR CURR MODE

AX = 7000h
BX = 0004h


Return:
ES:DI -> FAR paging function (call with DL = page to set)

Note: The word preceding ES:DI is the length of the function in bytes, and the last byte of the function is a FAR return instruction.

See Also: AX=4F05h - AX=7000h/BX=0000h - AX=7000h/BX=0005h
