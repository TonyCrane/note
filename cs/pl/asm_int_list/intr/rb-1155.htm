VMiX - sys_pinput - PROMPTED CONSOLE INPUT

AH = 03h.

STACK:
DWORD   pointer to ASCII prompt
WORD    field outline character
WORD    length of input field (max 7Fh)
DWORD   address of pointer to input buffer


Return:
AX = length of input (input buffer is padded with blanks)

See Also: AH=04h"VMiX"

Category: Other Operating Systems - Int 15h - V
