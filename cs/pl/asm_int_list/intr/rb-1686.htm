DESQview v2.00+ - GETMEM - ALLOCATE SYSTEM MEMORY

AX = DE0Ch
BX = number of bytes


Return:
ES:DI -> allocated block or 0000h:0000h (DV 2.26+)

Note: Use SETERROR (AX=DE15h) to avoid a user prompt if there is insufficient system memory

See Also: AX=1001h - AX=102Eh - AX=DE0Dh - AX=DE15h -
