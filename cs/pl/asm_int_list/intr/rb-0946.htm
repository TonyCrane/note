Alloy MW386 - ATTACH LOGICAL COMMUNICATIONS PORT TO PHYSICAL PORT

AH = 20h
AL = logical port (01h COM1, 02h COM2)
DX = physical port number


Return:
AX = status
0000h successful
FFFFh failed
