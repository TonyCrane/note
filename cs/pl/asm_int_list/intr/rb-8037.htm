G8BPQ v4.07+ - HOST MODE - ALLOCATE/DEALLOCATE STREAM

AH = 0Dh
AL = stream number (01h-FFh)
CL = function
01h allocate stream

Return:
CX = status (0000h successful, else already in use)
02h deallocate stream

See Also: AX=0D00h
