Int 2F/AX=168Fh/DH=02h

Interrups Categories Contents

Windows95 - CLOSE-AWARENESS - ACKNOWLEDGE CLOSE

AX = 168Fh
DH = 02h
DL = 00h (reserved)


Return:
AX = status
0000h successful
else failed

Note: Once a Close command has been issued, no further keyboard input is available to the application until it calls this function to acknowledge the Close request

See Also: AX=168Fh/DH=00h - AX=168Fh/DH=03h
