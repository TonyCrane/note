Adaptec and OMTI controllers - DRIVE 0 DATA

See Also: INT 41"HARD DISK 0" - INT 61"Adaptec" - INT 62"Adaptec" - INT 63"Adaptec"

See Also: INT 64"Adaptec" - INT C0"AMI" - #00732 at INT 1A/AX=B102h

Notes: This vector stores the first four bytes of the parameter table for hard disk 0. These vectors are used by the following Adaptec controllers: ACB 2370 A/B/C, ACB 2372 A/B/C, ACB 2333 A/B, 2322B-8, 2322B-16. These vectors are NOT used by the following Adaptec controllers: ACB 2310, ACB 2312, ACB 2320D, ACB 2322D

Category: Disk I/O Enhancements - Int 60h - A
