Int 10/AH=12h/BL=8Fh

Cirrus Logic BIOS v3.02 - LCD panel - ENABLE/DISABLE EXPAND MODE

AH = 12h
BL = 8Fh
AL = new state (00h enabled, 01h disabled)


Return:
Nothing

Note: When expand mode is enabled, the vertical dimension is enlarged to full screen
