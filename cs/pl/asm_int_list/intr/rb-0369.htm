HUNTER 16 - GET BACKLIGHT TIMEOUT

AX = 6500h


Return:
BX = timeout in seconds (0-300)

Desc: Returns the time after which the backlight will turn off

See Also: AH=63h - AH=64h - AX=6501h - INT 15/AH=74h
