Western Digital WD1002-27X SuperBIOS - DRIVE DATA (NOT A VECTOR!)

Note: The second WD1002-27X controller in a system uses the low byte to store the number of drives controlled by the second controller, and the high word for temporary storage during track recalculation; the first controller uses offsets 74h-77h in the BIOS data area (refer to MEMORY.LST) to store data

See Also: INT 47"SuperBIOS"
