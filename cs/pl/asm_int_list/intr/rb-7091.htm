FGDRIVER v4.02 - FG_SHOWFLIC - DISPLAY IMAGE FROM FLI/FLC FILE

AX = 00B8h
ES:BX -> ASCIZ filename for FLI/FLC file
CX = number of times to display image file (0000h = continuously)
DX = control flags (see #03479 at AX=0031h)


Return:
AX = status
0000h successful
0001h file not found
0002h not a valid FLI/FLC file

Notes: The FGDRIVER functions are rearranged with each major release, but their parameters do not change (see #03474). The FLI/FLC display may be stopped by the user by pressing Esc

See Also: AX=0030h - AX=0031h - AX=00B9h
