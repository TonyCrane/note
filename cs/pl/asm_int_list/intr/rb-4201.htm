Cloaking - SET 8259 IRQ BASE VECTORS

AX = 002Ah
BL = base vector of master interrupt controller
CL = base vector of slave interrupt controller

Notes: This call merely informs RM386 that the caller has changed the interrupt mappings. This function was first introduced with RM386 (RAM-MAN/386) v6.00

See Also: INT 67/AX=DE0Bh
