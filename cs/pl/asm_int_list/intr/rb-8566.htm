Int E0/CL=A0h

Interrups Categories Contents

REAL/32 - L_SET - SET DEFAULT LIST DEVICE

CL = A0h
DX = ID of list (printer) device


Return:
AX = status (0000h success, FFFFh if invalid printer number)
may destroy SI,DI

See Also: INT E0/CL=92h - INT E0/CL=9Eh - INT E0/CL=A4h

Category: Other Operating Systems - Int E0h - R
