DR DOS 5.0-6.0 internal - SET EXTENDED FILE ATTRIBUTES

AX = 4305h
DS:DX -> ASCIZ filename
???


Return:
CF clear if successful
CF set on error
AX = error code (see #01680 at AH=59h/BX=0000h)

Desc: This function allows the extended attributes, and optionally the encrypted password, of a file to be set.

Note: This function is only supported by DR DOS 5.0 and 6.0 and DRMDOS 5.1

See Also: AX=4304h - AX=4311h

Category: Other Operating Systems - Int 21h - D
