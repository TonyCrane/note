WHOA!.COM - SET DELAY COUNT

AX = 8902h
BX = delay count (larger values slow system down more)


Return:
AL = status
FDh successful
FEh error

Program: WHOA!.COM is a system slow-down utility by Brad D Crandall

See Also: AX=8900h
