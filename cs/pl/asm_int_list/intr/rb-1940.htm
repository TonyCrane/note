PC Tools v7 only CPSCHED/DESKTOP - ALTERNATE INSTALLATION CHECK

AX = FEF1h
BX = ???


Return:
CX = 5555h if PC Tools scheduler (CPSCHED or DESKTOP) installed
DX = 5555h

Note: This function is identical to AX=FFD3h, and is implemented by the same code in DESKTOP

See Also: AX=FFF1h
