JAM.SYS v1.10+ - UnLock - UNLOCK COMPRESSED DRIVE

AX = 5205h
DL = drive number (0-default, 1-A:, etc.) to unlock


Return:
AH = status (00h,01h,04h,06h,08h,09h,0Dh) (see #02835)
AL = host drive error code (see #02598 at INT 2F/AX=0802h)

See Also: AX=5204h - AX=5206h - AX=5207h

Note: Lock and UnLock functions were added to the JAM API to prevent asynchronous physical-level access (see AX=5206h,AX=5207h) to compressed data on JAM drives. In other words, two or more programs which use JAM API (say, JMAX optimizer and JCHKDSK - disk checker) cannot be run on the same JAM drive simultaneously.

Category: File/Disk Compression - Int 2Fh - J
