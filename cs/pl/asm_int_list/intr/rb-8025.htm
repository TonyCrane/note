MultiLink Advanced v1.0+ - SET TASK PRIORITY

AH = 09h
AL = priority (0-7)


Return:
Nothing

InstallCheck:
Ensure that the interrupt vector is not pointing at segment
0000h, then test whether the byte at offset 0000h in the interrupt
handler's segment is E9h

Index:
Installation check;MultiLink Advanced

See Also: AH=00h"MultiLink" - AH=0Ah"MultiLink"
