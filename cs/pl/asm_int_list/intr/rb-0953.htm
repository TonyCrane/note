PC-MOS/386 v5.01 $serial.sys v5.04 - OUTPUT CHARACTER, WITH TIMEOUT

AH = 21h
AL = char to send
DX = port number
SI = timeout in timer ticks (0000h = default)


Return:
After character is sent or timeout expires
