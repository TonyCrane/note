HP Vectra EX-BIOS - MODIFY MONITOR INFO

AX = 6F03h
BH = exclude mask (set bits are not modified)
BL = new values for bits indicated by BH (see #00190)


Return:
AX destroyed

Notes: This function is only valid when an HP MultiMode Video Display Adapter is installed. Supported by original HP Vectra AT and by ES/QS/RS series Vectras

See Also: AX=6F01h - AX=6F02h

Category: Video - Int 10h - H
