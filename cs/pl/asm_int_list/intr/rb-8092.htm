Alloy MW386 v2+ - SET ERROR MODE

AH = 4Eh
AL = error mode flags

bit 0:
Display critical disk errors

bit 1:
Display sharing errors.
DX = 4E58h ("NX")


Return:
AL = status
00h successful

See Also: AH=4Fh
