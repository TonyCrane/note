HUNTER 16 - RESET SERIAL NUMBER

AX = 7200h


Return:
AH = 00h if successful

Desc: Reset the serial number to the default serial number present when the Hunter 16 was shipped

See Also: AH=6Ch - AH=72h

Category: Vendor-specific BIOS Extensions - Int 15h - H
