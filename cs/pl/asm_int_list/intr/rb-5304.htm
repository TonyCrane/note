LapLink RemoteAccess (LLRA) - INSTALLATION CHECK

AX = C000h


Return:
AL = FFh if one of the components is installed
CX:BX -> signature within LLRA component:
"TSI_LapLinkCore" for LLRA1.EXE
"TSI_Blackbird"   for LLRA2.EXE
"TSI_Redirector"  for LLRA3.EXE
"TSI_Compression" for LLRA4.EXE
DL = major version number
DH = minor version number

Range:
AH=C0h to AH=FFh, selected by scanning for first free multiplex number

Note: LapLink components are installed in the order listed above, but not necessarily all four; each gets its own multiplex number (default C0h-C3h)
