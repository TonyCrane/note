APL*PLUS/PC - DETERMINE IF MEMORY AVAIL WITHOUT GARBAGE COLLECTION

AL = FCh
BX = amount of memory needed (paragraphs)


Return:
CF clear if memory available
CF set if a workspace compaction is required
