VMiX - sys_spawn - START A CHILD PROCESS JOB SHELL

AH = 0Dh.

STACK:
DWORD   ASCIZ string starting with requested I/O channel and
followed by standard VMiX shell command string


Return:
AX = process ID or error code "SYS_ERROR"

Note: The maximum string length is 7Fh characters

See Also: AH=0Eh"VMIX" - AH=11h"VMiX" - INT 21/AH=4Bh

Category: Other Operating Systems - Int 15h - V
