PU_1700.COM - INSTALLATION CHECK

AH = 18h
CX = 5055h ('PU')
DL = 00h


Return:
AX = 7570h ('up') if PU_1700 is installed

Program: PU_1700 is a BIOS enhancer from PU Service Systems which permits formatting diskettes at higher capacity (1.78M instead of 1.44M)

See Also: AX=057Fh/SI=324Dh"2M"

Category: Disk I/O Enhancements - Int 13h - P
