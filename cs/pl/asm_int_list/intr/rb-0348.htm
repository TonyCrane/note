Chips & Technologies '64300'/'65530' BIOS - GET LINEAR MEMORY INFO

AX = 5F10h


Return:
AL = 5Fh if function supported
AH = status
00h unsuccessful
01h successful
BX:CX = linear memory base address
DX = virtual screen width offset (bytes)
SI;DI = linear memory size

See Also: AX=5F08h - AX=5FA0h
