MS Windows - VPOWERD - GET API ENTRY POINT

AX = 1684h
BX = 0026h (virtual device ID for VPOWERD device) (see #02642)
ES:DI = 0000h:0000h


Return:
ES:DI -> VxD API entry point (see #02664)
0000h:0000h if the VxD does not support API in current mode

Return:

Return:
DX:AX = APM BIOS version

Return:
DX:AX = power management level

DX:AX = 0000h:0000h if successful
else error code (see #02665)
