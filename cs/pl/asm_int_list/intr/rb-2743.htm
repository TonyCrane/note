FlashTek X-32VM - SET SELECTOR BASE ADDRESS

AX = 3503h
BX = selector
ECX = base address


Return:
CF clear if successful
CF set on error (invalid selector)

See Also: AX=3504h - AX=3505h - INT 31/AX=0007h
