PhysTechSoft PTS ROM-DOS - SET DIAGNOSTIC DOS ERROR FLAG

AX = E902h
CX = diagnostic flag mask (see #04091,#04092)


Return:
Nothing

Note: Provided by PhysTechSoft's PTS ROM-DOS and displayed by SETUP.COM. DOS error flags can be assigned by application software.

See Also: AX=E901h - INT 14/AH=05h"PTS ROM-DOS" - INT 14/AH=06h"PTS ROM-DOS"
