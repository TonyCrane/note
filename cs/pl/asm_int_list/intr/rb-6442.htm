TCPDRV 2.01 - TCP/IP Application Binary Interface (ABI)

Note: The handler for the interrupt will start with a 2-byte NEAR jump instruction, followed by the ASCIZ signature string "TCP DRVR".. To find the interrupt being used by the driver, an application should scan through interrupt vectors 60h to 7Fh until it finds one with the "TCP DRVR" string.. This specification is being proposed by Peter R. Tattam from the University of Tasmania.
