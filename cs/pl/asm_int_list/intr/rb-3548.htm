Novell NetWare - FILE SERVER - CHECK CONSOLE PRIVILEGES

AH = E3h subfn C8h
DS:SI -> request buffer (see #02009)
ES:DI -> reply buffer (see #02007)


Return:
AL = status (00h,C6h) (see #02002)

Desc: Determine whether the caller is a console operator

Notes: This function is supported by Advanced NetWare 2.1+. NetWare determines console privileges by checking the file server's OPERATOR property for the caller's object ID

See Also: AH=E3h/SF=C9h - AH=E3h/SF=D1h - AX=F217h/SF=C8h 


Format of NetWare "Check Console Privileges" request buffer:

Offset  Size    Description     (Table 02009)
00h    WORD    0001h (length of following data)
02h    BYTE    C8h (subfunction "Check Console Privileges")

See Also: #02007
