HP Vectra EX-BIOS - F_SET_LIMITS_X - SET HORIZONTAL TRACKING LIMITS

AH = 0Ch
BP = 005Ah (driver ID for V_STRACK)
CX = minimum X coordinate
DX = maximum X coordinate


Return:
AH = status (see #03744)
BP,DS destroyed
