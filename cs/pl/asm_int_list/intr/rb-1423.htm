APM SL Enhanced v1.0 - GET RESUME CONDITION

AX = 5380h
BH = 06h


Return:
CF clear if successful
BL = resume condition (see #00482)

Desc: Reads the value of 82360SL RESUME_MASK register

See Also: AX=5380h/BH=04h - AX=5380h/BH=07h - AX=5380h/BH=08h -

Bitfields for APM SL resume condition: Bit(s) Description (Table 00482) 7-2 reserved (0) 1 alarm enabled (resume on CMOS alarm) 0 ring enabled
