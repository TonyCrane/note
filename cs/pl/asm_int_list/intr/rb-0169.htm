VIDEO - ALTERNATE FUNCTION SELECT (VGA, MCGA) - VIDEO ADDRESSING

AH = 12h
BL = 32h
AL = new state
00h enable video addressing
01h disable video addressing


Return:
AL = 12h if function supported

Desc: Specify whether the CPU should have access to video memory and the display adapters I/O registers

Category: Video - Int 10h - V
