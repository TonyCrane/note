PCMCIA Socket Services v1.00 - GET SOCKET SERVICES VERSION NUMBER

AH = 83h
AL = adapter number


Return:
CF clear if successful
AX = Socket Services version (BCD)
BX = implementation version (BCD)
CX = 5353h ("SS")
DS:SI -> ASCIZ implementor description
CF set on error
AH = error code (01h) (see #00656)

Note: The current version (from the Revision A.00 documentation) of Socket Services is 1.00 (AX=0100h)

See Also: AH=80h"PCMCIA"

Category: Expansion Bus Bios - Int 1Ah - P
