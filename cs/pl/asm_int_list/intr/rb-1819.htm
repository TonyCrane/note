Netroom PRENET - GET OLD INTERRUPT VECTORS

AX = 5758h
BX = 5858h


Return:
CF clear
DX:BX -> saved copy of interrupt vector table

InstallCheck:
Call this function and compare BX against 5858h on return; if
it has changed, PRENET is installed

See Also: AX=5758h/BX=5859h 

Index:
