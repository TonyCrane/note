Interrups Categories Contents

Alloy MW386 - REMOVE USER FROM GROUP

AH = F9h
AL = group number
DL = user number
DH = machine number (currently 00h)


Return:
CF clear if successful
CF set if failed

See Also: AH=F8h - AH=FAh
