MDEBUG command driver - START DRIVER

AX = D111h


Return:
DL = FFh

Notes: Only called by the transient part of the driver to inform the resident part that it is installed. The function must be reentrant and the driver mustn't chain this function to the old INT 2F

See Also: AX=D101h - AX=D112h
