PC/TCP kernel v2.05+ - ad_res_name - GET HOST NAME GIVEN ADDRESS

AH = 25h
DX,BX = IP address in network order
CX = length of buffer for name
DS:SI -> buffer for host name


Return:
CF clear if successful
buffer filled with ASCIZ host name
CF set on error
AX = error code (see #03319 at INT 61"FTP Software")

Note: This function will use the host table and or DNS to resolve the address, depending on kernel configuration. Use this call for the normal gethostbyaddr function.
