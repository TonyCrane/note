DoubleDOS - OTHER PROGRAM STATUS

AH = E5h


Return:
AL = status
00h no program in other partition
01h program in other partition is running
02h program in other partition is suspended

See Also: AX=E400h"DoubleDOS" - AH=F5h"DoubleDOS"
