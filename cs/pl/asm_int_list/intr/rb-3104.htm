LANtastic - SET MESSAGE PROCESSING FLAGS

AX = 5F9Bh
DL = bits describing processing for received unsolicited messages
(see #01731)


Return:
CF clear if successful
CF set on error
AX = error code

See Also: AX=5F9Ah - AX=5F9Eh
