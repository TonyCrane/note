CPU-generated (80286+) - STACK FAULT

Desc: This interrupt is generated in protected mode on a stack overflow or underflow, or if an inter-level transition or task switch references a stack segment marked "not present"; it is generated in real mode on accessing a word operand at SS:FFFFh

Note: The 80286 will shut down in real mode if SP=1 before a push. On the PC AT and compatibles, external circuitry generates a reset on shutdown.

See Also: INT 0B"CPU" - INT 0D"CPU"

Category: CPU-Generated - Int 0Ch - C
