HUNTER 16 - SET DISKETTE PORT

AX = 8401h.

BH = Drive (0:
A:, 1: B:).

BL = Port (0:
COM1, >0: COM2)

Desc: Set the COM port used for the floppy drive

See Also: AX=8400h

Category: Vendor-specific BIOS Extensions - Int 15h - H
