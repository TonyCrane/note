HP Vectra EX-BIOS - F_CMOS_RET - WRITE CMOS MEMORY

AH = 24h
BP = 0012h (driver ID of V_SYSTEM)
BL = address of CMOS byte to write
AL = new value


Return:
AH = status (see #03744)
BP,DS destroyed

Note: Updates the CMOS checksum fields (both IBM-standard and HP checksums)

See Also: AH=00h"HP Vectra" - AH=22h/BP=0012h
