ZIPKEY - CONVERT TWO-LETTER ABBREVIATION TO STATE CODE

AH = 71h
BX = abbreviation, in either case (first letter in BL)


Return:
CF set on error
AL = FFh
CF clear if successful
AL = ZIPKEY state code

See Also: AH=72h
