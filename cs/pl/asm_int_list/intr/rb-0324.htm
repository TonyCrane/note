VIDEO - AX PC - READ CHARACTER

AX = 5101h
BH = character width in bits
BL = character height
DX = character code (DH = 00h if 8-bit character)
ES:BP -> buffer for character bitmap


Return:
AL = status (00h successful, 01h failed)

See Also: AH=09h - AX=5100h
