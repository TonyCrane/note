HP OmniShare - Pen Driver - GET PEN BATTERY CHARGE

AX = BA13h


Return:
CF clear if successful
BL = pen battery state (00h good, 01h low charge)
CF set on error

Note: Returns the last value set by AX=BA12h.

See Also: AX=BA12h

Category: Non-Traditional Input Devices - Int 15h - H
