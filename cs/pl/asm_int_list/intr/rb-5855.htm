DPMI 0.9+ - RELOCK REAL MODE REGION

AX = 0603h
BX:CX = starting linear address
SI:DI = size of region in bytes


Return:
CF clear if successful
CF set on error
none of the memory is relocked
AX = error code (DPMI 1.0+) (8002h,8013h,8025h) (see #03143)

Notes: Pages at beginning and end will be relocked if the region overlaps them. Pageability of real mode pages is binary, not a count

See Also: AX=0602h
