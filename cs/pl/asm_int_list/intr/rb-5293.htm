AD-DOS - INSTALLATION CHECK

AX = C000h


Return:
AL = FFh if installed
BX = 4144h ('AD')
CX = 2D44h ('-D')
DX = 4F53h ('OS')

Program: AD-DOS is the DOS version of the After Dark screen blanker for MS Windows
