HUNTER 16 - CONTROL RS232 DRIVERS

AH = 25h
AL = new state of RS232 drivers (00h off, 01h on)


Return:
Nothing

Note: This function can be used to save power by turning off the RS232 drivers. It can also be used to turn on the RS232 drivers before connecting to a remote system to avoid "garbage" while the drivers turn on.
