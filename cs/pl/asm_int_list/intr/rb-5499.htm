Multiplex - Lotus CD/Networker GET DATA AREA

AX = D002h
BX = 4D44h
DX = bitmap identifying one loaded CD/Networker TSR


Return:
ES:DI -> data area owned by TSR

Note: The format of the data area changes with each minor revision, so it cannot be counted on

See Also: AX=D000h"Lotus"
