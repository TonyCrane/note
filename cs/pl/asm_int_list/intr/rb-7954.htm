SCSILink - PUT DATA

AH = 06h
CX = number of bytes to store (0001h-0100h)
ES:BX -> data to be saved


Return:
CF clear if successful
CF set on error
AL = error code (01h = too much data)

Note: The specified data is stored in SCSILink's PSP

See Also: AH=07h
