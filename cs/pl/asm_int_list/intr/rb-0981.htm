I1541 - SELECT LPT PORT FOR OUTPUT

AX = 4002h
BL = LPT number (1..3)


Return:
CF clear if successful
CF set otherwise

Desc: Force the input output routines to work on the cable placed on LPT BL

See Also: AX=4001h

Category: TSR Libraries - Int 14h - I
