PC Tools v7+ CPSCHED/DESKTOP - INSTALLATION CHECK

AX = FEEFh
CX = 0000h


Return:
CX = ABCDh if PC Tools scheduler (CPSCHED or DESKTOP) installed
BX = segment of resident portion
DX = (CPSCHED v8.0) resident CS

Note: This function is identical to AX=FFD3h, and is implemented by the same code in DESKTOP

See Also: AX=FFEFh

Category: Resident Utilities - Int 16h - P
