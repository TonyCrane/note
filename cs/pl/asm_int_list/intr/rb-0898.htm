FOSSIL - EXTENDED ^C/^K CHECKING AND TRANSMIT ON/OFF

AH = 10h
AL = bit mask

bit 0:
Enable/disable ^C/^K checking

bit 1:
Enable/disable the transmitter.
DX = port number


Return:
Nothing

See Also: AH=0Fh"FOSSIL"
