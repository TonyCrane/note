AMI WinBIOS - SECOND HARD DRIVE AUTODETECTION

Notes: Used by WinBIOS with core version of July 1994 or later. QEMM 7.5 began using this interrupt internally after the QPAT3 maintenance release when searching for "ROM holes", causing a lengthy delay during bootup. Adding the QEMM parameter RH:N will avoid the QEMM call to INT 81 and speed up the boot process

Category: Vendor-specific BIOS Extensions - Int 81h - A
