Novell NetWare - PRINT SERVICES - CANCEL LPT CAPTURE

AH = DFh
DL = 02h


Return:
AL = status
00h successful

Desc: This function ends the capture of the default LPT, removes the job from the print queue, and deletes the capture file unless it is a permanent capture file

Notes: This function is supported by NetWare 4.0+, Advanced NetWare 1.0+, and Alloy NTNX; under NTNX, it sends a print break (see INT 17/AH=84h). After this call, the default LPT defaults to local printing

See Also: AH=DFh/DL=00h - AH=DFh/DL=06h

Category: Netware - Int 21h - N
