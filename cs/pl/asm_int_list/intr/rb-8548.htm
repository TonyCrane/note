Int E0/CL=8Ah

Interrups Categories Contents

ConcCP/M,DR Multiuser DOS - Q_CREAD - CONDITIONALLY READ MSG QUEUE

CL = 8Ah
DS:DX -> queue parameter block (QPB) (see #04044,#04045)


Return:
AX = status (0000h success, FFFFh failure)
CX = error code (see #04021)

Note: Also supported by REAL/32

See Also: INT E0/CL=87h - INT E0/CL=89h - INT E0/CL=8Ch - Category: Other Operating Systems - Int E0h - C
