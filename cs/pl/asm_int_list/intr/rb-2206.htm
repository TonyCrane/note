Alloy MW386 - DEFINE TEMPORARY FILENAME

AH = A8h
CX:SI -> ASCIZ filename without extension (max 8 chars)


Return:
AL = status (see #00642)

Note: Allows application to specify banner page filename for spool files collected from the application's printer output

See Also: AH=A9h
