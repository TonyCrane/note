KEYBOARD - AX PC - GET KEYBOARD COUNTRY CODE

AX = 5001h


Return:
AL = status
00h successful
BX = country code
02h error

Note: Also supported by K3PLUS v6.00+

See Also: AX=5000h - INT 10/AX=5001h - INT 17/AX=5001h
