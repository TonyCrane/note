PCMCIA Socket Services v1.00 - START ERROR DETECTION CODE

AH = 98h
AL = adapter number
BH = EDC generator


Return:
CF clear if successful
AH destroyed
CF set on error
AH = error code (01h,04h) (see #00656)

See Also: AH=96h - AH=99h - AH=9Bh - AH=9Ch

Category: Expansion Bus Bios - Int 1Ah - P
