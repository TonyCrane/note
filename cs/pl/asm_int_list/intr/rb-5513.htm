MDEBUG command driver - END DRIVER

AX = D112h


Return:
DL = FFh

Notes: Only called by the transient part of the driver to inform the resident part that it will be released after this function. The function must be reentrant and the driver mustn't chain this function to the old INT 2F

See Also: AX=D101h - AX=D111h
