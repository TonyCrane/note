LANtastic - SET REDIRECTED PRINTER TIMEOUT

AX = 5FD1h
CX = printer timeout in clock ticks of 55ms, 0000h to disable timeouts


Return:
CF clear if successful
CF set on error
AX = error code

See Also: AX=5FD0h
