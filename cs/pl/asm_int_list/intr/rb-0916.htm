FOSSIL - INSERT/DELETE FUNCTION FROM TIMER TICK CHAIN

AH = 16h
AL = function
00h = delete
01h = add
ES:DX -> routine to call


Return:
AX = status
0000h successful
0001h unsuccessful

See Also: AH=07h"FOSSIL"
