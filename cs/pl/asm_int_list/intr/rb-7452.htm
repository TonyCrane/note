RM386 - GET RM386 INTERNAL DATA

AX = 5BF2h
CX = size of buffer
DS:SI -> buffer for internal data
(documentation says ES:BX -> buffer, SI = offset within RM386)


Return:
Buffer filled

Note: The data returned by this function is release-specific

See Also: AX=5BF0h
