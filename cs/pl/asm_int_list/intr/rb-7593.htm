DECnet DOS LOCAL AREA TRANSPORT - COPY LAT COUNTERS

AX = D400h
DH = FFh
CX = buffer size
ES:BX -> buffer for LAT counters


Return:
AX = status
0000h counters copied into buffer
FFFFh buffer too small
