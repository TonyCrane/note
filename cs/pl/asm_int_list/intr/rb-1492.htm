HUNTER 16 - SELECT POWER UP KEYS

AH = 71h
BL = column
BH = row
AL = column switch (00h disable, 01h enable)
CL = row switch (00h disable, 01h enable)


Return:
AH = status

See Also: AH=7Bh - AH=88h

Category: Vendor-specific BIOS Extensions - Int 15h - H
