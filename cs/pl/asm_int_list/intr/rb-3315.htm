Novell NetWare - PRINT SERVICES - SET CAPTURE PRINT QUEUE

AX = B806h
DH = LPT port (00h-02h)
BX:CX = print queue's object ID


Return:
AL = status
00h successful
FFh job already set

Desc: Specify the print queue on which a print job is to be placed the next time a capture is started on the given printer port

Note: This function is supported by Advanced NetWare 2.1+

See Also: AX=B801h - AX=B807h - AH=E0h/SF=09h
