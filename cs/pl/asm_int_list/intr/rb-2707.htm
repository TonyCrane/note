DOS 1+ - SET SYSTEM TIME

AH = 2Dh
CH = hour
CL = minute
DH = second
DL = 1/100 seconds


Return:
AL = result
00h successful
FFh invalid time, system time unchanged

Note: DOS 3.3+ also sets CMOS clock; due to the limitations of the CLOCK$ driver interface, the CMOS date is also updated to the current DOS date
