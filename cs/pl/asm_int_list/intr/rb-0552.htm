EGA Register Interface Library - WRITE ONE REGISTER

AH = F1h
DX = group index (see #00223)
if single register:
BL = value to write
otherwise
BL = register number
BH = value to write


Return:
BL = data

Note: The RIL is provided by EGA.SYS, the Microsoft Mouse driver, the OS/2 compatibility box, and others

See Also: AX=7F05h - AH=F0h"EGA" - AH=F3h"EGA" - AH=FAh"EGA"
