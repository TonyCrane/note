Interconnections Inc. TES - GET LIST OF SESSIONS WITH STATUS

AH = A1h


Return:
CX = number of active sessions
ES:SI -> status array (see #00395)

See Also: AH=A2h"TES" - AH=A3h"TES" 


Format of Interconnections TES status array entry:

Offset  Size    Description     (Table 00395)
00h    BYTE    status
01h    WORD    offset of name
