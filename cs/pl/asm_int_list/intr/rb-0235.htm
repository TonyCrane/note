VIDEO - FLAT-PANEL - CONTRAST SETTING

AX = 1E08h
BH = function

bit 7:
=1 set contrast control, =0 query contrast

bit 6:
Use standard contrast

bits 5-0:
Reserved (0).
---if BH bits 7,6=10---.
BL = contrast (00h = minimum, FFh = maximum)


Return:
AL = 1Eh if function supported
BH = results

bit 7:
Query/set (copied from input)

bit 6:
Standard/custom (copied from input)

bits 5-2:
Reserved (0)

bit 1:
Software contrast control is supported

bit 0:
Set operation was succesful (always clear on get)
BL = contrast (00h = minimum, FFh = maximum)

Note: This function operates independently of AX=1E06h

See Also: AX=1E00h - AX=1E06h - AX=1E07h
