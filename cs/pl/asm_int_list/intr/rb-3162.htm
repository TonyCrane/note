Windows95 (OSR2) - SET GENERAL INTERNATIONALIZATION INFO

AX = 6500h
ES:DI -> buffer containing internationalization info (see #01750)
CX = size of buffer (>= 7)


Return:
CF clear if successful
AX = system code page
CX = number of bytes copied from supplied buffer
CF set on error
AX = error code

Note: This function will not overwrite the country or code-page numbers, but can be used to modify all other data about the country and code page

See Also: AH=65h"GET EXTENDED" - AH=70h

Category: Uncatagorized - Int 21h - W
