Int 21/AH=DFh/DL=05h

Novell NetWare - PRINT SERVICES - END SPECIFIC LPT CAPTURE

AH = DFh
DL = 05h
DH = LPT port (00h-02h)


Return:
AL = status
00h successful

Desc: Stop redirecting the specified LPT, close the capture file, and release the job in the print queue for printing

Notes: This function is supported by Advanced NetWare 2.1+. After this call, the specified LPT defaults to local printing

See Also: AH=DFh/DL=01h - AH=DFh/DL=04h - AH=DFh/DL=06h - Category: Netware - Int 21h - N
