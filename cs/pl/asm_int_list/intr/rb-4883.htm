JAM.SYS v1.10+ - Read - PHYSICAL READ DATA FROM JAM ARCHIVE

AX = 5206h
DL = drive number (0-default, 1-A:, etc.)
DS:BX -> disk transfer packet (see #02836)


Return:
AH = status (00h,01h,04h,06h,07h,08h,0Dh) (see #02835)
AL = host drive error code (see #02598 at INT 2F/AX=0802h)

Program: JAM.SYS is a main component of the JAM Real-Time Data Compression Utilities by George A. Reznik and friends (JAM Software).

See Also: AX=5207h 


Format of disk transfer packet:

Offset  Size    Description     (Table 02836)
00h    DWORD   sector number
04h    WORD    number of sectors to read(write)
06h    DWORD   transfer address

Category: File/Disk Compression - Int 2Fh - J
