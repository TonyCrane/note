HUNTER 16 - SET LOW POWER THRESHOLD

AH = 62h
AL = level (00h 5%, 01h 10%, ... 12h 95%)
BX = interval between power low warnings in seconds (1..600)


Return:
AH = Status

Desc: Set the level (relative to full power) when power-low warnings begin and the interval between the warnings

See Also: AH=61h - AH=65h - AH=66h

Category: Vendor-specific BIOS Extensions - Int 15h - H
