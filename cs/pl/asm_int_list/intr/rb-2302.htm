SNAP.EXE 3.2+ - SNAP_ALLOC - ALLOCATE BUFFER IN SNAP DATA SEGMENT

AX = 6A01h.

STACK:
WORD    number of bytes to allocate


Return:
AX = offset of allocated buffer or 0000h if out of memory
STACK unchanged

Program: SNAP.EXE is a TSR written by IBM and Carnegie Mellon University which implements the Simple Network Application Protocol

See Also: AX=6B01h

Category: Network - Int 1Ah - S
