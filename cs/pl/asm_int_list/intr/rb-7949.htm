SCSILink - CONNECT TO TARGET

AH = 01h
AL = target ID


Return:
CF clear if successful
CF set on error
AL = initiator error
AH = target error

Desc: Arbitrate the use of the bus and select the specified target device

Category: Debuggers/Debugging Tools - Int 7Dh - S
