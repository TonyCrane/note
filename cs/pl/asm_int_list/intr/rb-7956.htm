SCSILink - TERMINATE SESSION

AH = 08h

Desc: Indicate to any other programs that intercept INT 7D that the program has finished with the link

Note: The program should call this function even if a SCSI error caused its termination

Category: Debuggers/Debugging Tools - Int 7Dh - S
