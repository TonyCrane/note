InterWave Game API - MIXER SETTINGS CHANGED BROADCAST

AX = CD80h


Return:
Nothing

Desc: Called by an application controlling the InterWave chip's mixer whenever the mixer's settings are changed 

Range:
AH=CDh-ECh, selected by scanning for an available multiplex number

Note: When an application intercepts this function call, it must read the new mixer settings from the InterWave chip

See Also: AX=CD00h"InterWave" - AX=CD01h"InterWave" - AX=CD02h"InterWave"

See Also: AX=CD05h"InterWave" - AX=CD21h"InterWave" - AX=CD22h"InterWave"
