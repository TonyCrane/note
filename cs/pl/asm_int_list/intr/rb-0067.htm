CPU-generated (80286+) - COPROCESSOR ERROR

Desc: This interrupt is generated by the CPU when the -ERROR pin is asserted by the coprocessor

Note: AT's and clones usually wire the coprocessor to use IRQ13, but not all get it right

See Also: INT 09"hardware" - INT 75
