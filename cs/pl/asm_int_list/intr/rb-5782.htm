NORTON UTILITIES 6.0 NCACHE - REBOOT

AX = FE10h
DI = 4E55h ("NU")
SI = TSR identifier (see #03140)


Return:
SI = TSR reply (lowercase version of entry SI, i.e. SI or 2020h)
AX = status

Note: Probably used to flush NCACHE buffers and reboot when Ctrl-Alt-Del is detected

See Also: AX=FE03h

Category: Resident Utilities - Int 2Fh - N
