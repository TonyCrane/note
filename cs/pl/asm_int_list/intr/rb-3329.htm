Novell NetWare, Alloy NTNX - LOG/LOCK RECORD (FCB)

AH = BFh
AL = flags (see #01808)
DS:DX -> opened FCB (see #01345 at AH=0Fh)
BX:CX = offset
BP = lock timeout in timer ticks (1/18 sec) if AL nonzero
SI:DI = length


Return:
AL = error code (see #01807)

Note: This function was added in NetWare 4.6, but was removed some time prior to Advanced NetWare 2.15, and is no longer listed in current Novell documentation

See Also: AH=BCh - AH=C0h"NetWare" - AH=C2h"NetWare"
