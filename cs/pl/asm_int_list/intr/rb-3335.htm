Novell NetWare, Alloy NTNX - CLEAR RECORD (FCB)

AH = C1h
DS:DX -> opened FCB (see #01345 at AH=0Fh)
BX:CX = offset


Return:
AL = error code (see #01807)

Note: Unlocks record and removes it from log table. This function was added in NetWare 4.6, but was removed some time prior to Advanced NetWare, and is no longer listed in current Novell documentation

See Also: AH=BEh - AH=C0h"NetWare" - AH=C4h
