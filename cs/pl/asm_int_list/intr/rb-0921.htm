Digiboard - CCB COMMAND - SET MODEM LINES

AX = 1649h
BL = bits to set
BH = bits to clear
CL = byte 1
DX = port number (00h-03h) (04h-43h for XAPCM232.SYS)


Return:
AH = status
00h successful
80h error
FFh error

See Also: AH=16h"Digiboard" - AX=1647h
