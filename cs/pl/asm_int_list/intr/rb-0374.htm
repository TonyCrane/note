DGIS - INQUIRE INT 10 OUTPUT DEVICE

AX = 6A02h
ES:DI = 0000h:0000h


Return:
ES:DI = 0000h:0000h  if current display is non-DGIS
else address of the current DGIS INT 10 display

See Also: AX=6A00h - AX=6A01h

Category: Video - Int 10h - D
