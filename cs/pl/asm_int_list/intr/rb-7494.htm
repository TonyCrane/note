Virtual Control Program Interface - GET NUMBER OF FREE 4K PAGES

AX = DE03h


Return:
AH = 00h  successful
EDX = number of free 4K pages.

AH nonzero:
Failed

Notes: Returns total number of pages available to ALL tasks in system. Also available in protected mode by calling the protected-mode VCPI entry point (see AX=DE01h,#03664)

See Also: AX=DE04h

Category: DOS Extenders - Int 67h - V
