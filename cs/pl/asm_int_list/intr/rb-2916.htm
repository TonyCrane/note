DR DOS 5.0-6.0 - SET GLOBAL PASSWORD

AX = 4414h
DS:DX -> password string (blank-padded to 8 characters)

Desc: Specify the master password for accessing files.

Note: This obsolete call, which is no longer supported in Novell DOS 7, is identical to AX=4454h

See Also: AX=4454h

Category: Other Operating Systems - Int 21h - D
