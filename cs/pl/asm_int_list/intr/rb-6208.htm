TI Professional PC - DISPLAY SYSTEM ERROR MESSAGE

AH = 08h
BX = error number


Return:
Nothing

Desc: Displays the error message " ** System Error ** - xxxx" where xxxx is the hexadecimal value in BX

See Also: AH=00h - AH=01h - AH=02h - AH=03h - Category: Vendor-specific BIOS Extensions - Int 48h - T
