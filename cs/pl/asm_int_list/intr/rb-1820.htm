Netroom POSTNET - GET OLD INTERRUPT VECTORS

AX = 5758h
BX = 5859h


Return:
CF clear
DX:BX -> saved copy of interrupt vector table

InstallCheck:
Call this function and compare BX against 5859h on return; if
it has changed, POSTNET is installed

See Also: AX=5758h/BX=5858h
