DOS 3.3+ internal - OPEN FILE

AX = 1226h
CL = access mode
DS:DX -> ASCIZ filename
SS = DOS DS (must be using a DOS internal stack)


Return:
CF set on error
AL = error code (see #01680 at INT 21/AH=59h/BX=0000h)
CF clear if successful
AX = file handle

Notes: Can only be called from within DOS. Equivalent to INT 21/AH=3Dh. Used by NLSFUNC to access COUNTRY.SYS when invoked by the DOS kernel

See Also: AX=1227h - INT 21/AH=3Dh
