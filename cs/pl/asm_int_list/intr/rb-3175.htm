Int 21/AX=69FFh/DX=CFBFh

CUBIT v4.00 - UNINSTALL

AX = 69FFh
DX = CFBFh
CX = EFCFh
BX = CB00h (magic value)


Return:
ES:BX -> CUBITR.EXE handler for INT 21
CX = status
2020h successful
2222h failed

Note: If DX is neither 0000h nor CFBFh on entry, some other code is executed

See Also: AX=69FFh/DX=0000h
