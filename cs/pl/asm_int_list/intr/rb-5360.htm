Explosiv v2.00+ - UNINSTALL

AX = C052h
DX:BX = address to return to on successful uninstall


Return:
At specified address if successful
AL = 53h on error

Note: Specified return address must have the segment of the caller's PSP

See Also: AX=C050h 

Index:
Screen saver;Explosiv
