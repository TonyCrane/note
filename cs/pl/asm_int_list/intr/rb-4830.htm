SMARTDRV v4.00+ - CHECK IF DRIVE CACHEABLE

AX = 4A10h
BX = 0006h
CL = drive number (01h = A:)


Return:
AX = 0006h if drive should not be cached by SMARTDRV

Note: Called by SMARTDRV at startup to determine whether it should cache a particular drive

See Also: AX=4A10h/BX=0000h

Category: Caches/Spoolers - Int 2Fh - S
