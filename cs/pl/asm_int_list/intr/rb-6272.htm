Virtual DMA Specification - ENABLE DMA TRANSLATION

AX = 810Ch
BX = DMA channel number
DX = 0000h


Return:
CF clear if successful
ZF set if disable count decremented to zero
CF set on error
AL = error code (see #03218)

See Also: AX=810Bh

Category: Disk I/O Enhancements - Int 4Bh - V
