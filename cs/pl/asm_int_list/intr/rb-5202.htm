WinDOS v2.11 - MAKE PROGRAM UNBREAKABLE

AX = AF06h
DS:DX -> break handler
DI:BX = stack to use for break handler


Return:
Nothing

Desc: Set a handler which should be invoked when Ctrl-C is pressed instead of aborting the program

See Also: AX=AF00h"WinDOS" - AX=AF0Ah"WinDOS" - AX=AF0Ch"WinDOS"
