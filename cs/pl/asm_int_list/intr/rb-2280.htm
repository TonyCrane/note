Int 1A/AH=05h

Interrups Categories Contents

TIME - SET REAL-TIME CLOCK DATE (AT,XT286,PS)

AH = 05h
CH = century (BCD)
CL = year (BCD)
DH = month (BCD)
DL = day (BCD)


Return:
Nothing

See Also: AH=04h - INT 21/AH=2Bh"DATE" - INT 4B/AH=00h"TI"
