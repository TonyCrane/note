INT 1A - PCI BIOS v2.0c+ - INSTALLATION CHECK
AX = B101h
EDI = 00000000h


Return:
AH = 00h if installed
CF clear
EDX = 20494350h (' ICP')
EDI = physical address of protected-mode entry point (see #00731)
AL = PCI hardware characteristics (see #00730)
BH = PCI interface level major version (BCD)
BL = PCI interface level minor version (BCD)
CL = number of last PCI bus in system
EAX, EBX, ECX, and EDX may be modified
all other flags (except IF) may be modified

Notes: This function may require up to 1024 byte of stack; it will not enable interrupts if they were disabled before making the call. Some BIOSes do not change EDI, so applications looking for the protected-mode entry point should set EDI to 00000000h before calling this function

See Also: AX=B181h
