Qualitas 386MAX v6.01+ - TURN 386MAX OFF

AX = 4403h subfn 01h
BX = handle for device "386MAX$$"
DS:DX -> BYTE 01h
CX ignored


Return:
DS:DX -> BYTE status (00h = successful)
CF clear if successful
AX destroyed
CF set on error
AX = error code (01h,05h,06h,0Dh) (see #01680 at AH=59h/BX=0000h)

Note: This function will fail if any EMS or UMBs are in use

See Also: AX=4403h/SF=02h - AX=4403h/SF=03h - AX=4403h/SF=05h
