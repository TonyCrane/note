VIDEO - GRAPH-MODE CHARGEN - SET ROM 8x14 GRAPHICS CHARS (PS,EGA,VGA)

AX = 1122h
BL = row specifier (see AX=1121h)


Return:
Nothing

Notes: This function is meant to be called immediately after a mode set; results are unpredictable at other times. UltraVision v2+ sets INT 43 to the appropriate font on this call

See Also: AX=1111h - AX=1120h - AX=1121h - AX=1123h -
