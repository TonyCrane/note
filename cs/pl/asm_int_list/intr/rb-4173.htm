Cloaking - SET SEGMENT BASE ADDRESS

AX = 0002h
SI = selector
EBX = new physical base addres


Return:
CF clear if successful
CF set on error
AX = error code (see #02556)

Note: This function was first introduced with RM386 (RAM-MAN/386) v6.00

See Also: AX=0000h - AX=0003h - AX=0004h - INT 31/AX=0007
