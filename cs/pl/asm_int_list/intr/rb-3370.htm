Novell NetWare - SYNCHRONIZATION SERVICES - RELEASE FILE SET

AH = CDh


Return:
None

Desc: Unlock all files listed in the log table, but don't remove them from the table

Note: This function is supported by NetWare 4.0+, Advanced NetWare 1.0+, and Alloy NTNX

See Also: AH=CBh - AH=CCh - AH=CFh - AH=D3h

Category: Netware - Int 21h - N
