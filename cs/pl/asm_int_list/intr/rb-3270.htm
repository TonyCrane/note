European MS-DOS 4.0 - HARD ERROR PROCESSING

AH = 95h
AL = new state
00h enabled
01h disabled, automatically fail hard errors


Return:
AX = previous setting

Desc: Specify whether hard (critical) errors should automatically fail the system call or invoke an INT 24

See Also: INT 24 - INT 21/AH=59h/BX=0001h
