AD-DOS - CHECK FOR NEW INPUT

AX = C002h


Return:
AL = 00h if successful
BX = status
0000h no input since last check
0001h new input available

Note: This call also resets the new-input flag

See Also: AX=C000h"AD-DOS" - AX=C004h
