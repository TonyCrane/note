Intel Image Processing Interface - GET VERSION NUMBER

AX = CD02h


Return:
AL = status
00h/01h successful
BH = major version number (BCD)
BL = minor version number (BCD)
80h error

Note: If return AL = 01h, the IPI supports network redirection

See Also: AX=CD00h"Image"
