Novell NetWare - WRITE PROPERTY VALUE

AX = F217h subfn 3Eh
CX = length of request packet in bytes
DX = 0000h (no reply packet)
DS:SI -> request packet (see #01944 at AH=E3h/SF=3Eh)
ES:DI ignored


Return:
AX = status (see #02860 at INT 2F/AX=7A20h/BX=0000h)
