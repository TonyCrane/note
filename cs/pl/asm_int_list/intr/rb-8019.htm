G8BPQ v4.00+ - HOST MODE - SESSION CONTROL

AH = 06h
AL = stream number (01h-40h)
CX = subfunction
0000h connect to node

DL bit 0:
Use BBS callsign instead of Node Call
0001h connect to node
use BBS Call if APPLMASK=1
0002h disconnect
0003h return user to node

See Also: AH=01h"G8BPQ" - AH=04h"G8BPQ"
