Advanced Power Management v1.0+ - RESTORE POWER-ON DEFAULTS

AX = 5309h
BX = device ID for all devices power-managed by APM
0001h (APM v1.1)
FFFFh (APM v1.0)


Return:
CF clear if successful
CF set on error
AH = error code (03h,09h,0Bh) (see #00473)

Note: Should not be called from within a hardware interrupt handler to avoid reentrance problems

See Also: AX=5308h

Category: Power Management - Int 15h - A
