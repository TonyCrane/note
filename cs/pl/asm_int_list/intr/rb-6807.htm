PC/TCP kernel v2.05+ - net_globalize - MAKE NET DESCRIPTOR GLOBAL

AH = 07h
BX = local network descriptor


Return:
CF clear if successful
AX = global network descriptor
CF set on error
AX = error code (see #03319)

Notes: The new network descriptor can be accessed from all processes and is independent of DOS. This function is not supported by Beame&Whiteside's BWPCTCP v3.0a shim
