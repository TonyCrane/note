FGDRIVER v4.02 - FG_GETCHAR - GET CHARACTER FOR SCREEN POSITION

AX = 003Dh
BX = row
CX = column


Return:
AX = character at specified location on active video page

Notes: The FGDRIVER functions are rearranged with each major release, but their parameters do not change (see #03474 at AX=0000h). This call is ignored in graphics modes
