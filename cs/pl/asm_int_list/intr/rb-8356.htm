ZIPKEY - CONVERT STATE CODE TO TWO-LETTER ABBREVIATION

AH = 72h
BL = ZIPKEY state code


Return:
CF set on error
AX destroyed
CF clear if successful
AX = abbreviation, in upper case

See Also: AH=71h - AH=73h
