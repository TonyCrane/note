VIDEO - Hercules GRAFIX - GPAGE - SELECT DRAWING PAGE

AH = 43h
AL = page number (0,1)


Return:
Nothing

Desc: Specify which of the two pages of video memory is to be used for output

See Also: AH=05h - AH=44h - AH=45h
