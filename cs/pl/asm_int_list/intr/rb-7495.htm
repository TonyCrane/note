Virtual Control Program Interface - ALLOCATE A 4K PAGE

AX = DE04h


Return:
AH = 00h successful
EDX = physical address of allocated page.

AH nonzero:
Failed

Notes: The client program is responsible for freeing all memory allocated with this call before terminating. Also available in protected mode by calling the protected-mode VCPI entry point (see AX=DE01h,#03664)

See Also: AH=3Fh - AX=DE03h - AX=DE05h
