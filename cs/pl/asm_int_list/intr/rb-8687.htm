Common ISDN API (CAPI) v1.1 - API-GET-VERSION

AH = F1h
ES:BX -> 64-byte buffer for CAPI version number


Return:
ES:BX buffer filled with ASCIZ version string

Notes: The caller is required to provide at least 512 bytes of stack space

See Also: AH=01h - AH=F0h - AH=F2h - AH=FFh
