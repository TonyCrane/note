HUNTER 16 - GET SERIAL NUMBER

AH = 6Ch


Return:
BL:CX = BCD serial number

Note: When shipped, all Hunter 16s have the same Serial Number, but a different number can be stored (see AH=72h)

See Also: AH=6Bh - AH=72h

Category: Vendor-specific BIOS Extensions - Int 15h - H
