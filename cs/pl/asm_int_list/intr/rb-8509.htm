DR Multiuser DOS - A_STATIN - GET INPUT STATUS OF AUX DEVICE

CL = 07h


Return:
AL = status
00h not ready
FFh character available

Desc: Determine whether the current AUX device has input available

See Also: INT E0/CL=03h - INT E0/CL=08h

Category: Other Operating Systems - Int E0h - D
