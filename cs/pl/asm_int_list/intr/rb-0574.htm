CPU-generated (80486+) - ALIGNMENT CHECK

Desc: Automatically generated by the CPU when the AC flag is set, the current privilege level is 3, and a misaligned memory access (WORD not on an even address or DWORD not on a multiple of 4) is made

Note: Not all V86 monitors allow the AC flag to be set, such as Turbo Debugger 386

See Also: INT 12"CPU"

Category: CPU-Generated - Int 11h - C
