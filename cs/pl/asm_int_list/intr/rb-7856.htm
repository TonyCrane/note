Novell NetWare - IPX Driver - internal - TERMINATE SOCKETS

BX = 000Eh


Return:
Nothing

Notes: This function terminates all sockets opened with the current mode; this may be intended for future enhancements as the socket mode never changes in v2.15. Called by the NetWare shell if a program terminates
