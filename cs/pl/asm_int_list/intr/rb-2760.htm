XPACK v1.65 - SET TEMPORARY DIRECTORY NAME

AX = 37A7h
BX = 6A6Dh ('jm')
DS:DX -> ASCIZ name of temporary directory (max 64 chars)


Return:
AX = 0000h

Note: The specified directory name must include a drive letter and end with a backslash (e.g. 'c:\dos\',0)

See Also: AX=37A0h/BX=6A6Dh - AX=37A6h/BX=6A6Dh

Category: Uncatagorized - Int 21h - X
